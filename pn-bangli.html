8. Jarak denpasar ke bangli adalah 60 km.ayah berangkat dari denpasar menuju bangli menggunakan mobil .waktu yang di perlukan ayah untuk menempuh perjalanan tersebut 45 menit kecepatan rata rata mobil ayah adalah


Diketahui jarak Kota Denpasar dan kota Bangli adalah 60 km. Jika ayah menempuh perjalanan selama 45 menit dari kota denpasar menggunakan mobil. Maka kecepatan rata rata ayah (V) adalah 80 km/jam.

Penyelesaian Soal :

Diketahui : Jarak tempuh (s) = 60 km

                  waktu tempuh (t) = 45 menit

Ditanya : Kecepatan rata rata (V) ?

Jawab :

LANGKAH PERTAMA (I)

Konversikan satuan waktu menit menjadi bentuk jam dengan menggunakan cara sebagai berikut :

1 menit = ¹/₆₀ jam → 45 menit = 45 × ¹/₆₀ jam = ³/₄ jam

LANGKAH KEDUA (II)

Hitung kecepatan ayah (V) dengan menggunakan cara sebagai berikut :

V = [tex]\frac{s}{t}[/tex]

keterangan : v = kecepatan (km/jam)

                      s = jarak yang ditempuh (km)

                     t = waktu tempuh (jam)

Maka caranya yaitu :

V = s/t

   = 60 km / ³/₄ jam

   = 80 km/jam

∴ Kesimpulan kecepatan ayah (V) adalah 80 km/jam.

13. pola bilangan persegi panjang jika Pn = 552 suku keberapa Pn ?


Matematika
Pola Bilangan VIII SMP

Pembahasan :
Pola Persegi Panjang

Pn = 552
n(n + 1) = 552
n² + n = 552
n² + n - 552 = 0
(n - 23)(n + 24) = 0
n = 23 —» Pada suku ke-23

22. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722
