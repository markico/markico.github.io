6. jika diketahui cos x drajat = -4/5 dan (90 drajat < x drajat < 180 drajat).Maka nilai cosec x drajat adalah


cosx = -4/5
sin²x + cos²x = 1
sin²x = 1 - (-4/5)² = 1 - (16/25) = 9/25
sinx = 3/5

cosecx = 1/sinx = 1/(3/5) = 5/3 = 1(2/3)

7. Hitunglah nilai dari : a. sin (-45 drajat) b. tan (-240 drajat) c .cos (-135 drajat) d. sin (-225 drajat) e. cotan (-120 drajat)


a. sin(-45) = - sin 45 = -1/2 √2
b. tan(-240) = - tan 240 = - tan(180 + 60) = - tan 60 = -√3
c. cos(-135) = cos 135 = cos(180-45) = - cos 45 = -1/2√2
d. sin(-225) = - sin 225 = - sin(180+45) = sin 45 = 1/2√2
e. cotan(-120) = - cotan 120 = - cotan(180 - 60) = cotan 60 = 1/3√3

19. 120 drajat R =...... drajat F​


Jawaban:

120°R = 302°F

Penjelasan:

tF = (9/4 x tR) + 32°

tF = (9/4 x 120°) + 32°

   = 270° + 32°

   = 302°F

Jawaban beserta langkah-langkah:
