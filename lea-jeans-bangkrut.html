12. harga sebuah celana jeans dan 3 buah kaos Rp.145.000 sedangkan harga 2 celana jeans dan sebuah kaos Rp.165.000. harga sebuah celana jeans dan sebuah kaos adalah​


Penjelasan dengan langkah-langkah:

cara diatas

semoga membantu :)

Jawaban:

Diketahui :

Celana jeans dan 3 buah kaos Rp. 145,000

2 celana jeans dan sebuah kaos Rp. 165,000

Ditanyakan :

Harga sebuah Celana jeans dan sebuah kaos?

Jawab :

Soal ini membahas tentang SPLDV atau Sistem persamaan linear dua variabel, didalam SPLDV ada metode substitusi, eliminasi, grafik, determinan, dll.

Pada soal ini kita gunakan metode Eliminasi , caranya yaitu :

misal :

x = celana jeans

y = kaos

maka , model matematika nya seperti ini :

x + 3y = 145,000

2x + y = 165,000

Metode Eliminasi :

x + 3y = 145,000 |×2

2x + y = 165,000 |×1

2x + 6y = 290,000

2x + y = 165,000

----------------------------- -

5y = 125,000

y = 125,000 : 5

y = 25,000

substitusi kan y = 25,000 ke :

2x + y = 165,000

2x + 25,000 = 165,000

2x = 165,000 - 25,000

2x = 140,000

x = 140,000 : 2

x = 70,000

Diperoleh harga x(celana jeans) dan y(kaos)

maka,

Harga sebuah Celana jeans dan kaos yaitu :

x = harga celana jeans Rp. 70,000

y = harga kaos Rp. 25,000

total sebuah Celana jeans dan kaos adalah :

= x + y

= 70,000 + 25,000

=Rp. 95,000

semoga membantu

21. Harga satu jeans dan dua kaos Rp150.000,- sedangkan harga 3 jeans dan 3 kaos Rp270.000,- . Harga 2 jeans dan 5 jeans adalah.


Jawab:

Jeans = x

Kaos = y

Diketahui :  x + 2y = 150.000

                 3x + 3y = 270.000

Ditanya :  2x + 5y

Dijawab :

x + 2y  = 150.000  ║ × 3

3x + 3y = 270.000  ║ × 1

-----------------------------------

3x + 6y = 450.000

3x + 3y = 270.000

----------------------------- -

       3y = 180.000

         y = Rp 60.000  ( Harga 1 kaos = Rp 60.000 )

Ganti y ke salah satu persamaan

x + 2y = 150.000

x + 2(60.000) = 150.000

x + 120.000 = 150.000

x = 150.000 - 120.000

x = 30.000 ( Harga 1 jeans = Rp. 30.000 )

Karena soal kurang tepat maka saya ksih 2 soal

a) Harga 2 jeans dan 5 kaos (2x + 5y)b) Harga 2 kaos dan 5 jeans  (2y + 5x)

jawab :

a) Harga 2 jeans dan 5 kaos (2x + 5y)

⇔ 2(30.000) + 5(60.000)

⇔ 60.000 + 30.000

⇔ Rp 90.000

b) Harga 2 kaos dan 5 jeans  (2y + 5x)

⇔ 2(60.000) + 5(30.000)

⇔ 120.000 + 150.000

⇔ Rp 270.000

harga awalnya adalah Rp120.000,00

Haga Awal = x

Besar Diskon = 15%.x

= 0,15x

Harga Awal = Besar Diskon + Harga setelah Diskon

x = 0,15x + Rp 102.000,00

x-0,15x = Rp 102.000,00

0,85x = Rp 102.000,00

x = Rp 120.000,00

Jadi, harga awalnya adalah Rp120.000,00

ATAU CARA SIMPLENYA

100%-15% = 85% = 0,85

102000: 0,85 = 120000 jadi harga celana jeans sebelum didiskon 120.000,00
