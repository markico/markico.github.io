Write the correct word in each blank to complete the following conversation.

1. A: Where does Mr. Simon Baba live?

B: his flat is on Karadenan Street.

2. A: How far is it from here?

B: it’s only two kilometers

3. A: What is his address?

B: I think it’s 31 Jasmine street.

4. A: How can I get there?

B: You can go by bus.

5. A: Is there a bus stop near here?

B: yes, there’s just around the corner.

6. A: Do I turn right here?

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran bentuk kalimat Simple Present Tense karena menjelaskan fakta dan aktivitas kegiatan sehari hari.

Simple Present Tense:

Kalimat Positive (+)  :

- Verbal     :  Subject + Verb1/Verb1+s/es + Object + adverb of time (every day / every morning)

- Nominal  :  Subject + Tobe (am, is are) + Object

Kalimat Negatif (-)    :

- Verbal     :  Subject + do/does not +Verb1 + Object + adverb of time (every day / every morning)

- Nominal  :  Subject + Tobe (am, is are) +NOT + Object

Kalimat Interrogative (?)  :

- Verbal    :  Do/Does + Subject + Verb1 + Object + adverb of time (every day / every morning) + ?

- Nominal :  Tobe (am, is are) + Subject + Object + ?

Jawaban di atas mengikuti Formula kalimat Verbal dan untuk bentuk kalimat verbal negatif Auxiliary Verbs seperti kata Do not dan Does not dalam bentuk negatif bisa disingkat menjadi bentuk Contraction forms yaitu: Don't dan Doesn't.

Kalimat Nominal contoh lainnya:

(+) We are doctors.

(-) We are not doctors.

(?) Are we doctors?

Dalam kalimat nominal, bentuk kata kerja To be (is / am / are) dalam bentuk negatif is not dan are not juga bisa disingkat menjadi bentuk Contraction forms yaitu: isn't dan aren't namun am not tidak bisa disingkat.

Simple Past Tense:

Kalimat Verbal Positif:

(+) Subject + verb 2 + object + adverb of time (yesterday / last week)

(+) We attended the festival last week

Kalimat Verbal Negatif:

(-) Subject + did not + Verb 1 + object + adverb of time (yesterday / last week)

(?) Did + subject + Verb 1 + object + adverb of time (yesterday / last week)?

PEMBAHASAN

Simple past tense adalah sebuah kata kerja yang digunakan untuk berbicara tentang aktivitas masa lalu. Simple Past sendiri biasanya menggunakan past verb/verb 2. Keterangan waktu yang biasa digunakan yaitu last week, last night, last month, last year, yesterday dsb.

Rumus simple past tense :

Verbal setence

(+) S + Verb₂ + o/c

(-) S + did not/didn’t + +Verb₁ + o/c

(?) Did + S + Verb₁ + o/c ?

Nominal setence

(+) S + to be (was/were) + Adj/Adv/Noun

(-) S + to be (was/were) + not + Adj/Adv/Noun

(?) To be (was/were) + S + Adj/Adv/Noun
