11. 4² × 3²=.......... Ramaikan​


PENDAHULUAN

• Pangkat adalah suatu perkalian berulang-ulang suatu bilangan sesuai dengan pangkatnya. Misalnya : 2³ = 2×2×2 3³ = 3×3×3

• Perkalian merupakan proses aritmatika dasar di mana satu bilangan dilipatgandakan sesuai dengan bilangan pengalinya.

≈ Contoh Perpangkatan 2

• 5² = 5×5

• 4² = 4×4

• 3² = 3×3

≈ Contoh Perpangkatan 3

• 5³ = 5×5×5

• 4³ = 4×4×4

• 3³ = 3×3×3

PEMBAHASAN

• 4² × 3²

= ( 4×4 ) × ( 3×3 )

= ( 5×3+1 ) × 9

= 12 × 12

= 16 × 9

= 144

Kesimpulan

Mak, hasil dari 4² + 3² = 144

2² + 3² + 4²

= ( 2×2 ) + ( 3×3 ) + ( 4×4 )

= ( 2×2 ) + 9 + 16

= 4 + 9 + 16

= 20 + 9

= 29
