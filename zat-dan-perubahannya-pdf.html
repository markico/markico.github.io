6. untuk mengubah file dalam format pdf publish. dan mengirim file melalui email menggunakan​


Excel, PowerPoint, Project, Publisher, Visio, dan Word

1. Klik File > Bagikan > Email, lalu pilih salah satu opsi berikut ini:

Kirim sebagai Lampiran: Membuka pesan email dengan salinan file terlampir dalam format file aslinya.

Kirim sebagai PDF: Membuka pesan email dengan salinan file terlampir dalam format .pdf.

Kirim sebagai XPS: Membuka pesan email dengan salinan file terlampir dalam format .xps.

Kirim sebagai Faks Internet: Membuka halaman web di mana Anda bisa memilih dari daftar penyedia yang memungkinkan Anda mengirim faks melalui internet.

2. Masukkan nama alias penerima, edit baris subjek dan isi pesan seperlunya, kemudian klik Kirim.

OneNote

1. Klik File > Kirim, lalu pilih salah satu opsi berikut ini:

Kirim sebagai Lampiran: Membuka pesan email dengan lampiran salinan file dalam format file aslinya dan salinan file sebagai halaman web.

Kirim sebagai PDF: Membuka pesan email dengan salinan lampiran file dalam format .pdf.

2. Masukkan nama alias penerima, edit baris subjek dan isi pesan seperlunya, kemudian klik Kirim.

Mengirim sebagai isi pesan email

Jika menggunakan Excel, Publisher, atau Word, Anda dapat mengirim file sebagai konten pesan email sendiri, bukan sebagai lampiran. Untuk melakukannya, tambahkan perintah Kirim ke Penerima Email terlebih dahulu ke Bilah Alat Akses Cepat. Lihat Mengustomisasi Bilah Alat Akses Cepat untuk mempelajari cara menambahkan perintah.

1. Buka file yang akan dikirim.

2. Dalam Bilah Alat Akses Cepat, klik Kirim ke Penerima Email untuk membuka pesan email. File Anda akan ditampilkan dalam isi pesan.

3. Masukkan nama alias penerima, edit baris subjek dan isi pesan seperlunya, kemudian klik Kirim.

23. Bagaimana cara mengubah file pdf ke MC Word secara online dan tanpa menggunakan aplikasi?


Cara mengubah file PDF ke Ms. Word secara online:

1. akses situs https://online2pdf.com/pdf2word

2. klik tombol select files

selanjutnya pilih file PDF yang ingin diubah ke Ms. Word

3. klik tombol Convert

tunggu proses perubahannya sampai selesai (100 %)

-bila sudah selesai, silakan download file yang sudah berubah menjadi Ms. Word (perhatikan langkahnya sebagai berikut/ pada tanda panah)-

30. apa fungsi dari .Pdf
