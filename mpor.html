2. Diketahui data berikut: Pengeluaran konsumsi : Rp30.000.000.000,00 nvestasi pengusaha : Rp20.000.000.000,00 Ekspor: Rp17.000.000.000,00 mpor: Rp7.000.000.000,00 Pengeluaran Pemerintah: Rp15.000.000.000,00 Maka, berapa besarnya pendapatan nasional jika dihitung dengan pendekatan pengeluaran?​


Jawaban:

Pendapatan nasional dapat dihitung dengan pendekatan pengeluaran dengan menjumlahkan total pengeluaran konsumsi (C), investasi pengusaha (I), ekspor (X), dan pengeluaran pemerintah (G).

Total pengeluaran = C + I + X + G

Diketahui:

Pengeluaran konsumsi (C) = Rp30.000.000.000,00

Investasi pengusaha (I) = Rp20.000.000.000,00

Ekspor (X) = Rp17.000.000.000,00

Impor (M) = Rp7.000.000.000,00

Pengeluaran pemerintah (G) = Rp15.000.000.000,00

Total pengeluaran = 30.000.000.000 + 20.000.000.000 + 17.000.000.000 - 7.000.000.000 + 15.000.000.000

Total pengeluaran = 75.000.000.000

Jadi, besarnya pendapatan nasional jika dihitung dengan pendekatan pengeluaran adalah Rp75.000.000.000,00.

Pendekatan Pengeluaran

Y = C + I + G + ( X - M )

Y = Rp30.000.000.000,00 + Rp20.000.000.000,00 + Rp15.000.000.000,00 + ( Rp17.000.000.000,00 - Rp7.000.000.000,00 )

Y = Rp75.000.000.000,00

Pembahasan

Metode Perhitungan Pendapatan Nasional :

1. Pendekatan Produksi

Rumus :

Y = ( Q1 × P1 ) + ( Q2 × P2 ) + .... ( Qn × Pn )

2. Pendekatan Pengeluaran

Rumus :

Y = C + I + G + ( X - M )

Ket :

Y = pendapatan nasionalC = konsumsi rumah tanggaI = investasiG = pengeluaran pemerintahX = eksporM = impor

3. Pendekatan Pendapatan

Rumus :

Y = w + r + i + p

Ket :

w = upah tenaga kerjar = sewa tanahi = bunga modalp = laba pengusaha
