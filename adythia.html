Penjelasan dengan langkah-langkah:

Diketahui:
--> harga awal = 1.750k
--> harga akhir = 1.675 k
Dicari persen diskon (%)
-----------------------------------------------
Potongan =
Harga awal - akhir =
1.750k - 1.675k =
(1.750-1.675)k =
Potongan = 75k
-----------------------------------------------
Jika
potongan = harga awal×(%)/100
harga awal×(%) = 100×potongan
----------------------------------------------
cari (%) persen diskon, dengan
Rumus
% = (100×potongan) ÷ harga awal
% = (100×75k) ÷ 1.750k
% = 7.500k/1.750k
% ≈ 4,29
Diskon ≈ 4,29..%

25. Aditia menabung di bank dengan suku bunga 3% per bulan , selama 10 bulan tabungan aditia menjadi Rp 975.000,00. Tabungan awal aditia sebesar... A. 800rb B. 750 rb C. 500rb D.450rb


Penjelasan dengan langkah-langkah:

Jumlah tabungan akhir = 975.000

Tabungan awal x (1 + suku bunga x waktu) = 975.000

Tabungan awal x (1 + 3/100 x 10) = 975.000

Tabungan awal x (1 + 0,3) = 975.000

Tabungan awal x 1,3 = 975.000

Tabungan awal = 975.000/1,3

Tabungan awal = 750.000

Tabungan awal = Rp 750.000,-

Jawabannya B

Tabungan awal=T.aw

[tex]t.aw \times (1 \times sb \times waktu) = t.ak \\ t.aw \times (1 \times \frac{3}{100} \times 10) = 975.000 \\ t .aw \times (1 \times 0.3) = 975.000 \\ t.aw \times 1.35 = 975.000 \\ t.aw = \frac{975.000}{1.35} \\ t.aw = 750.000[/tex]
