klorofil adalah zat hijau daun. zat hijau daun diperlukan untuk membantu dalam proses pembuatan makanan pada daun saat fotosintesis. jadi, tanpa klorofil, tumbuhan tidak akan diasupi oleh nutrisi yang baik. karena itu pada umumnya tumbuhan memiliki klorofil

klorofil/ zat hijau daun diperlukan oleh tanaman, hal ini disebabkan oleh cara tanaman mendapatkan makanan melalui fotosintesis. zat hijau daun / klorofil inilah yg membantu tanaman untuk berfotosintesis dg bantuan cahaya matahari.

9. Kegunaan PPTPD sever


1.Koneksi PPTP VPN

PPTP menjadi pilihan populer jaringan virtual private network karena kemudahan dalam setting maupun penggunaan. Kamu tidak perlu melakukan instalasi VPN client agar bisa terhubung dengan jaringan server PPTP VPN. Koneksi Point to Point Tunneling Protocol bisa berjalan di berbagai sistem operasi seperti Microsoft Windows, Linux, MacOS X, iPhone OS, dan Android OS sudah memiliki built in protokol ini.

Kamu hanya membutuhkan nama server, vpn account meliputi nama pengguna dan kata sandi untuk dapat terhubung dengan server PPTP VPN. Koneksi jenis ini paling banyak digunakan pada area wifi hotspot untuk melindungi privasi online dan mencegah pihak lain yang ingin meretas perangkat smartphone atau tablet yang kamu gunakan di area hotspot.

memposting setitik ilmu dengan Materi baru yaitu PPTP. Apa itu PPTP? Point To Point Tunneling Protocol adalah salah satu protocol untuk membuat VPN (Virtual Private Network) . Sebuah koneksi PPTP terdiri dari Server dan Client, Mikrotik dapat difungsikan sebagai server maupun client bahkan dalam satu mesin, fungsi PPTP client juga sudah ada disemua OS, kita bisa gunain di PC/Laptop jadinya. dan satu lagi PPTP membentuk tunnel antar Ip menggunakan protocol TCP dan GRE (Generic Routing Encapsulation), dan menggunakan port 1723

Name : username yang akan digunakan client

Remote Address : Ip yg digunakan client
