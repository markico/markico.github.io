jadi 16 × 8 = 128 cm

erwin = 16 × 8 cm = 128 cm

miopi

keadaan 1
P = - 100 / PR
-⅔ = - 100 / PR

keadaan 2
P' = - 100 / PR'
-1 = - 100 / PR'
PR' = 100 cm

∆PR = 100 - 150 = - 50 cm
jawaban [C]P = 1 / -PR → PR = -1 / P
P' = 1 / -PR' → PR' = -1 / P'

ΔPR = PR' - PR
         =  -1 / P' + 1 / P
         = -1 / (-1 D) + 1 / (-2/3 D) = -0,5 m = -50 cm
