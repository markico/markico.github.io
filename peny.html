29. 1.Tentukan Himp peny dari x kuadrat + 3x -54= 0 2.tentukan Himp Peny dari 3x kuadrat + x -10 = 0


x^2 + 3x - 54 = 0

x^2 + 3x = 54

x^2 + 3x + (3/2)^2 = 54 + (3/2)^2

(x + 3/2)^2 = 54 + 9/4

(x + 3/2)^2 = 216/4 + 9/4

(x + 3/2)^2 = 225/4

x + 3/2 = +- V(225/4)

x + 3/2 = +- 15/2

x = 15/2 - 3/2 atau x = -15/2 - 3/2

x = 12/2 atau x = -18/2

x = 6 atau x = -9

jadi HP nya adalah {-9 , 6}

2]

3x^2 + x - 10 = 0

3x^2 + x = 10 (× 1/3)

x^2 + 1/3 x = 10/3

x^2 + 1/3 x + (1/6)^2 = 10/3 + (1/6)^2

(x + 1/6)^2 = 10/3 + 1/36

(x + 1/6)^2 = 120/36 + 1/36

(x + 1/6)^2 = 121/36

x + 1/6 = +- V(121/36)

x + 1/6 = +- 11/6

x = 11/6 - 1/6 atau x = -11/6 - 1/6

x = 10/6 atau x = -12/6

x = 5/3 atau x = -2

jadi HP nya adalah {-2 , 5/3}
