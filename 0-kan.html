[tex]2) \: {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

[tex]3) \: {a}^{m} \div {a}^{n} = {a}^{m - n} [/tex]

[tex]4) \: ( {a}^{m} ) ^{n} = {a}^{m \times n} [/tex]

[tex]5) \: ({a \times b})^{m} = {a}^{m} \times {b}^{m} [/tex]

[tex]6) \: ( {a \div b})^{m} = {a}^{m} \div {b}^{m} [/tex]

[tex]7) \: {a}^{m} = \frac{1}{ {a}^{ - m} } [/tex]

[tex]8) \: {a}^{ - m} = \frac{1}{ {a}^{m} } [/tex]

[tex]9) \: {a}^{ \frac{m}{n} } = \sqrt[n]{ {a}^{m} } [/tex]

~Angka 0 ( Nol )

____________________

Pembahasan

Hasil dari

0 + 0 = 00 - 0 = 00 × 0 = 0

Hasil tersebut demikian karena 0 adalah angka yang memiliki sifat netral atau jika dijabarkan tidak positif maupun negatif, Angka 0 disebut juga sebagai perbatasan angka negatif dan angka positif oleh karena itu jika dijumlahkan, dikurangi, dan dikali hasilnya akan tetap 0, sama halnya dengan notasi tak hingga " ∞ " dimana jika dijumlahkan, dikurangi, maupun dikali, hasilnya akan tetap sama yakni ∞.

Berbeda dengan 0 ÷ 0 = Tak terdefinisi atau Tak tentu, 0 ÷ 0 atau 0/0.

Misalnya saja sifat-sifat bilangan terhadap 0 berbeda-beda, Seperti berikut ini

• 0 × n = 0, Maka 0 × 0 = 0 (Untuk n bilangan real atau bulat)

• 0 ÷ n = 0, Maka 0 ÷ 0 = 0 (Untuk n bilangan real atau bulat)

• n/0 = ∞, Maka 0/0 = ∞ (Untuk n bilangan real atau bulat)

• n ÷ n = 1, Maka 0 ÷ 0 = 1 (Untuk n bilangan real atau bulat)

Karena hal tersebut banyak berspekulasi jika sifat 0 terhadap pembagian bervariasi 0, 1, ∞ sehingga diambil kesimpulan bahwa 0 ÷ 0 atau sifat 0 terhadap pembagian adalah Tak Tentu atau Tak Terdefinisi (Indeterminate)
