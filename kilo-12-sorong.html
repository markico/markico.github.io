8x + 6y = 94.000 (x1) > 8x + 6y = 94.000

y = 5000 (harga 1 kilo mangga)

4x + 12y = 92.000
4x + 12(5000) = 92.000
4x + 60.000 = 92.000
4x = 92.000-60.000
4x = 32.000
x = 8.000 (harga 1 kilo apel)

harga 5 kilo apel dan 3 kilo mangga

= 40.000 + 15.000
