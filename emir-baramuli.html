14. Perbandingan usia Emir : Naufal : Hisyam adalah 2 : 3 : 5. Jika jumlah usia Emir dan Naufal adalah 35 tahun, selisih usia Hisyam dan Naufal adalah …. ​


Jawab:

Selisih usia Hisyam dan Naufal adalah 14 tahun.

Perbandingan ditentukan dengan menghitung bentuk paling sederhana/kecil dari dua bilangan atau lebih.

Perbandingan dibagi menjadi dua jenis, yaitu:

1. Perbandingan senilai

Apabila sebuah variabel diperbesar maka nilainya variabel turunanya akan bertambah besar. Begitu juga sebaliknya.

Rumus: X1 : Y1 = X2 : Y2

2. Perbandingan berbalik nilai

Apabila sebuah variabel diperbesar maka nilai variabel turunannya bertambah kecil .

Rumus: X1 : Y1 = Y2 : X2

Ingat!

Prinsip menyederhanakan perbandingan sama seperti menyederhanakan bilangan pecahan. Yaitu dengan cara menentukan nilai FPB kedua bilangan.  Operasi penjumlahan dan pengurangan dapat dihitung secara langsung.

Penjelasan dengan langkah-langkah:

Diketahui:

Misal: usia Emir = e, usia Naufal = n, usia Hisyam = h

e : n : h = 2 : 3 : 5

e + n = 35

Ditanya:

h - n = ?

Jawab:

n : (e + n) = 3 : (2 + 3)

n : 35 = 3 : 5

n = 3/5 x 35

n = 21 tahun

n : (h - n) = 3 : (5 - 3)

21 : (h - n) = 3 : 2

3 x (h - n) = 2 x 21

(h - n) = 2/3 x 21

(h - n) = 14 tahun

Jadi selisih usia Hisyam dan Naufal adalah 14 tahun.
