10. aman membeli 2 buras 1 basreng dan 1 cilok dengan membayar Rp4700, cinta membeli 1 buras 2 basreng dan cilok dengan membayar Rp4300, Damai membeli 3 buras 2 basreng dan 1 cilok dengan membayar Rp7100, berapakah masing masing harga buras basreng dan cilok tersebut


Jawaban:

harga buras = Rp. 1400

harga basreng = Rp. 1000

harga cilok = Rp. 900

Penjelasan dengan langkah-langkah:

2 buras + 1 basreng + 1 cilok = Rp. 4700

1 buras + 2 basreng + cilok = Rp. 4300

3 buras + 2 basreng + 1 cilok = Rp. 7100

misal : buras = x

basreng = y

cilok = z

persamaan (1) : 2x + y + z = 4700

persamaan (2) : x + 2y + z = 4300

persamaan (3) : 3x +2y + z = 7100

menggunakan persamaan (1) dan (2)

2x + y + z = 4700

x + 2y + z = 4300

—————————— -

x - y = 400 ...(4)

mencari x menggunakan persamaan (3) dan (2)

3x + 2y+ z = 7100

x + 2y + z = 4300

—————————— -

2x = 2800

x = 1400

mencari y menggunakan persamaan (4)

x - y = 400

1400 - y = 400

y = 1400 - 400

y = 1000

mencari z menggunakan persamaan (2)

x + 2y + z = 4300

1400 + (2×1000) + z = 4300

1400 + 2000 + z = 4300

3400 + z = 4300

z = 4300 - 3400

z = 900

jadi, harga buras = Rp. 1400, harga basreng = Rp. 1000, dan harga cilok = Rp. 900.
