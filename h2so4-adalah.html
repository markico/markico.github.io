2. Diketahui : 1,806 × 10^20 molekul H2SO4 Ditanya : .......mol H2SO4 =...... gram H2SO4 ?


JUMLAH PARTIKEL = n x L
1,806 x 10^20 : 6,02 x 10^23 = n
0,3 x 10^-3 = n
0,0003 = n


massa = n x M
gr = 0,0003 x 98
gr = 0.0294 gr

Stoikiometri
Kimia X

Diketahui
Jumlah molekul H₂SO₄ = 1,806 x 10²⁰

Ditanyakan jumlah mol dan massa

Penyelesaian

[a]
Jumlah mol
n = [jumlah molekul] / [bilangan Avogadro]
n = [1,806 x 10²⁰] / [6,02 x 10²³]

Jadi n = 0,3 x 10⁻³ atau 3 x 10⁻⁴ mol

[b]
Massa (gram)
Mr H₂SO₄ = 2(1) + 32 + 4(16) = 98
Massa = mol x Mr
Massa = 3 x 10⁻⁴ x 98

Jadi massa = 0,0294 gram atau 29,4 mg

9. 500 mL larutan H2SO4 mempunyai pH =2 jika Mr H2SO4 = 98 maka tentukan banyaknya H2SO4 yang dilarutkan


Diketahui : H2SO4 (Mr = 98 gr/mol), V = 500 mL = 0.5 Lt
                   pH = 2
Ditanya : massa H2SO4 ?
Jawab  :               
                 [H+] = a * Ma
                 pH = - log [H+]
                 2 = - log [H+]
                log 2 = - log [H+]
                 log -2 = log [H+] ----------[H+] = 0.01
 
                a untuk H2SO4 = 2 ------------ 0.01 = 2 . Ma--- Ma = 0.005 mol /Lt
                 untuk 0.5 Lt ----------- mol H2SO4 = 0.005 mol Lt * 0.5 Lt
                                                                        = 0.0025 mol
 
               Jadi massa H2SO4 = 0.0025 mol * 98 gr/mol = 0.245 gram
 









10. 500 ml larutan H2SO4 mempunyai pH=2 jika Mr H2SO4 =98 Maka banyaknya H2SO4 yang terlarut adalah


diketahui :
H₂SO₄ (Asam Sulfat)
V H₂SO₄ = 500 mL = 0,5 L
pH = 2
Mr = 98

ditanya :
massa H₂SO₄  = ...?

pH = - log [H⁺] = 2
[H⁺] = 10^{- pH} = 10⁻²

[H⁺] = valensi × [H₂SO₄]
10⁻² = 2 × [H₂SO₄]
[H₂SO₄] = 5 × 10⁻³ M

n = [H₂SO₄] × V
   = 5 × 10⁻³ × 0,5
   = 2,5 × 10⁻³ mol

massa = Mr × n
            = 98 × 2,5 × 10⁻³
            = 0,245 gram = 245 mg

Jadi banyaknya H₂SO₄ yang terlarut adalah 245 mg

13. Berapa gram H2SO4 yang terlarut dalam 200ml larutan H2SO4 0,1molar


Jawaban

Mr H₂SO₄ = 98

M H₂SO₄ = [tex]\frac{a}{Mr}[/tex] x [tex]\frac{1000}{mL}[/tex]

0,1 = [tex]\frac{a}{98}[/tex] x [tex]\frac{1000}{200}[/tex]

0,1 = [tex]\frac{a}{98}[/tex] x 5

massa H₂SO₄ = [tex]\frac{0,1~x~98}{5}[/tex]

massa H₂SO₄ = 1,96 gram

Pembahasan

Molaritas / Konsentrasi

Konsentrasi atau molaritas adalah banyaknya mol zat terlarut dalam satu liter atau 1000 mL larutan.

[tex]\boxed{\boxed{\bold{M~=~\frac{n}{V}}}}[/tex]

Atau

[tex]\boxed{\boxed{\bold{M~=~\frac{a}{Mr}~x~\frac{1000}{mL}}}}[/tex]

Atau

[tex]\boxed{\boxed{\bold{M~=~\frac{10~x~\%~x~p}{Mr}}}}[/tex]

Dengan            :  

M = Kemolaran/molaritas larutan  

v = volume larutan (liter)

n = jumlah mol zat terlarut                                

a = massa zat terlarut (gram)

p = massa jenis

M =  p x % berat x 10 / Mr
M = 1/3 x 49 x 10 / 98 = 6,5 molar
M₁ x V₁ = M₂ x V₂
6,5 x V₁ = 0,05 x 260
        V₁ = 2 mL

[tex]M_2=0,05~M \\ V_2=260~ml \\ \\ M_1= \frac{10\times \rho \times \%}{Mr~H_2SO_4} = \frac{10\times 1,3 \times 49}{98}=6,5 ~M\\ \\ M_1\times V_1=M_2\times V_2 \\ \\ 6,5\times~V_1=0,05\times 260 \\ \\ V_1= \frac{0,05\times 260}{6,5} \\ \\ V_1=2~ml[/tex]
