28. Ketua PMR sedang membeli dua jenis obat yaitu obat diare dan obat maag. harga 4 bungkus obat diare dan 2 bungkus obat maag adalah Rp 28.000. Sedangkan harga 8 bungkus obat diare dan 3 bungkus obat maag adalah Rp 50.000. Harga masing-masing 1 bungkus obat diare dan 1 bungkus obat maag berturut-turut adalah ....


Pemisalan :

Obat diare : x

Obat maag : y

4x + 2y = 28.000 ... (pers. 1)

8x + 3y = 50.000 ... (pers. 2)

Uraikan pers. 1

4x + 2y = 28.000

4x = 28.000 - 2y

x = 7.000 - y/2

Substitusi x = 7.000 - y/2 ke pers. 2

8x + 3y = 50.000

8(7.000 - y/2) + 3y = 50.000

56.000 - 4y + 3y = 50.000

6.000 = y

y = 6.000

Obat maag = Rp. 6.000

Substitusi y = 6.000 ke x = 7.000 - y/2

x = 7.000 - y/2

x = 7.000 - 6.000/2

x = 4.000

Obat diare = Rp. 4.000

Jadi, harga 1 bungkus obat diare dan 1 bungkus obat maag berturut-turut adalah Rp. 4.000 dan Rp. 6.000.

pH=11 -> pOH=3. Maka OH-=10^-3 [OH-]=akar (Kb.M). Kb= [OH-]^2/M Kb= (10^-3)^2/0,01 Kb= 10^-4Diketahui :
Molaritas Larutan basa organik(Obat) = 0,01 M
pH = 11

Ditanya : Nilai Kb Obat ?

Jawab :

pOH :
= 14 - pH
= 14 - 11
= 3

[OH-] :
- log [OH-] = pOH
- log [OH-] = 3
- log [OH-] = - log 10-³
[OH-] = 10-³ M

Kb :
= [OH-]² / M
= [10-³ M]² / 0,01 M
= 10^-6 M² / 0,01 M
= 10-⁴ M
