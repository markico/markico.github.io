20. berapa hasil dari 4 pangkat seperdua​


Hasil dari [tex] {4}^{ \frac{1}{2} } [/tex] adalah 2.

Pembahasan

Sebelum menjawab pertanyaan tersebut, mari kita bahas sedikit bilangan berpangkat. Bilangan berpangkat adalah bentuk sederhana dari perkalian berulang.

2³

bentuk 2³ disebut bilangan berpangkat

3 disebut pangkat atau eksponen.

2 disebut basis atau bilangan pokok

Berikut beberapa jenis bilangan berpangkat.

Bilangan Berpangkat Bulat Positif

Perhatikan perpangkatan bilangan a berikut.

[tex] a = {a}^{1} [/tex]

[tex] a × a = {a}^{2} [/tex]

[tex] a × a × a = {a}^{3} [/tex]

Secara umum, bentuk perpangkatan bilangan bulat positif dapat dituliskan sebagai berikut.

[tex] {a}^{n} = a × a × ... × a (n faktor) [/tex]

Keterangan: n = bilangan bulat positif

Bilangan Berpangkat Bulat Negatif

Secara umum bilangan berpangkat bulat negatif dapat ditulis dalam bentuk bilangan berpangkat bulat positif.

[tex] {a}^{-m} = \frac{1}{{a}^{m}} [/tex]

Sebaliknya, bilangan berpangkat bulat positif dapat juga diubah menjadi bilangan berpangkat negatif. Bentuk umumnya sebagai berikut.

[tex] {a}^{m} = \frac{1}{{a}^{-m}} [/tex]

Bilangan Berpangkat Nol

Setiap bilangan yang dipangkatkan nol hasilnya adalah satu. Simak penjelasan berikut.

[tex] \frac{{3}^{2}}{{3}^{2}} = \frac{3 × 3}{3 × 3} = \frac{9}{9} = 1 [/tex]

Berdasarkan penjelasan di atas dapat disimpulkan sebagai berikut.

Untuk sembarang a ∈ bilangan real dengan a ≠ 0, maka [tex] {a}^{0} = 1 [/tex]

Sementara itu, apabila a = 0 maka [tex] {a}^{0} [/tex] tidak didefinisikan.

Bilangan Berpangkat Pecahan

Secara umum bentuk tersebut dapat dituliskan:

[tex] \sqrt[n]{b} = a \: ⇔ \: {a}^{n} = b \: dan \: {a}^{n} = b ⇔ a = {b}^{ \frac{1}{n} } [/tex]

Soal diatas merupakan contoh bentuk bilangan berpangkat pecahan. Sesuai dengan pembahasan diatas, kita dapat mengerjakannya sebagai berikut.

Pertama-tama, kita ubah [tex] {4}^{ \frac{1}{2} } [/tex] menjadi bentuk akar.

[tex] {4}^{ \frac{1}{2} } = \sqrt{4} [/tex]

Setelah diubah, maka kita dapat menghitung hasil akhirnya.

[tex] \sqrt{4} = 2 [/tex]

Cara lain:

[tex] {4}^{ \frac{1}{2} } = ( {2}^{2} )^{ \frac{1}{2} } = {2}^{2 \times \frac{1}{2} } = {2}^{ \frac{2}{2} } = 2[/tex]

Jadi, hasil dari [tex] {4}^{ \frac{1}{2} } [/tex] adalah 2.

Dalam Persen

[tex] = \frac{1}{2} 0,05\%[/tex]

[tex] = ( \frac{1}{40} )\%[/tex]

Dalam Pecahan

[tex] = \frac{1}{2} 0,05\%[/tex]

[tex] = \frac{1}{2} \times \frac{5}{100}[/tex]

[tex] = \frac{1 \times 5}{2 \times 100} [/tex]

[tex] = \frac{5}{200} [/tex]

Sederhanakan

[tex] = \frac{5÷5}{200÷5} [/tex]

[tex] = \frac{1}{40} [/tex]

Dalam Desimal

[tex] = \frac{1}{2} 0,05\%[/tex]

[tex] = \frac{1}{2} \times \frac{5}{100}[/tex]

[tex] = 0,5 \times 0,05[/tex]

[tex] = 0,025 [/tex]
