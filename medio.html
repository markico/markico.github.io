16. Apa perbedaan data yang diolah dengan database dengan data yang diolah dengan spreadsheet?


Jawaban:

Efektivitas, batasan, isi data.

Uraian:

1. Efektivitas : Database lebih efektif karena data di dalamnya dapat dengan mudah dibandingkan dan dianalisis dengan fitur-fitur otomasi kueri dan laporan sesuai dengan prosedur yang diberikan (Sudah ada di sebagian besar aplikasi pengolah database) sedangkan spreadsheet cenderung hanya melakukan filter saja sehingga sulit dan lama untuk menganalisis data.

2. Batasan : Database memiliki kemampuan untuk menangani jutaan data dengan efisien sedangkan spreadsheet cenderung memiliki batasan, misalnya Microsoft Excel hanya dapat mencapai 1,048,576 baris dan 16,384 kolom. Memang suda jumlah besar, namun apabila untuk data, tentu database akan lebih cocok.

3. Isi data : Database memiliki keahlian lebih dalam mengelola berbagai bentuk, baik teks maupun media visual seperti gambar. Spreadsheet juga dapat menyertakan gambar namun justru kurang cocok bahkan sulit untuk mengelola media visual tersebut dan cenderung hanya digunakan untuk angka-angka.
