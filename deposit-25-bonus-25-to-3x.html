28. Quiz [ Bonus ]1) ⅔ - ⅓ + ⅝ = ....2) 25% - ⅛ + ¾ = ....​


Penjelasan dengan langkah-langkah:

No.1

2/3 - 1/3 + 5/8

= 1/3 + 5/8

= 8/24 + 15/24

= 23/24

No.2

25% - 1/8 + 3/4

= 25/100 - 12,5/100 + 75/100

= 87,5/100

= 0,875

Penjelasan dengan langkah-langkah:

[tex]1.) \frac{2}{3} - \frac{1}{3} + \frac{5}{8} = \frac{1}{3} + \frac{5}{8} = \frac{15}{24} + \frac{8}{24} = \frac{23}{24} [/tex]

[tex]2.)25\% - \frac{1}{8} + \frac{3}{4} \\ \frac{25}{100} - \frac{1}{8} + \frac{3}{4} = \frac{1}{8} + \frac{3}{4} = \frac{4}{8} + \frac{3}{8} = \frac{7}{8} [/tex]
