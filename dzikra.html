20. Uang dzIkra 1/4 uang zaskia. Uang zaskia 6/4 uang rani. Jika jumlah uang mereka rp.184.000 maka jumlah uang mereka masing2?


Penjelasan dengan langkah-langkah:

d = 1/4 z

4d = z

4d - z = 0...(1)

8d - 2z = 0

z = 6/4 r

4z = 6r

2z = 3r

2z - 3r = 0...(2)

8d - 2z = 0...(1)

---------------- +

8d - 3r = 0...(3)

d + z + r = 184.000...(4)

4d - z = 0...(1)

--------------------------- +

5d + r = 184.000 x 3

15d + 3r = 552.000

8d - 3r = 0...(3)

---------------------------- +

23d = 552.000

d = 24.000

4d - z = 0...(1)

z = 96.000

d + z + r = 184.000...(4)

r = 184.000 - 24.000 - 96.000

r = 64.000
