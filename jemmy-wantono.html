= 23

Jumlah anggur Dibagi dengan Jumlah hari ( 7 hari ) .

= Senin + Selasa + rabu + kamis + jumat + sabtu + minggu : 7

= ( 25 + 20 + 15 + 23 + 22 + 26 + 30 ) : 7 hari

= 161 : 7

= 23
