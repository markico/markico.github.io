14. Hari minggu jumlah uang tora dan ani berbanding 3:1 pada hari senin tora memberikan uang sejumah rp. 50.000,00 kepada ani.sekarang perbandingan uang tora dan ani 1:2. Berapa jumlah uang tora dan ani pada hari minggu adalah


Jawaban:

A = jumlah uang Ani pada Hari Minggu

T = jumlah uang Tora pada Hari Minggu

Maka, pada hari minggu, T : A = 3 : 1 ⇔ T = 3A

Sehingga, pada Hari senin:

Uang Tora = T - 50.000

Uang Ani = A + 50.000

Perbandingan uang Tora dan Ani = 1 : 2

sehingga, (T - 50.000) : (A + 50.000) = 1 : 2

⇔ 2T - 100.000 = A + 50.000

⇔ 2T - A = 150.000

Karena T = 3A, maka ketika disubsitusikan ke 2T - A = 150.000 menjadi:

2 (3A) - A = 6A - A = 5A = 150.00

sehingga, dapat disimpulkan bahwa:

A = 30.000

T = 90.000

Jadi, jumlah uang Ani dan Tora pada hari Minggu adalah Rp. 120.000,00

Pada hari minggu kuturut ayah ke kota ~ Eh, pada nyanyi. Oke, ane bakal jawab pertanyaannya.

Misalkan uang T = Uang Tora dan A = Uang Ani. Perbandingannya pada hari minggu adalah 3 : 1, maka
T : A = 3 : 1 atauh T = 3A

Sekarang hari senin, Tora memberikan uang 50000 kepada Ani. Kenapa Tora memberikan uang kepada Ani? Hmm, soal ini penuh teka-teki. Akibat dari perlakuan yang mencurigakan Tora, perbandingannya menjadi 1 : 2. artinya
(T - 50000) : (A + 50000) = 1 : 2
atau
2(T - 50000) = (A + 50000)
2T - 100000 = A + 50000
2T = A + 150000
Karena T = 3A maka
2(3A) = A + 150000
6A = A + 150000
5A = 150000
A = 30000

T = 3A

28. Perbandingan banyak kelereng Tora dan Yori adalah 6 : 5, sedangkan perbandingan banyak kelereng Tora dan Abi adalah 3 : 2. Jika jumlah kelereng Yori dan Abi adalah 243, maka selisih banyak kelereng Tora dan Abi adalah


Jawaban:

liat bawah

Penjelasan dengan langkah-langkah:

3/2 × 243 = 364,5

6/5 × 243 = 291,6

364,5 - 291,6 = 72,9

Penjelasan dengan langkah-langkah:

Diketahui :

Perbandingan banyak kelereng Tora dan Yori = 6 : 5

Perbandingan banyak kelereng Tora dan Abi = 3 : 2

Jika jumlah kelereng Yori dan Abi = 243

Ditanyakan : Selisih banyak kelereng Tora dan Abi ..... ?

Jawab :

Dari pernyataan di atas kita tahu :

Perbandingan Kelereng Yori = 5

Perbandingan Kelereng Abi = 2

Jadi perbandingannya seperti ini :

Kelereng Yori :

5/7 × 243 = 173,57 dibulatkan menjadi 174 kelereng

Kelereng Abi :

2/7 × 243 = 69,42 dibulatkan menjadi 69 kelereng

Perbandingan Kelereng Tora dan Abi = 3 : 2

Kelereng Abi :

2/5 × Jumlah kelereng Tora dan Abi = 69 kelereng

Jumlah Kelereng Tora dan Abi = 172,5 dibulatkan menjadi 173

Banyak kelereng Tora = 3/5 × 173 = 103,8 dibulatkan menjadi 104 kelereng

Selisih Kelereng Tora dan Abi = Banyak Kelereng Tora - Banyak Kelereng Abi

= 104 - 69

= 35 kelereng

Jadi, selisih Kelereng Tora dan Abi adalah 35 kelereng
