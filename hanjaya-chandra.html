3. Uang Andri 2/3 uang Chandra dan uang Chandra 4/5 uang Burhan. Jika jumlah uang Burhan dan Chandra Rp540.000,00 maka selisih uang Andri dan Chandra adalah …​


A = Andri

B = Burhan

C = Chandra

A = 2/3 C

C = 4/5 B --> B = 5/4 C

B + C = 540.000

5/4 C + C = 540.000

9/4 C = 540.000

9 C = 2.160.000

C = 240.000 (uang Chandra)

A = 2/3 C = 2/3 x 240.000 = 160.000 (uang Andri)

Selisih uang Andri dan Chandra = 240.000 - 160.000 = 80.000

PKP

[tex]350.000.000 - 121.500.000 \\ =Rp \: 228.500.000[/tex]

PPh Terutang (tahun)

[tex]5\% \times 50.000.000 = 2.500.000 \\ 15\% \times 178.500.000 = 26.775.000[/tex]

TOTAL

[tex]Rp \: 29.275.000[/tex]
