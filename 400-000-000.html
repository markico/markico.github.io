21. Diketahui TFC: 1000. 000, AVC : 400. 000 dan harga barang 500. 000, Tentukan Q BEP!.


Jawaban:

jumlah produksi BEP adalah 10 unit

Penjelasan:

Untuk menentukan jumlah produksi break-even point (BEP), pertama-tama kita perlu menghitung biaya tetap rata-rata (ATC) yaitu dengan menjumlahkan biaya tetap (TFC) dengan biaya variabel rata-rata (AVC) dan membagi hasilnya dengan harga per unit. Dengan menggunakan data yang diberikan, ATC dapat dihitung sebagai berikut:

ATC = (TFC + AVC) / harga per unit

= (1000.000 + 400.000) / 500.000

= 1.400.000 / 500.000

= Rp 2.800,-

Setelah mengetahui ATC, selanjutnya kita dapat menggunakan rumus BEP sebagai berikut:

BEP (dalam unit) = TFC / (harga per unit - AVC)

= 1000.000 / (500.000 - 400.000)

= 1000.000 / 100.000

= 10 unit

Jadi, jumlah produksi BEP adalah 10 unit. Artinya, jika perusahaan menjual 10 unit produk tersebut, maka perusahaan akan tidak mengalami keuntungan atau kerugian (break-even). Semakin banyak produk yang dijual di atas jumlah BEP, maka semakin banyak keuntungan yang akan diperoleh perusahaan.

25. jika diketahui harta lancar rp 400 000 harta tetap 64 000 000 prive 100 000 laba bersih 350 000 modal awal 60 000 maka besar utangnya adalah. .?


Utang = Rp. 730.000,-

Pembahasan

Diketahui:

Harta lancar = Rp. 400.000,-Harta tetap = Rp. 640.000,-Prive = Rp. 100.000,-Laba = Rp. 350.000,-Modal awal = Rp. 60.000,-

Ditanyakan:

Utang...?

Jawab :

Menghitung modal akhir:

Modal akhir = modal awal + (laba - prive)

Modal akhir = 60.000 + (350.000 - 100.000)

Modal akhir = 60.000 + 250.000

Modal akhir = Rp. 310.000,-

Menghitung utang:

Aktiva = Pasiva

Harta = Utang + modal

(Harta lancar + harta tetap) = Utang + modal

(400.000 + 640.000) = Utang + 310.000

1.040.000 = Utang + 310.000

Utang = 1.040.000 - 310.000

Utang = Rp. 730.000,-
