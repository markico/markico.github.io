26. Find the last digit of(Carilah angka terakhir dari) 1^2015+2^2015+3^2015+...+10^2015​


Jawab:

5

Penjelasan dengan langkah-langkah:

Akan dicari digit terakhir dari ekspresi yang ada pada soal, sehingga kita akan bekerja pada [tex]\mathbb{Z}/10\mathbb{Z}[/tex] atau kelas kongruensi 10 untuk mendapatkan digit terakhir dalam radix-10

Triknya adalah menggunakan modulo 10 yang dapat mengubah nilai-nilai seperti berikut

[tex]\begin{aligned}6^n &\equiv (-4)^n \pmod{10}\\7^n &\equiv (-3)^n \pmod{10}\\8^n &\equiv (-2)^n \pmod{10}\\9^n &\equiv (-1)^n \pmod{10}\\\end{aligned}[/tex]

[tex]\begin{aligned}\sum_{i=1}^{10} i^{2015} &\equiv 1^{2015} + 2^{2015} + 3^{2015} + 4^{2015} + 5^{2015} +\\& 6^{2015} + 7^{2015} + 8^{2015} +9^{2015} + 10^{2015} &\pmod{10} \\& \equiv 1^{2015} + 2^{2015} + 3^{2015} + 4^{2015} + 5^{2015} +\\& (-4)^{2015} + (-3)^{2015} + (-2)^{2015} + (-1)^{2015} + 0^{2015} &\pmod{10} \\&\equiv 1^{2015} - 1^{2015} + 2^{2015} - 2^{2015} + 3^{2015} - 3^{2015} + \\& 4^{2015} - 4^{2015} + 5^{2015} &\pmod{10}\\&\equiv 5^{2015} &\pmod{10}\\& \equiv{5} &\pmod{10}\end{aligned}[/tex]

Maka dapat disimpulkan bahwa digit terakhir dari ekspresi yang ada pada soal adalah 5

Terlampir snippet python untuk sanity check dan jika ingin melihat brute force. Lampiran pertama memperlihatkan dengan modular exponentiation pow(a, b, c) python, lampiran kedua menggunakan BigInt python untuk bruteforce digit terakhir
