9. Dengan nominal uang saat ini adalah 50 Juta Rupiah. Berapa nominal uang di tahun ke 5 yang nilainya sama dengan nilai uang saat ini, bila Inflasi dianggap konstant di angka 5%?​


Jawab:

Rp60.775.312,50

Penjelasan dengan langkah-langkah:

Diketahui rumus harga setelah inflasi yakni: (Inflasi + 100%)/100% x Nominal Awal

Pada tahun pertama nominalnya adalah Rp50.000.000

Maka Rp50.000.000 sudah termasuk tahun ke-1

Maka pada tahun ke-2, nominalnya akan menjadi:

(5% + 100%)/100% x 50.000.000

= 105%/100% x 50.000.000

= 21/20 x 50.000.000

= 21 x 2.500.000

= 52.500.000

Maka pada tahun ke-3, nominalnya akan menjadi:

= 21/20 x 52.500.000

= 21 x 2.625.000

= 55.125.000‬

Maka pada tahun ke-4, nominalnya akan menjadi:

= 21/20 x 55.125.000‬

= 21 x 2.756.250‬

= 57.881.250‬

Maka pada tahun ke-5 (Sesuai permintaan soal), nominalnya akan menjadi:

= 21/20 x 57.881.250‬

= 21 x 2.894.062,5‬

= Rp60.775.312,50

Semoga terbantu :)
