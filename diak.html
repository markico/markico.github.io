[tex]{\displaystyle \gamma ={\frac {1}{\sqrt {1-{\frac {v^{2}}{c^{2}}}}}}}[/tex]

[tex]{\displaystyle L={\frac {L_{0}}{\gamma }}}[/tex]

Lo = panjang awal benda

[tex]{\displaystyle t=t_{0}\times {\gamma }}[/tex]

[tex]{\displaystyle m=m_{0}\times \gamma }[/tex]

Uang Agan = 60.000 + uang Dika + 2 x uang Rini 
Uang Agan + uang Dika + uang Rini = 300.000
Uang Dika - uang Rini = 15.000 

Ditanya :
Banyak uang Agan =?

Pembahasan:
Pertama-tama mari kita ubah masing - masing bentuk uang ke dalam satuan uang Rini sehingg didapatkan hasil seperti berikut 

Uang Dika - uang Rini = 15.000
Uang Dika = Uang Rini + 15.000

Uang Agan = 60.000 + Uang Dika + 2 x Uang Rini
Uang Agan = 60.000 + Uang Rini + 15.000 + 2 x Uang Rini 
Uang Agan = 3 x Uang Rini + 75.000

Setelah itu, barulah kita bisa mencari jumlah uang rini dengan cara mensubstitusi nilai uang Agan dan uang DIka sebagai berikut :

Uang Agan + Uang Dika + Uang Rini = 300.000
3 x Uang Rini + 75.000 + Uang Rini + 15.000 + Uang Rini = 300.000
5 x Uang Rini + 90.000 = 300.000
5 x Uang Rini = 300.000 - 90.000
Uang Rini = [tex] \frac{210.000}{5} [/tex]
Uang Rini = 42.000

Setelah mendapatkan Uang Rini, maka kita dapat mencari jumlah uang yang dimiliki oleh Agan dengan cara sebagai berikut :

Uang Agan = 3 x Uang Rini + 75.000
Uang Agan = 3 x 42.000 + 75.000
Uang Agan = 126.000 + 75.000 
Uang Agan = 201.000 

Sehingga didapatkan bahwa uang yag dimiliki oleh Agan adalah sebanyak Rp. 201.000,-
