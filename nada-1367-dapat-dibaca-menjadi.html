9. rumus suku ke-n dari suatu barisan bilangan dinyatakan dengan Un = 2n^2 -3. besar suku ke-20 dari barisan tersebut adalah..a. 793b. 797c. 1367d. 1593e. 1597jawab nya pake cara ..


Rumus suku ke-n dari suatu barisan bilangan dinyatakan dengan Un = 2n^2 -3. besar suku ke-20 dari barisan tersebut adalah ...

Barisan aritmetika adalah barisan bilangan dengan hasil pengurangan setiap suku dengan suku sebelumnya selalu sama atau tetap.

Suku ke-n   →   Un = a + (n - 1) b

Deret aritmetika adalah penjumlahan dari suku-suku suatu barisan aritmetika.

Jumlah n suku pertama →   Sn = n/2 [2a + (n - 1) b]

                                     atau Sn = n/2 (a + Un)

Pembahasan

Diketahui :

Un = 2n² - 3

Ditanya :

U₂₀ = ... ?

Jawab :

Suku ke-20

Un = 2n² - 3

U₂₀ = 2 (20)² - 3

      = 2 (400) - 3

      = 800 - 3

      = 797

Jadi besar suku ke-20 dari barisan tersebut adalah 797

Jawaban : B
