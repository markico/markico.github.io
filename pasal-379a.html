25. tentukan nilai dari pembagian istimewa (379 + 184)2-(379 - 184)2/ 379×184


Bab Pembagian
Matematika SMP Kelas VII

((379 + 184)² - (379 - 184)²)/(379 x 184)
= ((379 + 184 + 379 - 184) x (379 + 184 - 379 + 184)) / (379 x 184)
= (2 x 379 x 2 x 184) / (379 x 184)
= 4
