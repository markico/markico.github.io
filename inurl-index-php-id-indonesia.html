19. Apa perbedaan antara unweighted price index dengan weighted price index


Price Index (Indeks harga) adalah suatu satuan tertentu yang menggambarkan harga pada tahun tertentu dengan membandingkan dengan harga sebelumnya. Price Index sangat bermanfaat untuk menggambarkan keadaan ekonomi terbaru sekaligus digunakan dalam perhitungan tingkat inflasi (kenaikan harga barang). Price Index terbagi atas 2 yaitu:

1) Unweighted price index (Indeks Harga tidak tertimbang / indeks harga sederhana)

Adalah salah satu metode perhitungan yang tidak memperhatikan banyak output yang ditransaksikan melainkan hanya melihat harga saja.


2) Weighted Price Index (Indeks Harga tertimbang)

Adalah metode perhitungan indeks harga yang juga memperhitungkan jumlah output transaksi yang terjadi. Metode ini terbagi menjadi 3 metode yang umum yaitu : Laspeyres , Paasche, dan Marshall.

Kesimpulan : Perbedaan unweighted dan weighted price index terletak pada unsur yang mencakup. Unweighted hanya mengukur harga sedangkan weighted mengukur harga dan kuantitas.

24. Berikan penjelasan angka index agregatif & angka index sederhana


Penghitungan angka indeks dapat dilakukan dengan beberapa metode. Oleh karena itu, perlu dilakukan pilihan yang tepat agar tujuan angka indeks yang telah ditetapkan dapat tercapai. Pada dasarnya terdapat dua metode penghitungan angka indeks yaitu sebagai berikut.

a. Angka indeks sederhana atau angka indeks tidak ditimbang (simple agregative methode) dibagi dalam bentuk agregatif sederhana dan rata-rata harga relatif atau agregative relative.

b. Angka indeks yang ditimbang, dibagi menjadi bentuk agregatif sederhana dan rata-rata harga relatif tertimbang.

Untuk lebih jelasnya, perhatikan pembahasan berikut ini.

a. Indeks Harga Tidak Tertimbang dengan Metode Agregatif Sederhana.

Angka indeks yang dimaksud dalam penghitungan indeks harga tidak tertimbang meliputi indeks harga, kuantitas, dan nilai. Marilah kita simak pembahasannya masing-masing.

1) Angka indeks harga (price = P)

Keterangan:

IA = indeks harga yang tidak ditimbang

Pn = harga yang dihitung angka indeksnya

Po = harga pada tahun dasar

Contoh:

Berdasarkan data di atas, maka angka indeks harga tahun 2004 adalah:

IA = 1.500/1.300 x 100 = 115,38%

Jadi, harga tahun 2004 mengalami kenaikan sebesar 15,38%.

28. Tuliskan syntax Php untuk menghapus data siswa berdasarkan ID siswa​


Jawaban:

berikut code nya :

<?php

// Koneksi ke database

$conn = mysqli_connect("localhost", "username", "password", "nama_database");

// Menangkap ID siswa yang akan dihapus dari parameter GET

$id_siswa = $_GET["id"];

// Membuat query untuk menghapus data siswa berdasarkan ID

$query = "DELETE FROM siswa WHERE id_siswa = $id_siswa";

// Menjalankan query untuk menghapus data

if (mysqli_query($conn, $query)) {

echo "Data siswa berhasil dihapus";

} else {

echo "Gagal menghapus data siswa: " . mysqli_error($conn);

}

// Menutup koneksi ke database

mysqli_close($conn);

?>

Penjelasannya :

1. Pertama-tama kita melakukan koneksi ke database dengan menggunakan fungsi `mysqli_connect()` . Di sini, kamu harus mengganti "localhost", "username", "password", dan "nama_database" dengan nilai yang sesuai untuk koneksi ke database kita.

2. menangkap ID siswa yang akan dihapus dari parameter GET dengan menggunakan variabel `$_GET["id"]`.

3. Setelah itu, kita membuat query SQL untuk menghapus data siswa berdasarkan ID menggunakan perintah `DELETE FROM`. Di sini, kita menggunakan variabel `$id_siswa` untuk menggantikan nilai ID siswa yang kita tangkap sebelumnya.

4. Kita mengeksekusi query menggunakan fungsi `mysqli_query()`. Jika query berhasil dijalankan, kita akan menampilkan pesan "Data siswa berhasil dihapus". Jika tidak, kita akan menampilkan pesan error menggunakan fungsi `mysqli_error()`.

5. Terakhir, kita menutup koneksi ke database menggunakan fungsi 'mysqli_close()'.

Jawaban:

<?php

// koneksi ke database

$servername = "localhost";

$username = "username";

$password = "password";

$dbname = "myDB";

$conn = mysqli_connect($servername, $username, $password, $dbname);

// periksa koneksi

if (!$conn) {

 die("Koneksi gagal: " . mysqli_connect_error());

}

// ambil ID siswa yang akan dihapus dari parameter URL

$id = $_GET['id'];

// query untuk menghapus data siswa berdasarkan ID siswa

$sql = "DELETE FROM siswa WHERE id=$id";

// eksekusi query

if (mysqli_query($conn, $sql)) {

 echo "Data siswa berhasil dihapus";

} else {

 echo "Error: " . $sql . "<br>" . mysqli_error($conn);

}

// tutup koneksi

mysqli_close($conn);

?>

Penjelasan:

Koneksi ke database diinisialisasi dengan variabel $servername, $username, $password, dan $dbname.Koneksi ke database dilakukan dengan fungsi mysqli_connect().ID siswa yang akan dihapus diambil dari parameter URL dengan variabel $id.Query untuk menghapus data siswa dengan ID $id dibuat dengan variabel $sql.Query dieksekusi dengan fungsi mysqli_query().Jika query berhasil dieksekusi, pesan berhasil dicetak ke layar. Jika tidak, pesan error dicetak ke layar.Koneksi ditutup dengan fungsi mysqli_close().
