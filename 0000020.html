4. tolong ya kak :000000


Ada 2 loop

Loop 1 :
I1 = V1 / R
= 12V / 6Ω
= 2A

Loop 2 :
I2 = V2 / R
= 22V / (5Ω + 6Ω)
= 22V / 11Ω
= 2A

Arus yang mengalir di R = 6Ω ada 2 yaitu dari loop 1 dan loop 2 sehingga besar arusnya sesuai hukum Kirchhoff:
I masuk = I keluar

Ir6 = I1 + I2
= 2A + 2A
= 4 A

Jadi, arus pada R = 6Ω sebesar 4 A

Conditional Sentences:

1. If Rina studied science, she would be a scientist

2. If you watched a new movie, you would be Happy.

3. If Father opened the door, he would see me in the living room.

4. If Mother went shopping, she would spend a lot of money.

5. If I had an airplane, I would fly to London.

Penjelasan:

Jawaban merupakan materi pembelajaran Conditional Sentence type 2 (kalimat pengandaian tipe 2) dalam Bahasa Inggris.

Kalimat pengandaian tipe 2

Fakta: Kondisi yang tidak mungkin terwujud di masa kini.

If + subject + V2/simple past tense + Subject + would + Vl/past future tense

atau

If + subject + were + adjective/noun+Subject + would be + adjective/noun

Contoh:

If l were you, I would accept that job offer. Fakta: I am not you.
