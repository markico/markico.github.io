Jawab:

Penjelasan dengan langkah-langkah:

Volume Bola = [tex]\frac{4}{3}[/tex] x [tex]\pi[/tex] x [tex]r^{3}[/tex]

= [tex]\frac{4}{3}[/tex] x 3,14 x 3 x 3 x 3 = 4 x 3 x 3 x 3,14 = 36 x 3,14 = 113,04 cm3

LP Bola = 4 x [tex]\pi[/tex] x [tex]r^{2}[/tex]

= 4 x 3,14 x 3 x 3 = 36 x 3,14 = 113,04 cm2
