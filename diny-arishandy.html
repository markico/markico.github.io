10. Umur Cika 2 tahun lebih muda dari umur Diny. Tiga tahun yang akan datang, perbandingan umur mereka 5:6. Tentukan umur mereka masing masing! mohon dijawab, besok dikumpul


Jawaban:

Cika 7 tahun dan Diny 9 tahun.

Penjelasan:

Misal,

c = umur Cika sekarangd = umur Diny sekarang

Dan, tiga tahun lagi, umur Cika = 5x

umur Diny = 6x

Diketahui umur Cika 2 tahun lebih muda dari umur Diny, maka :

c = d - 2

atau

d = c + 2

Tiga tahun lagi, umur Cika = 3 + c = 5x

umur Diny = 3 + d = 6x

Karena d = c + 2

3 + d = 3 + (c + 2)

= 5 + c

3 + c = 5x

5 + c = 6x

________ -

-2 = -x

x = 2

Maka, tiga tahun yang akan datang,

Umur Cika = 5x = 5(2) = 10 tahun

Umur Diny = 6x = 6(2) = 12 tahun

Jadi, c = 10 - 3 = 7 tahun

d = 12 - 3 = 9 tahun

Sekian dari saya, semoga membantu :)
