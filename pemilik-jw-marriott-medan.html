3. wilayah A berpenduduk 50.000 jiwa tahun 2007 dengan pertumbuhan penduduk 2 % pertahun maka jumlah penduduk wilayah A pada tahun 2012 adalah.... a. 65.000 jw b. 55.000 jw c. 35.000 jw d. 10.000 jw e.5.000 jw


Dik : Po = 50.000
no = 2007
nt = 2012
r = 2 %/tahun

Dit : Pt saat 2012

Jawab :
Pt = Po + (r x Δn x Po)
Pt = 50.000 + (2/100 x (2012-2007) x 50.000)
Pt = 50.000 + (2/100 x 5 x 50.000)
Pt = 50.000 + 5.000
Pt = 55.000 Jiwa
