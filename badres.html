3. apa arti badass,baddas,badas dsb,dll,dst,dsj


1. BADASS,BADDAS,BADAS : Secara makna kamus arti Badass,Baddas,Badas dapat diambil dari kata Bad/Badd berarti jelek/buruk dan Ass/as berarti bokong/pantat. Namun dalam penggunaannya Badass,Baddas,Badas bukan berarti bokong jelek.Secara istilah Badass Baddas,Badas berarti sangat keren/ keren banget/ keren gila.

2. DSB : Dan sebagai berikut/Dan sebagainya.Ungkapan tersebut digunakan untuk menyatakan rincian yang berjenjang atau berkelanjutan seperti contoh berikut ini.

3. DLL : Ungkapan “dan lain-lain” (dll.) digunakan pada perincian yang beragam atau berbeda.

4. DST : Ungkapan “dan seterusnya” (dst.) digunakan pada perincian yang berjenjang atau yang berkelanjutan secara berurutan

5. DSJ : DSJ adalah kata singkatan atau singkatan yang didefinisikan dalam bahasa sederhana.

a = b + 2.000.
c = 3b + a.

a + b + c = 70.000.
(b + 2.000) + b + (3b + a) = 70.000.
5b + 2.000 + (b + 2.000) = 70.000.
6b + 4.000 = 70.000.
6b = 70.000 - 4.000.
6b = 66.000.
b = 66.000/6.
b = 11.000.

Uang citra
= 3b + a.
= 3b + (b + 2.000).
= 4b + 2.000.
= 4(11.000) + 2.000.
= 44.000 + 2.000.
= Rp46.000,00.
