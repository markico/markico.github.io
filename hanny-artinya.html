4. hanny mengisi kolam nya yng kosong debgan air menggunakan selang...kolam hanny terisi air sebanyak 54m² dalam waktu 30 menit , brp liter debiy air dari selabf tersebut?bantu jawab ya!!​


Diketahui :

kolam hanny terisi air sebanyak 54m³ dalam waktu 30 menit

Ditanya :

berapa liter debit air dari selang tersebut?

Dijawab :

Debit = Volume : Waktu

Debit = 54 m³ : 30 menit

Debit = 54.000 dm³ : (30 × 60) detik

Debit = 54.000 liter : 1.800 detik

Debit = 30 liter/detik

Jadi, Debit air kolam hanny dari selang adalah 30 liter/detik.

19. Uang RISKI banding uang Hanny 2:3 uang Hanny banding uang nabiha 4:5 jika jumlah uang merek@ 420,000 berapakah uang mereka masing2?


R : H = 2 : 3 => 2H = 3R => R = 2H/3
H ; N = 4 : 5 => 4N = 5H => N = 5H/4
R + H + N = 420.000
2H/3 + H + 5H/4 = 420.000
8H/12 + 12H/12 + 15H/12 = 420.000
35H/12 = 420.000
35H = 5.240.000
H = 144.000
R = 2H/3
R = 2.144.000 / 3
R = 288.000/3
R = 96.000
N = 5H/4
N = 5.144.000/4
N = 180.000
Jadi uang Riski Rp.96.000,- uang Hani Rp 144.000,- dan uang Nabiha Rp 180.000,-
