4. PA dan PB garis singgung lingkaran dgn pusat O. panjang OP= 50 cm dan PA= 40 cm. panjang tali busur AB adalah


// misal di antara garis AB ada titik D dan AD ⊥ OP //


OA = √OP² - PA²

      = √(50cm)² - (40cm)²

      = √2500cm² - 1600cm²

      = √900cm²

      = 30cm


      LΔ OAP = LΔ OAP

AD × OP / 2 = PA × OA / 2

  AD × 50cm = 40cm × 30cm // per 2 di hilangakan //

               AD = 40cm × 30cm / 50cm

               AD = 40cm × 3 / 5

               AD = 8cm × 3

               AD = 24cm


AB = 2AD

      = 2(24cm)

      = 48cm

L= 2 x luas segitiga. L = 2 x 1/2 x OA x AP. L = 2 x 1/2 x 30 x 40. L = 2 x 1/2 x 1200. L = 2 x 600. L = 1200 cm². Lalu, cri pnjang tali busur. L = 1/2 x diagonal 1 x diagonal 2. 1200 = 1/2 x OP x AB. 1200 = 1/2 x 50 x AB. 1200 = 25 x AB. 1200/ 25 = AB. 48 = AB. Jd, pnjang AB = 48. Maaf, klo kepanjangan dn salah

30. Titik P adalah pusat segi enam beraturan ABC.DEF. Jika PA = a dan CD = b, maka FD =........


Titik P adalah pusat segi enam beraturan ABC.DEF. Jika PA = a dan CD = b, maka FD = –2a – b. Vektor adalah besaran yang memiliki nilai dan arah. Penulisannya bisa ditulis dalam 2 huruf kapital atau 1 huruf kecil. Suatu vektor dikatakan sama jika memiliki panjang dan arah yang sama. Penjumlahan vektor ada dua cara yaitu:

Cara segitiga Cara jajar genjang

(untuk lebih jelas bisa dilihat dilampiran)

Pembahasan

Diketahui

Segi enam ABCDEF

PA = a CD = b

Ditanyakan  

FD = .... ?

Jawab

Berdasarkan gambar segienam pada lampiran,

Vektor yang panjangnya sama dan searah dengan vektor a adalah

PA = EF = CB = DP = a AP = FE = BC = PD = –a  

Vektor yang panjangnya sama dan searah dengan vektor b adalah

CD = BP = PE = AF = b DC = PB = EP = FA = –b

Perhatikan:  

CB = a CD = b

Dengan penjumlahan vektor cara jajar genjang

CP = a + b

Sehingga

CP = PE = BA = DE = a + b PC = EP = AB = ED = –(a + b) = –a – b  

Penjumlahan dengan cara segitiga

FD = FE + ED  

FD = –a + (–a – b)

FD = –2a – b
