16. Tentukan n jika 3 + 3^2 + 3^3 + . . . + 3^n = 363


Soal

Tentukan n jika 3 + 3^2 + 3^3 + . . . + 3^n = 363

Diketahui :

U1 = a = 3

r = U2 / U1 = 3² / 3 = 3

Sn = 363

Ditanya :

n = ?

Jawab :

karena rasionya lebih dari 1 maka kita menggunakan rumus berikut.

Sn = a ( r^n - 1 ) / ( r - 1 )

kita subtitusikan nilai a, r dan Sn kedalam rumus sehingga :

363 = 3 ( 3^n - 1 ) / 3 - 1

363 = 3 ( 3^n - 1 ) / 2

363 / 3 * 2 = ( 3^n - 1 )

242 = 3^n - 1

243 = 3^n

3⁵ = 3^n Coret angka 3 nya

5 = n

n = 5

Jadi nilai dari n adalah 5
