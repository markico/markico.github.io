9. Kuis [Susah] Suku ke-5n barisan aritmatika adalah 1049. Suku ke-6n barisan aritmatika adalah 1259. Suku ke-5,5n nya adalah? A. 1089B. 1154C. 1185D. 1204E. 1250​


Jawab:

B. 1154

-------------------------------

Penjelasan dengan langkah-langkah:

Diketahui

--> U5n = 1049

--> U6n = 1259

Dicari U5,5n

-------------------------------

b = Uy-Ux / y-x dengan y > x

maka

b = 1259-1049/6n-5n

b = 210/n

-------------------------------

Jika U5,5n = x

Cari x

b = (1259-x) / (6n - 5,5n)

210/n = (1259-x) / (6n - 5,5n)

210/n = (1259-x) / (n/2)

210 = 2(1259-x)

210 = 2518-2x

2x = 2518-210

2x = 2308

x = 2308/2

x = 1154

-------------------------------

Suku ke-5,5n = 1154
