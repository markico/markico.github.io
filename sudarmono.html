8. Sudarmono mempunyai uang sebesar rp.15.000.000,00 didepositokan dgn bunga majemuk 10% pertahun. Periode konversinya 4 bulanan selama 3 thn. Modal dan bunga setelah 3 thn mjd....?


Dik : M = 15.000.000
         i  = 10% /4 bulan = 0,1% / 4 bulan
         n  = 3 tahun

Penyelesaian :

[tex]Mn = M (1 + i )^n[/tex]
[tex]Mn = 15.000.000 (1 + 0,1)^3[/tex]
[tex]Mn = 15.000.000 (1,1)^3[/tex]
[tex]Mn = 15.000.000 (1,331)[/tex]
[tex]Mn = 19.965.000[/tex]

Jadi, Modal setelah 3 tahun adalah Rp 19.965.000

Bunga = 19.965.000 - 15.000.000 = 4.965.000

Jadi, bunganya selama 3 tahun adalah Rp 4.965.000

26. Sudarmono mempunyai uang sebesar rp.15.000.000,- didepositokan dgn bunga majemuk 10% pertahun. periode konversinya 4bulanan slm 3th. modal & bunga stlh 3th mjd...?


Dik : M = 15.000.000
         i  = 10% /4 bulan = 0,1% / 4 bulan
         n  = 3 tahun

Penyelesaian :

[tex]Mn = M (1 + i )^n[/tex]
[tex]Mn = 15.000.000 (1 + 0,1)^3[/tex]
[tex]Mn = 15.000.000 (1,1)^3[/tex]
[tex]Mn = 15.000.000 (1,331)[/tex]
[tex]Mn = 19.965.000[/tex]

Jadi, Modal setelah 3 tahun adalah Rp 19.965.000

Bunga = 19.965.000 - 15.000.000 = 4.965.000

Jadi, bunganya selama 3 tahun adalah Rp 4.965.000
