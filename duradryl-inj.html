[tex]\left[\begin{array}{ccc}x\\y\end{array}\right]=\left[\begin{array}{ccc}-3&2\\2&-1\end{array}\right] \left[\begin{array}{ccc}x'\\y'\end{array}\right][/tex]

Sehingga diperoleh
x = -3x' + 2y'
y = 2x' - y'

Bayangan garis yang diperoleh
x + 3y + 1 = 0
(-3x' + 2y') + 3(2x' - y') + 1 = 0
-3x' + 2y' + 6x' - 3y' + 1 = 0
3x' - y' + 1 = 0
3x - y + 1 = 0

Jadi, persamaan bayangan dari garis tersebut adalah 3x - y + 1 = 0
