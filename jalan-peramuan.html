14. Berapa massa atom dari unsur Ag jika diketahui Ag memiliki dua isotopyaitu 51,82% 107 Ag (106,9 amu) dan 48,1896 109Ag (108.9 annu)?ПА.106,53 amuB. 107,94 amu107 86 amu] D 108,01 amu) E 108,15 amu​


Jawaban:

107,86

Penjelasan:

Ar Ag = ( % x massa 107 Ag ) + ( % x massa 109 Ag )

=> ( 51,82% x 106,9 ) + ( 48,19% x 108,9 )

=> 55,4 + 52,48

=> 107,88

Opsi terdekat 107,86
