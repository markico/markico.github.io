Diketahui data sebagal berikutPengeluaran konsumsiRp950 000 900,00InvestasiRp750 000 000,00BungaRp400 9300 900.00EksporRp300.000 100.00Pengeluaran pemerintahRp600 000 000.00LabaRp500 500.000,00ImporRp200 000.000,00Hitunglah besamya pendapatan nasional dengan pendekatan pengeluaranJawab:​


Besarnya pendapatan nasional dengan pendekatan pengeluaran adalah Rp 2.400.001.000 miliar

Pembahasan

Dalam perhitungan pendapatan nasional dengan pendekatan pengeluaran adalah penjumlahan atas segala sektor-sektor ekonomi (konsumsi rumah tangga, pengeluaran yang dikeluarkan pemerintah, investasi, dan juga ekspor neto). Metode pendekatan pengeluaran digunakan rumus berikut :

Y = C + G + I + (X - M)

Keterangan :

Y = Pendapatan nasional

C = Konsumsi rumah tangga

G = Pengeluaran pemerintah

I = Investasi

(X - M) = (Ekspor - Impor)

________________________________

Diketahui :

C = Rp 950.000.900

G = Rp 600.000.000

I = Rp 750.000.000

(X - M) = (Rp 300.000.100 - Rp 200.000.000)

Ditanyakan :

Pendapatan nasional (Y)?

Penyelesaian :

Y = C + G + I + (X - M)

Y = Rp 950.000.900 + Rp 600.000.000 + Rp 750.000.000 +  (Rp 300.000.100 - Rp 200.000.000)

Y = Rp 2.300.000.900 + Rp 100.000.100

Y = Rp 2.400.001.000 miliar

Jadi, besarnya pendapatan nasional dengan pendekatan pengeluaran adalah Rp 2.400.001.000 miliar

13. Tuan herman mendeposito Rp400. 000 pada akhir setiap bulan di Bank selama 5 tahun. Tentukanlah nilai uang yang terakumulasi dari auitas ini, jika tingkat bunga adalah 5% dimajemukan secara tahunan, semester, kuartalan, bulanan​


Tingkat bunga efektif per bulan = (1 + tingkat bunga / frekuensi pemajemukan)^(frekuensi pemajemukan/12) - 1

Jadi, tingkat bunga efektif per bulan untuk tingkat bunga 5% yang dimajemukan secara tahunan adalah:

(1 + 0.05 / 1)^(1/12) - 1 = 0.00417

Kemudian, kita dapat menggunakan rumus berikut untuk menghitung nilai uang yang terakumulasi:

Nilai uang terakumulasi = deposito per bulan * ((1 + tingkat bunga efektif per bulan)^jumlah bulan - 1) / tingkat bunga efektif per bulan

Jadi, nilai uang yang terakumulasi untuk deposito Rp400.000 per bulan selama 5 tahun (60 bulan) adalah:

Rp400.000 * ((1 + 0.00417)^60 - 1) / 0.00417 = Rp3.524.491

Jadi, nilai uang yang terakumulasi dari auitas ini adalah Rp3.524.491.

Untuk menentukan tarif aktivitas dengan sistem ABC, Anda perlu menghitung biaya total aktivitas yang dianggarkan untuk setiap produk, kemudian membagi biaya tersebut dengan jumlah unit produksi masing-masing produk.

Jadi, tarif aktivitas rekayasa untuk produk permint adalah Rp200.000,- / 2000 unit = Rp100,-/unit. Tarif aktivitas rekayasa untuk produk jahe adalah Rp30.000,- / 4000 unit = Rp7,50/unit.

Setelah menghitung tarif aktivitas untuk setiap produk, Anda dapat menghitung biaya per unit dengan sistem ABC dengan menjumlahkan tarif aktivitas yang telah ditentukan untuk setiap produk dan biaya utama yang telah ditentukan untuk setiap produk.

Biaya per unit produk permint adalah Rp100,-/unit (tarif aktivitas rekayasa) + Rp1.000,-/unit (tarif aktivitas set up) + Rp400,-/unit (tarif aktivitas packaging) + Rp20.000,-/unit (biaya utama) = Rp521.000,-/unit

Biaya per unit produk jahe adalah Rp7,50/unit (tarif aktivitas rekayasa) + Rp250,-/unit (tarif aktivitas set up) + Rp100,-/unit (tarif aktivitas packaging) + Rp10.000,-/unit (biaya utama) = Rp367.500,-/unit.
