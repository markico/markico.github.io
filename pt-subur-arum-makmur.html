Jawaban:

Penjualan Rp 8.500.000.000

Persediaan Awal : 1.850.000.000

Pembelian : 6.550.000.000

Persediaan Akhri : 1.950.000.000

HPP = Penjualan bersih - Pembelian bersih

Pembelian bersih = (Persediaan Awal + Pembelian - Persediaan akhir)

HPP =  8.500.000.000 - ( 1.850.000.000 + 6.550.000.000) -6.550.000.000

HPP = 8.500.000.000 - (  8.400.000.000 - 6.550.000.000)

HPP = 8.500.000.000 - (  8.400.000.000 - 6.550.000.000)

HPP = 8.500.000.000 -  1.850.000.000

HPP = 6.650.000.000
