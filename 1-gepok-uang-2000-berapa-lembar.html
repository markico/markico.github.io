1. Rini memiliki 120 lembar uang yang terdiri atas lembaran uang 2000-an,. 5000-an dan 10000-an. Banyak lembaran uang 2000-an ada 15 lembar lebih banyak dari lembaran uang 10000-an, banyak lembaran uang 5000-an 3 kali banyak dari pecahan 2000-an. Nilai uang yang dimiliki rini adalah...


misal banyaknya uang 2000-an adalah x, banyaknya uang 5000-an adalah y, dan banyaknya uang 10000-an adalah z
[tex]x + y + z = 200 \\ x = z + 15 \\ y = 3x[/tex]

[tex]x + y + z = 200 \\ x + 3x + x - 15 = 200 \\ 5x = 215 \\ x = 43 \\ y = 3x = 129 \\ z = x - 15 = 28[/tex]

[tex]2000 \times 43 + 5000 \times 129 + 10000 \times 28 \\ = 86000 + 645000 + 280000 \\ = 1011000[/tex]jika

uang 5000 = b
uang 10000 = c

a - 15 = c
b = 3a

a + b + c = 120
a + 3a + a - 15 = 120
5a - 15 = 120
5a = 120 + 15 = 135
a = 135 ÷ 5 = 27

b = 3a = 3 x 27 = 81

c = a - 15 = 27 - 15 = 12

jumlah uang
= ( 27 x 2000 ) + ( 81 x 5000 ) + ( 12 x 10000 )
= 54000 + 405000 + 120000
= 579000
