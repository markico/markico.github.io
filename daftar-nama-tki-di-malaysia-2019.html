11. Bagaimana pemerintah Malaysia memperlakukan TKI ilegal


Kategori Soal : Bab TKI { Tenaga Kerja Indonesia }
Kelas : 8 SMP/ VII SMP
Kata Kunci : Bagaimana pemerintah Malaysia memperlakukan TKI ilegal 

=> Pembahasan :

** TKI { Tenaga Kerja Indonesia } 
TKI Terdiri dari 2. 
Yaitu :
-> TKI { Tenaga Kerja Indonesia } Legal 
-> TKI { Tenaga Kerja Indonesia } Ilegal 

** Penjelasan Tentang TKI 

-> TKI { Tenaga Kerja Indonesia } Legal 
Yaitu :
=> Tenaga Kerja Indonesia yang Bekerja di Luar Negeri Yang Memiliki Izin Kerja atau lainnya.
Misalkan :
-- Passports Murni 
-- Tanda Kerja 
-- Izin Kerja 
-- Dan lainnya 

-> TKI { Tenaga Kerja Indonesia } Ilegal 
Yaitu : 
=> Tenaga Kerja Indonesia yang Bekerja di Luar Negeri Yang Tidak Memiliki Izin Kerja atau lainnya.
Misalkan :
-- Tidak Memiliki Passports Murni 
-- Tidak Memiliki Tanda Kerja 
-- Tidak Memiliki Izin Kerja 
-- Dan lainnya 

** Bagaimana pemerintah Malaysia memperlakukan TKI ilegal?
Jawab :

-> Jika Tenaga Kerja Indonesia { Tidak Bermasalah Dalam Kerja }
Yaitu :
1. Di Pulangkan Ke Negara Aslinya/Asalnya { Negara Indonesia }.
2. Di Cabut Dari Pekerjaannya.

-> Jika Tenaga Kerja Indonesia { Bermasalah Dalam Kerja }
Yaitu :
1. Di Pulangkan Ke Negara Aslinya/Asalnya { Negara Indonesia }.
2. Di Cabut Dari Pekerjaannya. 
3. Di Hukum Denda 
4. Di Hukum Mati { Apabila Seorang TKI Membunuh/Melakukan Kekerasan }
5. Tidak dapat Pekerjaan lagi.

17. TKI yang bekerja di Malaysia banyak bekerja di industri


Timah
maaf ya klo salahTenaga Kerja Indonesia (disingkat TKI) adalah sebutan bagi warga negara Indonesia yang lowlife dan unskill yang bekerja di luar negeri (seperti Malaysia, Timur Tengah, Taiwan, uganda dan somalia) dalam hubungan kerja untuk jangka waktu tertentu dengan menerima upah. Namun, istilah TKI seringkali dikonotasikan dengan pekerja kasar karena TKI sejatinya memang adalah kumpulan tenaga kerja unskill yang merupakan program pemerintah untuk menekan angka pengangguran. TKI perempuan seringkali disebut Tenaga Kerja Wanita (TKW).

TKI sering disebut sebagai pahlawan devisa karena dalam setahun bisa menghasilkan devisa 60 triliun rupiah (2006) [1], tetapi dalam kenyataannya, TKI menjadi ajang pungli bagi para pejabat dan agen terkait. Bahkan di Bandara Soekarno-Hatta, mereka disediakan terminal tersendiri (terminal III) yang terpisah dari terminal penumpang umum. Pemisahan ini beralasan untuk melindungi TKI tetapi juga menyuburkan pungli, termasuk pungutan liar yang resmi seperti punutan Rp.25.000,- berdasarkan Surat Menakertrans No 437.HK.33.2003, bagi TKI yang pulang melalui Terminal III wajib membayar uang jasa pelayanan Rp25.000. (saat ini pungutan ini sudah dilarang)

Pada 9 Maret 2007 kegiatan operasional di bidang Penempatan dan Perlindungan TKI di luar negeri dialihkan menjadi tanggung jawab BNP2TKI. Sebelumnya seluruh kegiatan operasional di bidang Penempatan dan Perlindungan Tenaga Kerja Indonesia di luar negeri dilaksanakan oleh Ditjen Pembinaan dan Penempatan Tenaga Kerja Luar Negeri (PPTKLN) Depnakertrans.
