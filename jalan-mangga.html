Mangga = y
2x + 4y = 42.000 | x1 |
x + y = 12.000 | x2 |
(Eliminasi x)
2x + 4y = 42.000
2x + 2y = 24.000
(Kurangkan semua.y)
=> 0 + 2y = 18.000
=> 2y = 18.000
=> y = 9.000
Sub. y ke per.2
=> x + y = 12.000
=> x + 9.000 = 12.000
=> x = 12.000 - 9.000
=> x = 3.000

jadi, Harga melon: 3.000
Harga Mangga: 9.000

selisih harga melon dan mangga adalah
= 3.000 - 9.000
= 6.000
(karena ini harga jdi tdak mngenal sistem negatif)
