29. 11. Sebuah kubus memiliki sisi-sisi 2 m, berdiri tegak diatas tanah. Apabila berat balok terse4.800 N, tekanan maksimum balok tersebut adalah ....a. 2.400 Pab. 1.200 Pac. 600 Pad. 300 Pa​


Sebuah kubus memiliki sisi-sisi 2 m, berdiri tegak diatas tanah. Apabila berat balok terse but 4.800 N, tekanan maksimum balok tersebut adalah B. 1.200 Pa

PEMBAHASAN

Rumus dasar tekanan yang perlu di pahami adalah :

Tekanan = Gaya / Luas Penampang

atau di simbolkan :

P = F / A

P = Tekanan ( Pa )

F = Gaya Tekan ( N )

A = Luas Penampang ( m² )

Tekanan yang di kerjakan oleh cairan atau di kenal dengan nama Tekanan Hidrostatik bisa di cari dengan rumus :

Tekanan Hidrostatik = ρ x  g x h

dimana :

ρ = massa jenis cairan ( kg /m³ )

g = percepatan gravitasi ( m/s² )

h = kedalaman cairan ( m )

 Mari kita mencoba menyelesaikan persoalannya.

Diketahui:

s = 2 m

w = 4.800 N

Ditanyakan:

P = ?

Penyelesaian:

P = F / A

P = F / s²

P = 4.800 / 2²

P = 4.800 / 4

P = 1.200 Pascal
