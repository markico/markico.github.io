= (a⁴b-²c-³)/(a-²b²c-⁴)
= a^(4+2)b^(-2-2)c^(-3+4)
= ca^6/b⁴

c) [(p^½q^¾r)/(4pq^-¼)]²
= [(p^½×2 q^¾×2 r²)/(4²p²q^-¼×2)]
= (pr²q^3/2)/(16p²q^-½)
= p^(1-2)q^(3/2+½)r²/16
= q²r²/16p
