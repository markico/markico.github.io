12. judul persamaan kuadratkls 9 SMP1: persamaan kuadrat x²+6x-5=0 akar akar a dan b.tentukan persamaan kuadrat baru yang akarnya ( a+2) dan (b+2) .Jangan lupa subscribe channel YouTube LETDA HYPER. ​


Jawaban :

x² + 2x - 3 = 0

Langkah penyelesaian :

x² + 6x + 5 = 0

= (x + 5) (x + 1)

Akar-akar persamaannya

x + 5 = 0

x1 = -5

x + 1 = 0

x2 = -1

Persamaan kuadrat yang baru

(x - x1) (x - x2) = 0

(x - (-5 + 2)) (x - (-1 + 2)) = 0

(x - (-3)) (x - 1) = 0

(x + 3) (x - 1) = 0

x² - x + 3x - 3 = 0

x² + 2x - 3 = 0 ✔

Keterangan :

Materi = Persamaan Kuadrat
