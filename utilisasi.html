4. 1. Sebuat RSX di kota Y mampu memproduksi jasa perawatan luka sebesar 120.000 unit dengan fix cost sebesar Rp.780,- dan variable cost sebesar Rp.700,-. Adapun harga jualnya adalah Rp. 60% dari total cost per unit. Tahun lalu utilisasi jasa tsb sebesar 90.000 unit jasa pelayanan. Akan tetapi karena terjadi perubahan pasar dan adanya persaingan antar RS lainnya tingkat utilisasinya hanya sebesar 40.000 unit. Dari data diatas perhitungkan laba/ruginya!​


Jumlah jasa yang mampu diproduksi RS X sebesar 120.000 unit. Dengan kapasitas tersebut maka biaya yang dikeluarkan per tahun sebesar Rp 117.600.000.

Pada tahun pertama RS X mampu menjual 90.000 unit dengan keuntungan 60%, maka penjualan yang dicapai sebesar Rp 213.120.000. Sehingga di tahun pertama RS X mengalami laba sebesar 95.520.000.

Pada tahun kedua RS X hanya mampu menjual 40.000 unit dengan keuntungan 60%. Maka penjualan mencapai sebesar Rp 94.720.000. Dengan kapasitas produksi per tahun yang memakan biaya Rp 117.600.000, RS X mengalami kerugian Rp 22.880.000

Sehingga jika tahun pertama dan kedua diakumulasi, RS X masih mengalami laba sebesar Rp 72.640.000

Penjelasan dengan langkah-langkah:

Diketahui:

Fix cost = Rp 780 per unit

Variable cost = Rp 700 per unit

Produksi = 120.000 unit

Harga jual = 60% lebih tinggi dari total cost per unit

Utilisasi tahun pertama = 90.000 unit

Utilisasi tahun kedua = 40.000 unit

Ditanyakan: kondisi laba atau rugi RS X?

Langkah yang harus dilakukan adalah, mencari total biaya untuk memproduksi per unit:

Total cost = Fix cost + variable costTotal cost = Rp 780 + Rp 700Total cost = Rp 1.480 per unitTotal cost produksi 120.000 unit = 1.480 x 120.000 = 117.600.000

Langkah kedua, mencari harga jual:

Harga jual = 60% lebih dari total cost per unitHarga jual = 1.480 + (60% x 1480)Harga jual = 1.480 + 888Harga jual = Rp 2.368 per unit.

Langkah ketiga, mencari penjualan tiap tahun =

Tahun pertama = 90.000 x Rp 2.368 = Rp 213.120.000Tahun kedua = 40.000 x Rp 2.368 = Rp 94.720.000

Sehingga:

Laba rugi = penjualan - biayaLaba rugi tahun pertama = Rp 213.120.000 - Rp 117.600.000 =  Rp 95.520.000. Laba rugi tahun kedua = Rp 94.720.000 - Rp 117.600.000 = (Rp 22.880.000)

13. 5. Terangkan bagaimana pengaruh ukuran memori (RAM Size) terhadap utilisasi CPU ?


aplikasi yg berjalan, sebagian data disimpan kedalam memori ram untuk kemudian diproses oleh cpu. ukuran ram berpengaruh terhadap banyaknya data sementara yang tersimpan untuk dapat diproses cpu, apabila ram penuh maka data yang akan dimasukkan harus menunggu alamat kosong pada ram. sehingga dapat menghambat laju cpu dalam memproses data. dengan kapasitas ram besar, maka data yg tersimpan juga lebih banyak dan mengurangi resiko ram penuh.
