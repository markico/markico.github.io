Rata rata = Jumlah Nomor/Jumlah siswa= Sn / n

Sn = n/2 x (a + Un)
Sn = 99/2 x (1+99)
Sn = 99/2 x 100
Sn = 99 x 50
Sn = 4950

Rata rata = 4950/99 = 50
