29. Q. Permutasi dari :- Kangen- Diaa7² + 6²​


Kaidah pencacahan

Kangen

Total huruf = 6

Unsur ganda = 2

6!/2!

= 720/2

= 360 susunan

Diaa

Total huruf = 4

Unsur ganda = 2

4!/2!

= 24/2

= 12 susunan

Eksponen

7² + 6²

= 7×7 + 6×6

= 49 + 36

= 85

Jawaban:

3)Kangen : 6

Unsur Ganda : 2(n)

K = 1a = 1n = 2g = 1e = 1

Rumus : n! / r!

P = n! / r!

P = 6! / 2!

P = (6 x 5 x 4 x 3 x 2!) / 2!

P = 6 x 5 x 4 x 3

P = 30 x 12

P = 360 Susunan

2) Diaa : 4

Unsur Ganda : 2(a)

D = 1i = 1a = 2

Rumus : n! / r!

P = n! / r!

P = 4! / 2!

P = (4 x 3 x 2!) / 2!

P = 4 x 3

P = 12 Susunan

3) 7² + 6²

= 7² + 6²

= (7 x 7) + (6 x 6)

= 49 + 36

= 85
