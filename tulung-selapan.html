8. TULUNG TULUNG TULUNG ​


Jawaban:

Degrees of Comparison:

Tree 1 is shorter than tree 2.Tree 2 is Taller than tree three.Tree 1 is the shortest.

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran Degrees of Comparison (derajat perbandingan) dalam Bahasa Inggris.

Degrees of Comparison

Kita menambahkan akhiran "-er" dan "than" pada kata sifat satu suku kata dan awalan "more" dan "than" pada kata sifat lebih dari dua suku kata untuk membentuk Comparative Degree (gelar perbandingan), contohnya:

Bigger than = lebih besar daripada (satu suku kata dari "Big")

More Beautiful than = lebih cantik dari pada (empat suku kata "Beautiful")

Kita menambahkan akhiran "-est" pada kata sifat satu suku kata dan awalan "The most" pada kata sifat lebih dari dua suku kata untuk membentuk Superlative Degree (derajat superlatif), contohnya:

Biggest = terbesar (satu suku kata "Big")

The most Beautiful = tercantik (empat suku kata "Beautiful")

Semoga membantu ya.

Jadikan jawaban tercerdas ya!

Penjelasan dengan langkah-langkah:

Misalkan : Sisi / = x, sisi // = y

x + y = 39 dan 2x = y, maka :

x + y = 39

x + 2x = 39

3x = 39

x = 39/3

x = 13 cm

2x = y

y = 2 x 13

L. Balok = (2 x ((p x l) + (p x t) + (l x t))) - (s²)

= (2 x ((26 x 26) + (26 x 39) + (26 x 39)))- (13²)

= (2 x (676 + 1.014 + 1.014)) - 169

= (2 x 2.704) - 169

= 5.408 - 169

= 5.239 cm²

L. Kubus = 5 x s²

= 5 x 13²

= 5 x 169

= 845 cm²

L. Bangun = 5.239 cm² + 845 cm² = 6.084 cm² (D)

Rp 8.000,-

Penjelasan dengan langkah-langkah:

diketahui:

harga baju Rp 165.000,-

diskon 20%

membayar Rp 150.000,-

ditanya:

berapa kembalian yang di terima Rani

cara:

Rp 165.000,- x diskon 20%

= 165.000 x 20/100

= 23.000 (diskon)

Kembalian = Bayar - ( harga baju - diskon )

= 150.000 - ( 165.000 - 23.000 )

= 150.000 - 142.000

= 8.000

jadi kembalian yang di terima Rani adalah Rp 8.000,-
