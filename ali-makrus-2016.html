8 + 31 + 21 = 60 hari

jadi akan bertemu lagi di tanggal 21 April 2016 (B)


5. pada tahun 2016,perbandingan umur ali dan budi adalah 3:4 jika jumlah umur mereka 28 tahun.umur ali adalah...tahun


umur ali:
[tex] \frac{3}{7} \times 28 = 12[/tex]
jadi umur ali 12 tahunPerbandingan

Diketahui:
A : B = 3 : 4
umur A + B = 28
Ditanya:
umur A = ?

Jawab:
Umur A = A/(A + B) x umur A + B
= 3/(3+4) x 28
= 3/7 x 28
= 3 x 4
= 12 tahun

Jadi umur Ali adalah 12 tahun.

26. hitunglah hasil dari ekspresi eksponensial berikut (1^-2016 + 2^-2016 +3^-2016 + 4^-2016 + 5^-2016 +.....) ÷ (1^-2016 + 3^-2016 + 5^-2016 + 7^-2016 + 9^-2016 + ....) semuanya dipangkat negatif 2016 ya. please di jawab


[tex]S=\frac{1^{-2016}+2^{-2016}+3^{-2016}+4^{-2016}+...}{1^{-2016}+3^{-2016}+5^{-2016}+7^{-2016}+...}[/tex]

[tex]\frac{1^{-2016}+2^{-2016}+3^{-2016}+4^{-2016}+...}{1^{-2016}+3^{-2016}+5^{-2016}+7^{-2016}+...}=\frac{\left(1^{-2016}+3^{-2016}+...\right)+\left(2^{-2016}+4^{-2016}+...\right)}{1^{-2016}+3^{-2016}+5^{-2016}+7^{-2016}+...}\\=1+\frac{2^{-2016}+4^{-2016}+...}{1^{-2016}+3^{-2016}+5^{-2016}+7^{-2016}+...}\\=1+2^{-2016}\times\left[\frac{1^{-2016}+2^{-2016}+3^{-2016}+4^{-2016}+...}{1^{-2016}+3^{-2016}+5^{-2016}+7^{-2016}+...}\right]\\=1+2^{-2016}S[/tex]

[tex]S=1+2^{-2016}S\\\left(1-2^{-2016}\right)S=1\\S=\frac{1}{1-2^{2016}}[/tex]
