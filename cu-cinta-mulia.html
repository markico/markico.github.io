5. ada dua isotop cu-63 dan cu-65 jika Ar cu=63,5 berpa kwlimpahan cu di alam


Materi : Stoikiometri

Isotop-63 Cu : a%
Isotop-65 Cu : (100-a)%

a% × 63 + (100-a)% × 65 = 63.5
2a = 150
a = 75

Kelimpahan :
Isotop-63 Cu = 75%
Isotop-65 Cu = 25%

8. Tembaga di alam terdapat dalam 2 isotop, yaitu Cu-63 dan Cu-65. Jika massa atom relatif (Ar Cu=63,5) maka kelimpahan Cu-63 dan Cu-65 masing" adalah.......%dan........%


Misal : Kelimpahan Cu-63 = x

KElimpahan Cu-65 = 100-x

---------------------------------------------------------

63. x + 65 (100-x) / 100 = 63.5

-2x +6500 = 6350

-2x = -150

x = 75

---------------------------------------------------

Kelimpahan Cu -63 = 75%

Kelimpahan Cu -65 = 25%

misalkan Cu-63 memiliki kelimpahaan sebesar x% dan Cu-65 (100-x)%

(63x + 6500 - 65x)% = 63,5
6500 - 2x = 63,5 .100
6500 - 2x = 6350
6500 - 6350 = 2x
150 = 2x
 x = 75
