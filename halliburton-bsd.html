6. Pada Gambar di Bawah, BSD=100°, dan BTD=40° Hitunglah: Besar SAD dan SBC --------------------------------------------------- Pakai Cara Auto Jawaban Terbaik!!


Besar sudut SAD adalah 50°. Sedangkan besar sudut SBC adalah 50°

Penjelasan dengan langkah-langkahDiketahui: Sebuah gambar dengan besar sudut BSD = 100°, dan sudut BTD = 40°Ditanya: Tentukan besar sudut SAD dan SBC!Jawab:

Langkah 1

Besar sudut BSD = 100°

Maka, besar sudut DSA = 80°

Langkah 2

Maka, besar sudut SAD = 180° - 80° - 50°

= 50°

Besar sudut BSC = 80°

Maka, besar sudut SCB = 50°

19. Ayya bersama keluarganya berangkat dari BSD pukul 06.000 WIB menuju Bandung. mobil ayya melaju dengan kecepatan 80 km/jam. jarak kora BSD dan bandung adalah 320 km. Pukul berapa Ayya tiba di bandung?


Jawaban:

72km/jam

Penjelasan dengan langkah-langkah:

Diketahui :

Waktu = 06.30 ke 09.30 = 3 jam

3 jam - (20+10) = 2,5 jam

Jarak = 180 KM

Ditanya :

Kecepatan rata-rata =?

Jawab :

Kecepatan = jarak ÷ waktu

= 180km ÷ 2,5jam

= 72 km/jam

Jadi, kecepatan rata-rata mereka adalah 72 km/jam

24. Fajar Mengendarai Mobil Dari Jakarta Ke BSD City Dengan Kecepatan 150 km/j Dan Dengan Waktu Tempuh 5 menit. Tentukan Jarak Dari Jakarta Ke BSD City Dan Ubahlah Ke Bentuk Milimeter / mm


Jawab:

Untuk menentukan jarak dari Jakarta ke BSD City, kita dapat menggunakan rumus jarak = kecepatan x waktu.

Jarak = 150 km/jam x 5 menit / 60 menit/jam = 12.5 km

Untuk merubah jarak dari km ke mm, kita dapat menggunakan konversi satuan jarak yaitu 1 km = 1,000,000 mm

Jarak dalam mm = 12.5 x 1,000,000 mm = 12,500,000 mm

Jadi jarak dari Jakarta ke BSD City adalah 12,500,000 mm atau 12.5 km

30. bagaimana cara melihat Ip Address di Log History di Free BSD ?


coba kamu ketikkan command berikut di terminal:
ifconfig

semoga membantu
Langkah pertama adalah masuk sebagai super user, dan ketikkan perintah sysinstall seperti dibawah ini:



Maka akan terlihat main menu seperti gambar dibawah, kemudian pilih Configure :



Selanjutnya pilih Networking,



Selanjutny kita akan dibawa ke banyak pilihan pada menu networking, pilih Interface seperti dibawah ini:



Setelah dipilih menu Interface, maka akan muncul pop-up window yang menenyakan apakah kita akan mencoba konfigurasi IPv6, pilih No karna kita akan mengkonfigurasi IP static v4



Setelah itu, akan muncul juga pop-up window kedua yang menanyakan apakah kita akan moncoba konfigurasi DHCP, pilih No juga karna kita akan mengkonfigurasi secara static.



Setelah itu akan muncul window untuk mengisi konfigurasi IP seperti dibawah ini. Untuk konfigurasi, alamat IP disesuaikan dengan IP yang akan anda isikan.



Jika sudah selesai, arahkan crusor dan pilih tombol Ok. Maka akan keluar pop-up windows yang menanyakan apakan kita akan mengaktifkan interface yang baru kita konfigurasi IPnya, pilih Yes.



Selanjutnya keluar dari menu sysinstall dengan mengarahkan crusor ke exit (biasanya berapa dipaling atas), dan pilih exit. Ulangi lagi sampai anda temui main menu sysinstall. Kemudian pilih Exit Install, maka anda akan kembali ke mode console FreeBSD. Dan untuk mengetahui apakah konfigurasi kita sudah berhasil, maka cek IP interface dengan mengetikkan perintah ifconfig.



Selesai.
