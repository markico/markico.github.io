Jawaban dan Pembahasan

Golekana batange tembung cangkriman ing ngisor iki!

a. Tuwan sinyo.

➡️ Untu kedawan gusi menyonyo.

b. Emboke wuda, anake tapihan.

➡️ Rebung/Pring. (Wit bambu lan anake)

c. Wong mati dilupampati kucing urip.

➡️ Sing urip kucinge, wonge tetep mati.

Penjelasan

Cangkriman yaiku unen-unen sing kudu dibedhek utawa dibatang maksude. Mula saka iku, cangkriman uga diarani bedhekan utawa batangan.

Cangkriman iku wujude ana 4, yaiku:

1. cangkriman sing wujude wancahan (cekakan),

2. cangkriman sing wujude pepindhan (irib-iriban),

3. cangkriman sing wujude plesetan, lan

4. cangkriman sing sinawung ing tembang.

Cangkriman Wancahan :

1. Gerbong tulis = pager Kobong watune mendhelis.

2. Burnas kopen = bubur panas kokopen.

3. Gowang pelot = jagoane ana Lawang cempene mencolot.

4. Buta buri = tebu ditata mlebu lori.

5. Tuwan sinyo = untu kedawan gusi menyonyo.

Cangkriman Blenderan :

1. Gajah numpak becak ketok apane?

➡️ ketok ngapusine.

2. Suru supaya bisa mlayu dikapakne?

➡️ digepuk

3. Tulisane arab, macane saka ngendi?

➡️ macane saka alas.

(macan = jenenge kewan )

4. Wong dodol klapa dikepruki.

➡️ sing dikeproki klapane, dudu wonge.

5. Wong dodol tempe di taleni.

➡️ sing ditaleni tempene, dudu wonge.

Cangkriman pepindhan :

1. Pitik walik sobo kebon.

➡️ tegese nanas.

2. Bocah cilik ngendong omah.

➡️ tegese bekecot.

3. Dikethok malah tambah dhuwur.

➡️ tegese clana.

4. Lawa lima,kalong telu dadi pira?.

➡️ 8 (5 + 3 ) = 8

5. Sega sakepel dirubung tinggi.

➡️ tegese salak.
