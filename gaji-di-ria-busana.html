5 September 2020

Retur Penjualan Rp6.000.000 ( D )

10 September 2020

Kas Rp8.730.000 ( D )

Potongan Penjualan Rp270.000 ( D )

Piutang Dagang Rp9.000.000 ( K )

Pembahasan : 3 September 2020, menjual barang secara kredit maka akan mempengaruh akun piutang dagang bertambah di debit dan penjualan di kredit.
