8. hasil penjualan x ayam dinyatakan dalam bentuk 130x-x2


Hasil penjualan maksimum ayam (dalam ribuan rupiah) adalah Rp4.225.000,00.

Penjelasan dengan langkah-langkah:

Soal tidak lengkap.

Penjualan akan mencapai maksimum bila turunan pertama dari fungsi penjualannya sama dengan nol.

Diketahui:

Hasil penjualan x ayam dinyatakan dalam bentuk fungsi f(x) = 130x - x² (dalam ribuan rupiah)

Ditanyakan:

Hasil penjualan maksimum (dalam ribuan rupiah) yang diperoleh.

Jawab:

f(x) = 130x - x²

f'(x) = 130 - 2x

Syarat maksimum bila f'(x) = 0

f'(x) = 0

⇔ 130 - 2x = 0

⇔ 2x = 130

⇔ x = 65

Hasil penjualan maksimum (dalam ribuan rupiah) saat x = 65, sehingga

f(65) = 130(65) - (65)²

⇔ f(65) = 8.450 - 4.225

⇔ f(65) = 4.225

Jadi, hasil penjualan maksimum ayam (dalam ribuan rupiah) adalah Rp4.225.000,00.
