Jawaban:

8.700.000 X 5 % = 435.000 ( Pajak Dalam 1 Thn )

435.000 : 12 = 36.250 ( Pajak Dalam 1 Bln )

Penjelasan:

Penambah

Gaji Pokok : 6.000.000

Penghasilan Bruto : 6.000.000

Pengurang

Biaya Jabatan : 300.000 (Max 500Rb/Bln Atau 6Jt/Thn)

Iuran Pensiun : 100.000

Penghasilan Neto : 5.600.000

Penghasilan Dalam 1 Thn : 5.6 Jt X 12 =67.2 Jt

PTKP K/0 58.500.000 - 67.200.000 = 8.700.0000

Tarif 1    5%

8.700.000 X 5 % = 435.000 ( Pajak Dalam 1 Thn )

435.000 : 12 = 36.250 ( Pajak Dalam 1 Bln )

Semoga Membantu & Jadikan Jawaban Tercerdas

Catatan Untuk Kedepanya

Tarif Ptkp (Note = K(Kawin), TK(Tidak Kawin)

K/0  58.500.000

K/1  63.000.000

K/2  67.500.000

K/3  72.000.000

TK/0  54.000.000

TK/1  58.500.000

TK/2  63.000.000

TK/3  67.500.000

Tarif Progresif

0 < PKP < =  60.000.000  5%

60.000.000  < PKP < =  250.000.000  15%

250.000.000  < PKP < =  500.000.000  25%

500.000.000  < PKP   -  30%

Biaya Jabatan  5%

29. Tommy bekerja di Universitas Nusantara. Ia memperoleh gaji sebulan berupa gaji pokok sebesar Rp. 6.000.000,00/bulan. Tommy juga membayar iuran pensiun sebesar Rp. 100.000/bulan. Tommy sudah menikah tapi belum mempunyai anak. Berapakah pajak terutang Tommy tiap tahun?​


Penghasilan sebulan :

Gaji = Rp6.000.000,00

Jumlah Penghasilan Bruto Sebulan = Rp6.000.000,00

Pengurangan :

Biaya Jabatan = Rp300.000,00Iuran pensiun = Rp100.000,00

Jumlah Pengurangan = Rp400.000,00

Penghasilan neto sebulan = penghasilan bruto – total pengurang

= Rp6.000.000,00 - Rp400.000,00

= Rp5.600.000,00

Penghasilan neto setahun = Rp67.200.000,00

PTKP :

Wajib pajak sendiri = Rp54.000.000Tambahan karena menikah = Rp4.500.000

Total PTKP = Rp58.500.000,00

PKP = Penghasilan neto setahun – PTKP

PKP = Rp67.200.000,00 - Rp58.500.000,00

PKP = Rp8.700.000,00

PPh pasal 21 terutang :

5% × Rp8.700.000,00 = Rp435.000,00

Jumlah PPh pasal 21 terutang setahun = Rp435.000,00

PPh pasal 21 dipotong sebulan = Rp36.250,00
