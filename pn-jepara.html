1. Jika Pn = 729, suku ke berapakah Pn?​


Jawaban:

jika ini adalah barisan bilangan ganjil mulai dari 1,3,5,7,9, ... ,

dengan rumus suku ke-n adalah Un=2n-1 dan jumlah suku ke-n adalah Sn=n²

n²=729

n=√729

n=27

Jadi Pn adalah suku ke-27 dengan nilai 2(27)-1=53

5. pola bilangan persegi panjang jika Pn = 552 suku keberapa Pn ?


Matematika
Pola Bilangan VIII SMP

Pembahasan :
Pola Persegi Panjang

Pn = 552
n(n + 1) = 552
n² + n = 552
n² + n - 552 = 0
(n - 23)(n + 24) = 0
n = 23 —» Pada suku ke-23

15. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722

20. a.jarak jepara dangan semarang pada peta 20,5cm b.jarak jepara dengan blora pada peta 22cm berapa selisih sesungguhnya jepara - semarang dengan jepara - blora​


Selisih jarak sebenarnya antara Jepara - Semarang dengan Jepara - Blora adalah 5,25 km.

Pembahasan

Diketahui:

Jarak Jepara - Semarang pada peta = 20,5 cm

Jarak Jepara - Blora pada peta = 22 cm

Ditanya:

Selisih jarak Jepara - Semarang dengan Jepara - Blora sebenarnya = .... ?

Jawab:

Ingat bahwa pada hubungan antara skala, jarak pada peta dan jarak sebenarnya berlaku:

                Skala = Jarak pada peta : Jarak sebenarnya

Dimisalkan skala pada soal di atas adalah 1 : 350.000

artinya 1 cm pada peta mewakili 350.000 cm pada keadaan sebenarnya.

Jarak Jepara - Semarang sebenarnya = jarak pada peta x skala

Jarak Jepara - Semarang sebenarnya = 20,5 x 350.000

Jarak Jepara - Semarang sebenarnya = 7.175.000 cm

Jarak Jepara - Semarang sebenarnya = 71,75 km

Jarak Jepara - Blora sebenarnya = jarak pada peta x skala

Jarak Jepara - Blora sebenarnya = 22 x 350.000

Jarak Jepara - Blora sebenarnya = 7.700.000 cm

Jarak Jepara - Blora sebenarnya = 77 km

Selisih jarak sebenarnya antara Jepara - Semarang dan Jepara - Blora adalah 77 km - 71,75 km = 5,25 km.
