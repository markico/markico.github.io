4. jika pn = 729 suku keberapakah pn


jika ini adalah barisan bilangan ganjil mulai dari 1,3,5,7,9, ... ,
dengan rumus suku ke-n adalah Un=2n-1 dan jumlah suku ke-n adalah Sn=n²
n²=729
n=√729
n=27
Jadi Pn adalah suku ke-27 dengan nilai 2(27)-1=53

14. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722

20. Sebuah peta pualu bacan di buat dengan skala 1:1.000.000 jika jarak kupal labuha pada peta 18 cm.maka jarak sebernanya adalah

_____


Skala = 1 : 1.000.000
Jarak peta = 18 cm
Jarak sebenarnya
= Jarak peta/Skala
= 18 cm/1/1.000.000
= 18cm x 1.000.000
= 18.000.000 cm
= 180 km

⚫ PEMBAHASAN ⚫
Jarak sebenarnya
= 18 × 1.000.000
= 18.000.000cm

Matematika
Pola Bilangan VIII SMP

Pembahasan :
Pola Persegi Panjang

Pn = 552
n(n + 1) = 552
n² + n = 552
n² + n - 552 = 0
(n - 23)(n + 24) = 0
n = 23 —» Pada suku ke-23
