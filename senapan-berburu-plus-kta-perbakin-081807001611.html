10. tolong bantu jawab..mobil pemburu melaju dengan kecepatan 72 km/jam dan rusa berlari dengan kecepatan 64,8 km/jam. ketika jarak antara mobil dan rusa 202 m, sang pemburu menembakkan senapannya. peluru keluar dari laras senapan dengan laju 200 m/s terhadap laras senapan. peluru mengenai rusa setelah berapa detik?


Mobil pemburu melaju dengan kecepatan 72 km/jam dan rusa berlari dengan kecepatan 64,8 km/jam. Ketika jarak antara mobil dan rusa 202 m, sang pemburu menembakkan senapannya. Peluru keluar dari laras senapan dengan laju 200 m/s terhadap laras senapan. Peluru akan mengenai rusa setelah 1 detik

PEMBAHASAN

Gerak lurus dapat di bedakan menjadi dua macam yakni:

Gerak Lurus Beraturan (GLB)

Gerak ini terjadi saat benda bergerak lurus dengan kecepatan tetap. Rumus untuk gerak ini adalah :

v = s / t

dimana :

v = kecepatan benda ( m/s )

s = jarak tempuh benda ( m )

t = waktu tempuh ( s )

Gerak Lurus Berubah Beraturan (GLBB)

Gerak ini terjadi saat benda bergerak lurus dengan kecepatan yang senantiasa berubah secara teratur. Dengan kata lain benda bergerak dengan percepatan tetap. Rumus untuk gerak ini adalah :

a = ( Vt - Vo ) / t

s = ( Vt + Vo ) . t / 2

s = Vo. t + (1/2) a . t²

( Vt )² = ( Vo )² + 2.a.s

dimana :

a = percepatan benda ( m/s² )

Vt = kecepatan akhir benda ( m/s )

Vo = kecepatan awal benda ( m/s )

t = waktu tempuh benda ( m/s )

s = jarak tempuh benda ( m )

Okay marilah kita gunakan prinsip ini untuk menyelesaikan soal yang di maksud.

Diketahui:

Vmobil = 72 km/jam = 20 m/s

Vrusa = 64,8 km/jam = 18 m/s

s = 202 m

Vpeluru terhadap laras = 200 m/s

Ditanya:

t = ?

Penyelesaian:

Vpeluru terhadap tanah = 200 + 20 = 220 m/s

Misalkan persoalan ini kita gambarkan seperti terlihat pada gambar di lampiran. Anggap peluru mengenai rusa pada titik C maka :

BC + 202 = AC

18t + 202 = 220t

202 = 220t - 18t

202 = 202t

t = 202/202

t = 1 detik

13. seorang pemburu menembakan senapannya. sesaat setelah peluru keluar dari moncong senapan dengan kecepatan 50m/s. jika massa peluru 0.02kg dan massa senapan 4kg hitung kecepatan senapan sesaat setelah di tembakan?​


Kecepatan senapan sesaat setelah ditembakkan adalah 0,25 m/s ke arah kiri atau berlawanan dengan arah gerak peluru saat ditembakkan.

Pembahasan

Hukum kekekalan momentum berbunyi, "Jika tida ada gaya luar yang bekerja pada sistem dalam peristiwa tumbukan, momentum total sistem sebelum tumbukan sama dengan momentum total sistem sesudah tumbukan."

Secara matematis, hukun kekekalan momentum dirumuskan sebagai berikut.

p1 + p2 = p1' + p2'

→ m1 · v1 + m2 · v2 = m1 · v1' + m2 · v2'

Keterangan.

p = momentum sebelum tumbukan (kgm/s)

p' = momentum setelah tumbukan (kgm/s)

m = massa benda (kg)

v = kecepatan benda sebelum tumbukan (m/s)

v' = kecepatan benda setelah tumbukan (m/s)

Penyelesaian Soal

Diketahui:

vp' = 50 m/s

mp = 0,02 kg

ms = 4 kg

Ditanya:

vs' = ...

Penyelesaian:

Berdasarkan hukum kekekalan momentum,

p1 + p2 = p1' + p2'

m1 · v1 + m2 · v2 = m1 · v1' + m2 · v2'

mp · vp + ms · vs = mp · vp' + ms · vs'

Karena sebelum tembakan vp = vs = 0, maka

0 = mp · vp' + ms · vs'

0 = 0,02 · 50 + 4 · vs'

0 = 1 + 4 · vs'

-1 = 4 · vs'

vs' = -1/4

vs' = -0,25 m/s

Jadi kecepatan senapan sesaat setelah ditembakkan adalah 0,25 m/s ke arah kiri atau berlawanan dengan arah gerak peluru saat ditembakkan.

Seorang pemburu mengejar rusa menggunakan mobil. Kecepatan mobil yaitu 72 km/jam dan rusa berlari dengan kecepatan 64,8 km/jam. Pemburu menembak rusa saat jarak antara mobil dan rusa 202 meter. Peluru keluar dari laras senapan dengan laju 200 m/s terhadap laras senapan. Peluru mengenai rusa setelah waktu 1 sekon.

Pembahasan dengan langkah-langkah

Pembahasan dengan langkah-langkahDiketahui:

Kelajuan mobil (vb) = 72 km/jam = 20 m/sKelajuan rusa (vr) = 64,8 km/jam = 18 m/sJarak antara mobil dan rusa (s) = 202 meterKelajuan peluru (vp) = 20 m/s + 200 m/s = 220 m/s. (Senapan berada di dalam mobil yang bergerak, sehingga kelajuan peluru ditambahkan dengan kelajuan mobil).

Ditanya: 

Selang waktu peluru mengenai rusa.

Dijawab:

Kita asumsikan mobil dan rusa bergerak lurus dengan kecepatan tetap.

v = s ÷ t atau s = v × t

dimana,

v = kelajuan

s = jarak

t = waktu

Jarak tempuh rusa = 202 + Xr = 202 + vr×t = 202 + 18t

Jarak tempuh peluru = Yp = vp×t = 220t

Ketika peluru mengenai rusa, jarak tempuh peluru = jarak tempuh rusa.

Jarak tempuh rusa = Jarak tempuh peluru

202 + 18t = 220t

202 = 220t – 18t

202 = 202t

t = 202/202

t = 1 sekon
