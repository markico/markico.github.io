18. pak tarno akan membuat meja kayu berbentuk persegi panjang dengan keliling mena tidak boleh kurang dari 7 meter. jika selisih panjang dah lebarnya adalah 50cm, maka luas minimjm meja tersebut adalah..


Pak tarno akan membuat meja kayu berbentuk persegi panjang dengan keliling meja tidak boleh kurang dari 7 meter. Jika selisih panjang dan lebarnya adalah 50cm, maka luas minimum meja tersebut adalah 3 m²

PEMBAHASAN :

Rumus-rumus penting untuk menyelesaikan soal ini adalah :

Luas Persegi Panjang = Panjang x Lebar

Keliling Persegi Panjang = 2 x ( Panjang + Lebar )

Okay mari kita artikan soal ceritanya ke dalam persamaan matematika.

Keliling meja tidak boleh kurang dari 7 meter itu artinya Keliling minimal meja adalah 7 meter atau 700 cm.

Selisih panjang dan lebarnya adalah 50 cm , bisa di tuliskan menjadi :

Panjang - Lebar = 50

→ Panjang = 50 + Lebar

Kita masukkin hubungan ini ke dalam rumus Keliling Persegi Panjang berikut:

Keliling Persegi Panjang = 2 x ( Panjang + Lebar )

700 = 2 x ( 50 + Lebar + Lebar )

700/2 = 50 + 2xLebar

350 = 50 + 2xLebar

350-50 = 2xLebar

300 = 2x Lebar

Lebar = 300/2 = 150 cm

Artinya Lebar minimal persegi panjang adalah 150 cm

Panjang = 50 + Lebar

Panjang = 50 + 150

Panjang = 200 cm

Artinya Panjang minimal persegi panjang adalah 200 cm

Akhirnya kita bisa mendapatkan Luas minimalnya dengan cara sebagai berikut:

Luas Persegi Panjang = Panjang x Lebar

Luas Persegi Panjang = 200 x 150

Luas Persegi Panjang = 30.000 cm² = 3 m²

Kesimpulannya Luas Persegi Panjang minimalnya adalah 3 m²
