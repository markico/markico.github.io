Asril membeli 2 buku, 2 pensil dan 2 pulpen dengan harga Rp 20.000,00 dan Yusuf membeli 3 buku, 4 pensil dan 3 pulpen dengan harga Rp 32.000,00. Jika kamu akan membeli 5 buku, 10 pensil dan 6 pulpen dan membayarnya dengan uang Rp 100.000,00, uang kembalian yang akan kamu terima adalah​


Buku = x

Pensil = y

Pulpen = z

Yudi ==> 2x + 3y + z = 17.000

Asril ==> 2x + 2y + 2z = 20.000

                  x + y + z = 10.000

Yusuf ==> 3x + 4y + 3z = 32.000

2x + 3y + z = 17.000

  x +   y + z = 10.000 -

  x + 2y       =   7.000

2x + 3y +    z = 17.000  (x3)

6x + 9y + 3z = 51.000

3x + 4y + 3z = 32.000 -

3x + 5y          = 19.000

 x + 2y     =   7.000  (x3)

3x + 6y    = 21.000

3x + 5y    = 19.000 -

y = 2.000

x + 2y     =   7.000

x + 2(2.000) = 7.000

x = 7.000 - 4.000 ⇔ x = 3.000

x + y + z = 10.000

3.000 + 2.000 + z = 10.000

z = 10.000 - 5.000 ⇔  z = 5.000
