19. If you are able to do that job,A. so am IB. so do IC. so will ID. so mustE. so canalasan!! p3njelasan!​


If you are able to do that, so am I

Answer : a. so am I

Penjelasan :

Pola penggunaan So :

So + auxiliary + Subject

Kalimat utamanya menggunakan are : If you are able to do that

Jadi, selanjutnya so juga menggunakan to be

Dan karena Subject nya adalah I, maka to be yang digunakan adalah am

K.[O2] = ([SO3]/[SO2])^2

[SO2]/[SO3] = 1/[O2]^1/2 = [O2]^(-1/2)
