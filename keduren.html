27. kembang duren diarani​


Jawaban:

, 250.390

=============================

Penjelasan dengan langkah-langkah:

Barisan Aritmetika

\boxed{\boxed{Un = a + (n-1)b}}

Un=a+(n−1)b

Deret Aritmetika

Jika diketahui suku ke-n

\boxed{\boxed{Sn = \frac{n}{2}(a + Un)}}

\boxed{\boxed{Sn = \frac{n}{2}(2a + (n-1)b)}}

Keterangan :

Sn = jumlah suku ke-n pada suatu barisan aritmetika

n = banyak suku

a = suku pertama (U1)

Un = suku ke-n

b = selisih/beda = Un - U(n-1)

=============================

Diketahui :

(-20) + (-18) + (-16) + ... + 1000

a = -20

Un = 1000

Ditanya :

Tentukan jumlahnya!

Jawab :

Mencari beda/selisih

b = Un - U(n-1)

b = -18 - (-20)

b = -18 + 20

b = 2

Suku ke-n

Un = a + (n - 1)b

1000 = -20 + (n - 1) . 2

1000 = -20 + 2n - 2

1000 = -22 + 2n

2n = 1000 + 22

2n = 1022

n = 1022/2

n = 511

Jumlah deret aritmetika

Sn = \frac{n}{2}(a + Un)Sn=

2

n

(a+Un)

Sn = \frac{511}{2}(-20 + 1000)Sn=

2

511

(−20+1000)

Sn = \frac{511}{2}(980)Sn=

2

511

(980)

Sn = 511(490)Sn=511(490)

\boxed{\bold{\blue{Sn = 250.390}}}

Sn=250.390

Jadi, jumlah dari deret aritmetika tersebut adalah 250.390
