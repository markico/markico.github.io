22. Quiz585² - 323 =87²​


» Pendahuluan ༶

ー Bilangan Pangkat ー

[tex]\tt { \red { { perpangkatan }}}[/tex]adalah sebuah bentuk operasi dalam ilmu matematika yang boleh digunakan apabila sebuah bilangan dikali dengan bilangan yang sama lebih dari satu kali. Pangkat sendiri merupakan banyaknya bilangan yang dikali dengan bilangan yang sama. Letak pangkat sendiri berada diatas bilangan yang dikalikan tersebut

༶Pembahasan༶

[tex]\tt { \pink { { bilangan \: berpangkat }}}[/tex]merupakan perkalian berulang suatu bilangan, dimana bilangan dapat berpangkat bulat positif, nol, maupun bulat negatif. Secara sederhana penulisan bilangan jenis ini adalah sebagai berikut :

[tex]\bf \purple{ {a}^{n} = a \times a \times a... \times a}[/tex]

a disebut bilangan pokok atau basis, sedangkan n disebut pangkat atau eksponen

✎ Contoh Bilangan Berpangkat ✎

Berikut adalah contoh bilangan yang berpangkat :

2³ = 2 × 2 × 2

Maksudnya adalah angka dua dikalikan dengan angka yang sama sebanyak tiga kali dan menghasilkan bilangan dua pangkat tiga.

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 4

3² = ( 3 x 3 ) = 9

4² = ( 4 x 4 ) = 16

5² = ( 5 x 5 ) = 25

[tex]\:[/tex]

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 8

3³ = ( 3 x 3 x 3 ) = 27

4³ = ( 4 x 4 x 4 ) = 64

5³ = ( 5 x 5 x 5 ) = 125

26. 585:13 dengan caranya​


Jawaban:

45

Penjelasan dengan langkah-langkah:

[tex]585 \: \div \: 13 \: = \: 13 \: \times \: 45 \: = \: 585[/tex]

[tex]\huge\colorbox{black}{\color{magenta}{\boxed{\pink{\mathscr{☯ \:Geng \: Pintar☯}}}}}[/tex]

585 : 13 = 45

Cara :

→Pertama,kalikan 13 yang menyamai/mendekati angka 58 yaitu dikali 4 = 52

→Langkah selanjutnya,kurangi 58-52 = 6,kemudian turunkan 5 menjadi 65

→Kalikan 13 yang menyamain/mendekati angka 65 yaitu dikali 5

→Sisa nya 0 berarti pembagian sudah habis dibagikan

Hasilnya 45
