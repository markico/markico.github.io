9. 1. Jelaskan pengertian beberapa jenis file dibawah ini berdasarkan ekstensinya:File Sistemb. File VideoC. File Dokumend. File Gambare. File Suaraf. File Archive​


Jawaban:

A. Pengertian file sistem : struktur logika yang digunakan untuk. mengendalikan akses terhadap data yang ada pada harddisk. q

B. Pengertian dari file video : berkas komputer yang digunakan untuk menyimpan kumpulan berkas digital seperti video, audio, metadata, informasi, pembagian chapter, dan judul sekaligus, yang dapat dimainkan atau digunakan melalui perangkat lunak tertentu pada komputer.

C. Pengertian dari file dokumen : File yang mengandung informasi tertentu dan dapat dibuka dengan program.

D. Pengertian dari file gambar : cara terstandarisasi untuk mengatur dan menyimpan gambar digital.

E. Pengertian dari file suara : jenis file yang merekam atau menyajikan bunyi, bisa berupa lagu, rekaman radio, rekaman telepon dan lain sebagainya.

F. Pengertian dari file archive : tipe file yang terdiri dari file-file lain yang sudah dimampatkan ukurannya (Compressed).
