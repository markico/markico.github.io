Berdasarkan data di atas, pendapatan nasional bisa dihitung dengan pendekatan pendapatan maupun pengeluaran

Untuk pendekatan pengeluaran :
Y = C + I + G + (X - M)
   = 23.000.000 + 7.500.000 + 21.000.000 + (15.000.000 - 5.500.000)
   = 51.500.000 + 9.500.000
   = 61.000.000

*ket : C = Konsumsi
           I = Investasi
         G = Konsumsi pemerintah
         X = Ekspor
         M = Impor


Untuk pendekatan pendapatan :
Y = r + w + i + p
   = 11.500.000 + 25.000.000 + 7.500.000 + 17.000.000
   = 61.000.000

*ket : R = Sewa
        W = Upah
