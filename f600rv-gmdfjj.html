4. sebuah transformator penurun tegangan dihubungkan dengan sumber tegangan 6000v, dan kuat arus pada kumparan primer 5a. jika pada daya sekunder 27000 watt, hitung efisiensi transformator tersebut!


diket :
Vp = 6000 v
Ip = 5 A
Ps = 27000 watt
dit : efisiensi trafo
jwb :
# Daya primer
Pp = Ip . Vp
Pp = 5 . 6000
Pp = 30.000 watt
# Efisiensi trafo
ef = ( Ps / Pp ) . 100%
ef = ( 27.000 / 30.000). 100%
ef = 90 %
Jadi efisiensi trafo nya 90%

13. kumparan primer sebuah trafo 8kali dari kumparan sekundernya. trafo dihubungkan pada sumber tegangan AC 200volt. kuat arus lilitan sekunder 4A, tegangan sekunder dan arus pada kumparan primer ? a. 1600V dan 0.5A b. 1600V dan 16A c. 44V dan 0.5A D. 44V dan 16A


Diket :
Np = 8 . Ns = Np : Ns = 8 : 1
Vp = 200 v
Is = 4 A

Dit : Vs & Ip ........?

Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 1 . 200 ) /8
Vs = 25 volt

# Arus Primer

Ip = ( Vs . Is ) / Vp
Ip = ( 25 . 4 ) / 200
Ip = 100 / 200
Ip = 0,5 A

Jadi tegangan sekundernya 25 v
dan arus primernya 0,5 A
