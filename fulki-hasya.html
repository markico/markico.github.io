8. jwaban dari hasya (buy/not) the bike if it (be) very expensif


conditional type I --> s + will + v IF simple present

Hasya WILL NOT BUY the bike if it IS very expensive.

conditional type II --> s + would + v IF simple past

Hasya WOULDN'T buy the bike if it WERE very expensive.

conditional type III --> s + would have + past participle IF past perfect

Hasya WOULDN'T HAVE BOUGHT the bike if it HAD BEEN very expensive.

12. Risa; will hasya buy the book? Lutfi; if the book ..... Very expensive ,she wo 't buy it. A.is B.was. C are D.had E.has


Jawaban:

A. is

PEMBAHASANConditional Sentence Type 1Digunakan untuk kondisi yg Mungkin akan terjadi

Rumus Conditional Sentence Type 1IF + Simple Present + Simple FutureSimple Future + IF + Simple Present

Berdasarkan pilihan ganda

A. Is

Tobe Present untuk Subjek tunggal

B. was

Tobe Past untuk Subjek tunggal

C. are

Tobe Present untuk Subjek jamak

D. had

Verb-2

E. has

Verb-1

Berdasarkan Rumus

IF + Simple Present + , + Simple Future

IF [the book is very expensive] , [she won't buy it]

Keterangan:

=> Simple Present

S + Tobe (is/am/are) + Complementthe book is Very expensive

=> Simple Future

S + won't + Verb-1 + Complementshe won't buy it

KESIMPULAN

Risa: Will hasya buy the book?

Akankah Hasya beli buku itu?

Lutfi: if the book is very expensive. She won't buy it.

Jika buku itu sangat mahal, dia tidak akan membelinya.

Catatan:

Menggunakan IS karena Subjek tunggal (the book)
