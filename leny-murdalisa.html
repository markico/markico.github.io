Penjelasan dengan langkah-langkah:

Diketahui bahwa

Dita : Leny = 3 : 2

Dita : Rafia = 4 : 5

Jumlah uang Leny dan Rafia adalah Rp46.000.

Bisa ditulis Leny + Rafia = 46.000

Pertanyaannya adalah

Selisih uang Dita dan Rafia adalah...

Jawaban :1) Tentukan nilai Leny : Rafia

Leny : Dita = 2 : 3

Rafia : Dita = 5 : 4

Leny : Rafia = (Leny : Dita) : (Rafia : Dita)

= (2 : 3) : (5 : 4) = (2 × 4) : (3 × 5) = 8 : 15

⇒ Leny : Rafia = 8 : 15

⇒ Leny = (8 : 15) × Rafia

2) Tentukan banyak uang Rafia

Leny + Rafia = 46.000

(8/15)Rafia + Rafia = 46.000

(23/15)Rafia = 46.000

Rafia = 46.000(15/23)

Rafia = 30.000

3) Tentukan selisih uang Dita dan Rafia

Dita : Rafia = 4 : 5

Dita = (4 : 5) × Rafia

Selisih uang Dita dan Rafia = | Dita - Rafia |

= | 4/5 × Rafia - Rafia |

= | -1/5 × Rafia |

= 1/5 × Rafia = 1/5 × 30.000 = 6.000

Jadi, jawabannya adalah Rp6.000

30. bu leny memiliki 2 jenis piring jumlah semuanya 8 lusin banyak piring jenis pertama dari banyak piring jenis kedua 5:7 berapa banyak masing masing jenis piring itu​


Jawab:

Piring Jenis Pertama = 40 piringPiring Jenis Kedua = 56 piring

Penjelasan dengan langkah-langkah:

Diketahui

Total Piring = 8 lusin = 96 piringTotal Perbandingan Piring = 5x + 7x = 12 x

Total Piring = Total Perbandingan Piring

96 = 12x  ⇒ x = 8

Piring Jenis Pertama = 5x = 40 piringPiring Jenis Kedua = 7x = 56 piring

Penjelasan dengan langkah-langkah:

jumlah piring 8 lusin

= 8 x 12 = 96 buah

perbandingan piring 1 : piring 2 = 5 : 7

* jumlah piring 1

= 5/(5+7) x 96

= 5/12 x 96

= 40 buah

* jumlah piring 2

= 7/(5+7) x 96

= 7/12 x 96

= 56 buah

jadi piring pertama 40 buah dan piring kedua 56 buah
