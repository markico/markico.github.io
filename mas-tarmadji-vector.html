[tex]Diketahui\\a = 3i + 4j\\b = 2i + 3j\\Ditanya\\|a + b|\\Jawab\\a + b = (3i + 4j)+(2i + 3j)\\a + b = (3i + 2i)+ (4j + 3j)\\a + b = 5i + 7j\\[/tex]

27. Diketahui vector a = 4i – 2j + 2k dan vector b = 2i – 6j + 4k. Proyeksi vector orthogonal vector a pada vector b adalah .....


vektor a = 4i – 2j + 2k

vektor b = 2i – 6j + 4k

a . b = (4).(2) + (-2).(-6) + (2).(4) = 8 + 12 + 8 = 28

|b|² = 2² + [-6]² + 4² = 4 + 36 + 16 = 56

Proyeksi vector orthogonal vector a pada vector b :

= ( [a . b] / |b|² ) . vektor b

= (28 / 56) . vektor b

= ½ . vektor b

= ½ . (2i – 6j + 4k)

= i - 3j + 2k
