• Mencari besar sudut PAC

sudut pusat => 2 x sudut keliling

<APB = 2 x <ACB

56 = 2 x < ACB

<ACB = 56 / 2

<ACB = 28°

Jumlah sudut segitiga : 180°

maka

<PBC + <ACB + <BDC = 180°

68° + 28° + x = 180°

x = 180 - 28 - 68

x = 84° ( sudut BDC)

<APB + <PDA + <PAD = 180°

56° + 84° + x = 180°

x = 180 - 84 - 56

x = 40° (<PAC = <PAD)
