Misalkan x = banyak uang krtas 2an rupiah dan y = banyak uang krtas 5rb-an rupiah. Diperoleh sistem persamaan linear:
x + y = 40 => 2x + 2y = 80
2000x + 5000y = 115.000 => 2x + 5y = 115
---------------------------- -
2y - 5y = 80 - 115
-3y = -35
y = 35/3.
x = 85/3.
Nilai ini tidak mungkin dlm bentuk pecahan. Jadi, soal ini tidak Valid
