lim x→∞ (1 - cos 4/x)/(1/x tan 3/x)

= lim y→0 (1 - cos 4y)/(y tan 3y)

= lim y→0 2 sin² 2y / y tan 3y

= lim y→0 2(sin 2y /y)(sin 2y/tan 3y)

= 2 × (2y/y) × (2y/3y)

= 8/3

22. tolong dibantu ya aaa​


NO 1 :

Pengertian Hidrokarbon :

Hidrokarbon adalah sebuah senyawa yang terdiri dari unsur atom karbon (C) dan atom hidrogen (H).

NO 2 :

Pendahuluan golongan Gas hidrokarbon pada opsi A dan B. :

Golongan Hidrokarbon pada Opsi A = Alkana, karena hanya terdapat 1 ikatan tunggal pada rantai Hidrokarbon.

Golongan Hidrokarbon pada Opsi B = Alkena, karena terdapat ikatan rangkap 2 pada rantai hidrokarbon.

Jawaban :

A. Alkana :

Rumus molekul :

C6H14

Rumus struktur :

CH3–CH2–CH2–CH2–CH2–CH3

B. Alkena :

Rumus molekul :

C5H10

Rumus struktur :

CH3–CH=CH–CH2–CH3

NO 3 :

Pendahuluan :

1 Pengertian atom C primer :

Atom C primer adalah atom C yang hanya mengikat satu atom C lainnya.

2. Pengertian atom C sekunder :

Atom C sekunder adalah atom C yang mengikat 2 atom C lainnya.

3. Pengertian atom C tersier :

Atom C tersier adalah atom C yang mengikat 3 atom C lainnya.

4. Pengertian atom C kuartener :

Atom C kuartener adalah atom C yang mengikat 4 atom C lainnya.

Jawaban :

misalkan :

CH3 diujung kanan = A.

CH3 diujung kiri = E.

CH di kiri = D.

CH di kanan = C

CH2 di kanan = B.

maka :

Jumlah atom C primer = 4, yakni berada di ujung kanan dan kiri (A, E, diatas C, dan dibawah D)

Jumlah atom C sekunder = 1, yakni di sebelah ujung atom C kanan. (B).

Jumlah atom C tersier = 2, yakni C dan D

Jumlah atom C kuartener = 0

30. Quiss¾ + ⅔ = ....?¼ + ¾ - ⅔ = ....?_______________Aaa kecoa gemoy (≧∇≦)/​


PENYELESAIAN :

= ¾ + ⅔

= 9 + 8/12

= 17/12

= 1 5/12

= ¼ + ¾ - ⅔

= 3 + 9 - 8/12

= 4/12

= 1/3

[tex] = \frac{3 \times 3}{4 \times 3} + \frac{2 \times 4}{3 \times 4} [/tex]

[tex] = \frac{9}{12} + \frac{8}{12} [/tex]

[tex] = \frac{9 + 8}{12} [/tex]

[tex] = \frac{17}{12} [/tex]

[tex] = 1 \frac{5}{12} [/tex]

____________

[tex] = \frac{1}{4} + \frac{3}{4} - \frac{2}{3} [/tex]

[tex] = \frac{1 \times 3}{4} - \frac{2}{3} [/tex]

[tex] = \frac{4}{4} - \frac{2}{3} [/tex]

[tex] = \frac{4 \times 3}{4 \times 3} - \frac{2 \times 4}{3 \times 4} [/tex]

[tex] = \frac{12}{12} - \frac{8}{12} [/tex]
