4. arti paribasa↓ngawur kasintu nyieuhkeun hayam


Ngawur kasintu nyieuhkeun hayam nyaeta hiji paribasa Sunda anu hartina ari ka deungeun ngaraeh jeung darehdeh sabab hayang kapuji, tapi ka dulur atawa ka baraya sorangan mah teu nolih jeung nyapirakeun. Paribasa teh nyaeta salahsahiji rupa tina pakeman basa.

Pembahasan

Paribasa ngawur kasintu nyieuhkeun hayam teh sok loba benerna, sabab kadang-kadang urang sok poho ari ka ka baraya atawa ka dulur sorangan teh, kukituna eta paribasa teh bisa dijadikeun panginget pikeun ka urang sarerea. Saluyu jeung fungsi tina paribasa, nyaeta pikeun piwuruk atawa pepeling.

17. Jawab beserta penyelesaiannya! Jangan ngawur (yang ngawur gk bisa baca berarti)!


Jawaban:

B. 2

Penjelasan:

Equation given :

[tex]x - \frac{5}{\sqrt{x} } = 27[/tex]

Assume that [tex]\sqrt{x}[/tex] = p so that :

[tex]p^{2} - \frac{10}{p}=27 \\[/tex]

p³ - 27p - 10 = 0

(p + 5)(p² - 5p - 2) = 0

p = -5 or p² - 5p = 2

Due that p > 0, p = -5 is an impossible solution. So p² - 5p = 2 is the solution of the equation with p as mentioned above = √x.

So x - 5√x = 2 ( B )
