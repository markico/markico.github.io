16. Kirun, Lusi, Mamik dan Neni adalah lima bersaudara. Kirun merupakan yang paling muda danNeni adalah yang paling tua. Sedangkan Mamik lebih tua dari Lusi. Umur mereka membentukbarisan aritmatika yaitu k, 8, m, n. Apabila umur Mamik tidak diikutkan maka umur mereka akanmembentuk barisan geometri dengan rasio positif. Tentukan jumlah umur mereka jika tidak adadiantara mereka yang merupakan saudara kembar!​


Misal, terdapat barisan u₁, u₂, u₃, u₄, ....

Barisan tersebut dikatakan barisan aritmatika jika suku-suku pada barisan memiliki beda (b)/selisih yang sama.

b = u₂ - u₁ = u₃ - u₂ = u₄ - u₃

Barisan tersebut dikatakan barisan geometri jika suku-suku pada barisan memiliki rasio (r) yang sama.

r = u₂/u₁ = u₃/u₂ = u₄/u₃

======================

Diketahui:

Umur Kirun, Lusi, Mamik, Neni membentuk barisan aritmatika

k, 8, m, n

maka

u₃ - u₂ = u₂ - u₁

m - 8 = 8 - k

k + m = 16

m = 16 - k

u₄ - u₃ = u₃ - u₂

n - m = m - 8

n - 2m = -8

Substitusi m = 16 - k

n - 2(16 - k) = -8

n - 32 + 2k = -8

2k + n = -8 + 32

2k + n = 24

n = 24 - 2k

Jika umur Mamik tidak diikutkan, umur mereka membentuk barisan geometri

k, 8, n

u₃/u₂ = u₂/u₁

n/8 = 8/k

kn = 64

Substitusi n = 24 - 2k

k(24 - 2k) = 64

24k - 2k² = 64

2k² - 24k + 64 = 0

(2k - 8)(k - 8) = 0

2k - 8 = 0 → 2k = 8

k = 4

k - 8 = 0 → k = 8

Karena tidak ada yang saudara kembar, maka k = 8 tidak memenuhi, sehingga nilai k = 4.

Substitusi k = 4 ke m = 16 - k dan n = 24 - 2k

m = 16 - k

m = 16 - 4

m = 12

n = 24 - 2k

n = 24 - 2(4)

n = 24 - 8

n = 16

Umur Kirun: k = 4 tahun

Umur Lusi: l = 8 tahun

Umur Mamik: m = 12 tahun

Umur Neni: n = 16 tahun

Umur Kirun, Lusi, Mamik, Neni membentuk barisan aritmatika dengan beda 4

4, 8, 12, 16

Jika umur Mamik tidak diikutkan, umur mereka membentuk barisan geometri dengan rasio 2

4, 8, 16

Jumlah Umur:

k + l + m + n

= 4 + 8 + 12 + 16

= 40

Semoga membantu.

Bentuk Operasi

= (3 4/5 + 1 2/5) ÷ 13/20

= (19/5 + 7/5) ÷ 13/20

= 26/5 ÷ 13/20

= 26/5 ÷ 20/13

= 520/65

= 8

Jadi, banyak botol yang diperlukan ibu Mamik adalah 8 botol

26. Kirun, Lusi, Mamik dan Neni adalah lima bersaudara. Kirun merupakan yang paling muda dan Neni adalah yang paling tua. Sedangkan Mamik lebih tua dari Lusi. Umur mereka membentuk barisan aritmatika yaitu , 8, , . Apabila umur Mamik tidak diikutkan maka umur mereka akan membentuk barisan geometri dengan rasio positif. Tentukan jumlah umur mereka jika tidak ada diantara mereka yang merupakan saudara kembar!


Jawab:

Misal, terdapat barisan u₁, u₂, u₃, u₄, ....

Barisan tersebut dikatakan barisan aritmatika jika suku-suku pada barisan memiliki beda (b)/selisih yang sama.

b = u₂ - u₁ = u₃ - u₂ = u₄ - u₃

Barisan tersebut dikatakan barisan geometri jika suku-suku pada barisan memiliki rasio (r) yang sama.

r = u₂/u₁ = u₃/u₂ = u₄/u₃

======================

Diketahui:

Umur Kirun, Lusi, Mamik, Neni membentuk barisan aritmatika

k, 8, m, n

maka

u₃ - u₂ = u₂ - u₁

m - 8 = 8 - k

k + m = 16

m = 16 - k

u₄ - u₃ = u₃ - u₂

n - m = m - 8

n - 2m = -8

Substitusi m = 16 - k

n - 2(16 - k) = -8

n - 32 + 2k = -8

2k + n = -8 + 32

2k + n = 24

n = 24 - 2k

Jika umur Mamik tidak diikutkan, umur mereka membentuk barisan geometri

k, 8, n

u₃/u₂ = u₂/u₁

n/8 = 8/k

kn = 64

Substitusi n = 24 - 2k

k(24 - 2k) = 64

24k - 2k² = 64

2k² - 24k + 64 = 0

(2k - 8)(k - 8) = 0

2k - 8 = 0 → 2k = 8

k = 4

k - 8 = 0 → k = 8

Karena tidak ada yang saudara kembar, maka k = 8 tidak memenuhi, sehingga nilai k = 4.

Substitusi k = 4 ke m = 16 - k dan n = 24 - 2k

m = 16 - k

m = 16 - 4

m = 12

n = 24 - 2k

n = 24 - 2(4)

n = 24 - 8

n = 16

Umur Kirun: k = 4 tahun

Umur Lusi: l = 8 tahun

Umur Mamik: m = 12 tahun

Umur Neni: n = 16 tahun

Umur Kirun, Lusi, Mamik, Neni membentuk barisan aritmatika dengan beda 4

4, 8, 12, 16

Jika umur Mamik tidak diikutkan, umur mereka membentuk barisan geometri dengan rasio 2

4, 8, 16

Jumlah Umur:

k + l + m + n

= 4 + 8 + 12 + 16

= 40
