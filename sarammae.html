Jawab:1. x + 2y

  p² - pq + 3q² + 2(p² + 2pq - q²)

  = p² - pq + 3q² + 2p² + 4pq - 2q²

  = (p² + 2p² ) + (3q² - 2q²) + (-pq + 4pq)

  = 3p² + q² + 3pq

2. 2x - y

  2(p² - pq + 3q² ) - ( p² + 2pq - q²)

  = 2p² - 2pq + 6q² - p² - 2pq + q²

  = (2p² - p²) + (6q² + q²) + (-2pq - 2pq)

  = p² + 7q² - 4pq

3. 3x - 2y

   3(p² - pq + 3q²) - 2(p² + 2pq - q²)

   = 3p² - 3pq + 9q² - 2p² - 4pq + 2q²

   = (3p² - 2p²) + ( 9q² + 2q²) + (-3pq - 4pq)

   = p² + 11q² - 7pq

4. 2y + x

   2(p² + 2pq - q² ) + p² - pq + 3q²

   = 2p² + 4pq - 2q² + p² - pq + 3q²

   = (2p² + p² ) + ( - 2q² + 3q² ) + (4pq - pq)

   = 3p² + q² + 3pq

5. 2x + 3y

   2(p² - pq + 3q²) + 3(p² + 2pq -  q²)

   = 2p² - 2pq + 6q² + 3p² + 6pq - 3q²

   = (2p² + 3p²) + ( 6q² - 3q²) + (-2pq + 6pq)

   = 5p² + 3q² + 4pq
