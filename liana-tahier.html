29. jumlah nilai ulangan matematika amira berliana dan chelsea 225 selisih nilai ulangan amira dan berliana adalah 14 sedangkan selisih nilai berliana dan chelsea adalah 5 maka nilai ulangan chelsea adalah​


Maka nilai ulangan chelsea adalah​ 122. Dengan menggunakan subtitusi.

Penjelasan dengan langkah-langkah:

Kita dapat menggunakan sistem persamaan linear untuk menyelesaikan masalah ini. Pertama, kita akan gunakan persamaan B - A = 14 untuk menentukan nilai A dalam persamaan A + B + C = 225:

B - A = 14

A = B - 14

Kita substitusikan A = B - 14 ke dalam persamaan A + B + C = 225 dan kemudian gunakan persamaan C - B = 5 untuk menentukan nilai C:

A + B + C = 225

(B - 14) + B + C = 225

2B + C = 239

C = 2B + 5

Substitusikan nilai C = 2B + 5 ke dalam persamaan A + B + C = 225 dan teruskan untuk menyelesaikan masalah ini:

A + B + C = 225

(B - 14) + B + (2B + 5) = 225

4B - 9 = 225

4B = 234

B = 58,5

Jadi, nilai ulangan Chelsea adalah:

C = 2B + 5 = 2(58,5) + 5 = 122
