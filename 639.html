Penjelasan dengan langkah-langkah:

n = 3

S3 = 639

b = 2

–

Tentukan suku pertama (a)

[tex]S_{n} = \frac{n}{2} (2a + (n - 1)b)[/tex]

[tex]S_{3} = \frac{3}{2} (2a + (3 - 1)2)[/tex]

[tex]639 = \frac{3}{2} (2a + 4)[/tex]

[tex]639 = 3a + 6[/tex]

[tex]3a = 639 - 6[/tex]

[tex]3a = 633[/tex]

[tex] a = \frac{633}{3} [/tex]

[tex] \boxed{a = 211}[/tex]

–

Tentukan 3 bilangan ganjil tersebut (U1, U1, U3)

Un = a + (n - 1) b

U1 = 211 + (1 - 1) 2 = 211U2 = 211 + (2 - 1) 2 = 213U3 = 211 + (3 - 1) 2 = 215
