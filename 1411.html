16. QUIZ (1411)Materi: Operasi HitungTentukan hasil dari [tex] xyz^2 [/tex] jika diketahui :[tex] x = 1 = \dfrac{ y-1}{ 3 } =\dfrac{ z-2y }{ 3 } [/tex]​


Jika diberikan persamaan [tex] \rm x = 1 = \frac{ y-1}{ 3 } =\frac{ z-2y }{ 3 } [/tex]. Maka hasil dari [tex]\rm = xyz^2 [/tex] adalah 484.

Pendahuluan

Sistem Persamaan Linier Tiga Variabel (SPLTV) adalah suatu persamaan yang memiliki tiga variabel. Bentuk umumnya, yaitu :

[tex] \rm ax+by+cz = p[/tex]

[tex] \rm dx+ey+fz = q[/tex]

[tex] \rm gx+hy+iz = r[/tex]

SPLTV dapat diselesaikan dengan beberapa metode, yaitu :

●Metode Eliminasi

●Metode Subtitusi

●Metode Campuran (Eliminasi & Subtitusi)

●Metode Determinan Matriks

●Metode Invers Matriks

●Metode Gauss Jordan

Pembahasan

Diketahui :

[tex] \rm x = 1 = \frac{ y-1}{ 3 } =\frac{ z-2y }{ 3 } [/tex]

Ditanya :

Hasil dari [tex] xyz^2 [/tex] ?

Jawab :

Dari persamaan yang diketahui didapat bahwa nilai x = 1.

[tex] \rm \boxed{x = 1} = \frac{ y-1}{ 3 } =\frac{ z-2y }{ 3 } [/tex]

Cari nilai y dari [tex] \rm x = \boxed{1 = \frac{ y-1}{ 3 }} =\frac{ z-2y }{ 3 } [/tex]

[tex] \rm 1 = \frac{y-1}{3}[/tex]

[tex] \rm 3 = y-1[/tex]

[tex] \rm y = 3+1[/tex]

[tex] \rm y = 4[/tex]

Cari nilai z dari [tex] \rm x = 1 =\boxed{ \frac{ y-1}{ 3 } =\frac{ z-2y }{ 3 } }[/tex]

[tex] \rm \frac{ y-1}{ 3 } =\frac{ z-2y }{ 3 } [/tex]

[tex] \rm \frac{ 4-1}{ 3 } =\frac{ z-2(4) }{ 3 } [/tex]

[tex] \rm \frac{ 3}{ 3 } =\frac{ z-8}{ 3 }[/tex]

[tex] \rm 1 = \frac{z-8}{3}[/tex]

[tex] \rm 3 = z-8[/tex]

[tex] \rm z = 3+8[/tex]

[tex] \rm z = 11[/tex]

Hitung nilai yang ditanyakan :

[tex]\rm = xyz^2 [/tex]

[tex]\rm = (1)(4)(11)^2 [/tex]

[tex]\rm = (1)(4)(121) [/tex]

[tex]\bf = 484 [/tex]

Kesimpulan

Jadi, hasilnya adalah 484.

Interval tangga nada diatonis minor (minor asli) adalah:

1 - 1/2 - 1 - 1 - 1/2 - 1 - 1

Pernyataan "Sekilas bila dilihat nada-nada yang  ada pada tangga nada ini adalah sama  dengan nada-nada pada tangga nada diatonis Mayor" karena dalam scaling nada dikenal istilah minor relatif dan mayor relatif. Jika tangga nada X minor memiliki tanda mula yang sama dengan tangga nada Y Mayor, maka X minor merupakan "minor relatif" dari Y Mayor, dan sebaliknya, Y Mayor adalah "mayor relatif" dari X minor.

Dengan menggunakan notasi angka, agak sulit menentukan minor relatif dan mayor relatif, karena notasi angka bersifat movable-do, artinya posisi nada Do bisa ada di pitch apapun. Jika Do = C, nada Do adalah C. Jika Do = D, nada Do adalah D. Lagu yang sama namun berbeda nada dasarnya, dan ditulis dalam notasi angka, akan memiliki notasi angka yang sama persis.

Sebagai contoh di bawah ini menggunakan notasi huruf/abjad.

C Mayor: C-D-E-F-G-A-B-C

                1-2-3-4-5-6-7-1

A minor: A-B-C-D-E-F-G-A

              6-7-1-2-3-4-5-6

A minor adalah minor relatif dari C Mayor. C Mayor adalah mayor relatif dari A minor.

D Mayor (2#): D-E-F#-G-A-B-C#-D

                       1-2-3-4-5-6-7-1

B minor (2#): B-C#-D-E-F#-G-A-B

                      6-7-1-2-3-4-5-6

B minor adalah minor relatif dari D Mayor. D Mayor adalah mayor relatif dari B minor.
