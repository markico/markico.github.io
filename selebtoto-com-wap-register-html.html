1. Cara Membuat Form Login Dan Register Dengan HTML, PHP


Untuk HTML, silahkan buat tag <form>.
Di dalam tag <form> tersebut, nanti ada tag lain yang berfungsi mendapat data dari user, yaitu <input>. <input> ini punya attribute type="" yang bisa menentukan apa yang diisi oleh user. Misal, <input type="password"> akan membuat apa yanh diketik user berubah jadi bulatan² seperti password pada umumnya. <input type="date"> akan mengisi tanggal. Jika ingin lebih tahu lagi, ada banyak attribute <input> yang lainnya silahkan lihat W3Schools.

Untuk PHPnya, nanti kamu harus mengintegrasikan HTML tag <form> dengan PHPmu. Yaitu dengan attribute action="namafilephp.php". File php ini berisi fungsi yang bisa membuat inputan user disimpan ke database, yaitu MySQL. Silahkan cari sumber lain lagi untuk lebih lengkapnya, karena kode PHP tiap orang beda².

5. Kepanjangan WAP dalam teknologi internet dan komputer


WAP merupakan singkatan dari Wireless Application Protocol. Pengertian WAP adalah suatu protokol aplikasi yang memungkinkan internet dapat diakses oleh ponsel dan perangkat wireless lainnya. WAP membawa informasi secara online melewati internet langsung menuju ke ponsel atau klien WAP lainnya. Dengan adanya WAP, berbagai informasi dapat kita akses setiap saat hanya dengan menggunakan ponsel. Cara kerja WAP hampir sama dengan cara kerja internet saat ini. Dibutuhkan WAP gateway untuk menjembatani ponsel dengan internet dalam mengirim dan menerima data. Hal ini sama halnya dengan pengguna komputer yang membutuhkan ISP sebagai gateway dalam menjembatani komputer dengan internet.

15. domain pada internet-CO-ID-COM-EDU-ORG-HTML-AU-ONIONJawab Plis​


Domain pada internet

".co" domain untuk beberapa negara. ( Jika di Indonesia menggunakan .co.id, namun negara kolombia menggunakan langsung .co )

".id" domain untuk negara Indonesia

".com" domain internasional

".edu" domain untuk pendidikan

".org" domain untuk organisasi

".html" domain untuk tipe ekstensi yang berbentuk html, domain ini juga bisa berupa file dan bisa dijalankan offline

".au" domain untuk negara australia

".onion" domain untuk pengguna aplikasi khusus ( Dalam hal ini butuh aplikasi yang bernama TOR ) terkadang disebut domain untuk dark web

Pembahasan

Yang kita liat diatas ialah top level domain, domain diatas berarti domain yang tidak bisa ditambahkan lagi. Sebagai contoh domain pada web brainly.co.id ialah ".co.id", ".co" adalah second level domain sedangkan ".id" adalah top level domain.

Diatas juga merupakan domain yang bisa digunakan sebagai website dan gmail ( Kecuali .html dan .onion ). Fungsi domain cukup penting sebagai identitas suatu organisasi ataupun website.

http://www.sekolahku.com/informasi/pendaftaran.html

1. Protokol yang digunakan adalah HTTP (Hypertext Transfer Protocol)
2. Top Level Domain = .com
3. Second Level Domain = sekolahku
3. Path domain = informasi
4. Nama file = pendaftaran.html
