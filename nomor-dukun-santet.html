27. tolong banh ☝️ hari ini di antar, jawab yang bener ngasal gua santet lu aseli bapak ku dukun ​


Jawab:
[1.] Untuk teman ke-10 = 30 perangkat
[2.] Jumlah semua perangkat = 165 perangkat

Penjelasan dengan langkah-langkah:
[1.] Diketahui deret {3, 6, 9, ...}
suku pertamanya 3 maka
--> a = 3, nilai beda (b) = suku kedua-a
maka nilai b = 6-3 = 3
--> b = 3
Untuk teman ke-10,
--> n = 10
Dengan rumus suku ke-n
Un = a+b(n-1)
U₁₀ = 3+3(10-1)
U₁₀ = 3+3(10)-3(1)
U₁₀ = 3(10)
U₁₀ = 30
Untuk teman ke-10 = 30 perangkat

[2.] Ditanya berapa jumlah perangkat
total dari 10 teman-teman Berta..
Dengan rumus Sn
Sn = ⁿ/₂ (a+Un)
S₁₀ = ¹⁰/₂ (3 + U₁₀)
maka S₁₀ = ¹⁰/₂ (3 + 30)
S₁₀ = 5 (3 + 30)
S₁₀ = 5(3) + 5(30)
S₁₀ = 15 + 150
S₁₀ = 165
Jumlah semua perangkat = 165 perangkat

(xcvi)
