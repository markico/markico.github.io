5. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722

25. pola bilangan persegi panjang jika Pn = 552 suku keberapa Pn ?


Matematika
Pola Bilangan VIII SMP

Pembahasan :
Pola Persegi Panjang

Pn = 552
n(n + 1) = 552
n² + n = 552
n² + n - 552 = 0
(n - 23)(n + 24) = 0
n = 23 —» Pada suku ke-23

26. 12. Jarak Malang - Kepanjen dalam peta yang berskala 1 : 500.000 adalah 5,5 cm. Sehinggajarak sebenarnya Malang - Kepanjen adalah ....A. 2,75 KmC. 275 KmD. 2750 KmB. 27,5 Kmdam adalah​


Jawaban:

B. 27,5 km

Penjelasan dengan langkah-langkah:

JP= S×JS

5,5 cm= 1/500.000× JS

5,5/100.000 km= 1/500.000km× JS

55/1.000.000 × 500.000/1 = JS

55/10×5=JS

11/2×5=JS

55/2=JS

27,5 km = JS

B

Jarak sebenarnya adalah 27,5 km (B)

.

Pembahasan

Skala adalah perbandingan antara jarak pada peta dengan jarak sebenarnya.

.

Rumus-rumusSkala

[tex]s = \frac{jp}{js} [/tex]

.

Jarak pada peta

[tex]jp = js \times s[/tex]

.

Jarak sebenarnya

[tex]js = \frac{jp}{s} [/tex]

.

Keterangan:

s=Skala

jp=Jarak pada peta

js=Jarak sebenarnya

.

Skala adalah perbandingan dari jarak pada peta dengan jarak sebenarnya.

.

PenyelesaianDiketahui

Jarak pada peta=5,5 cm

Skala=1 : 500.000

.

Ditanya

Jarak sebenarnya adalah...

.

Jawab

[tex]js = \frac{jp}{s}[/tex]

[tex]js = \frac{5,5}{\frac{1}{500.000} }[/tex]

[tex]js = 5,5 \div \frac{1}{500.000} [/tex]

[tex]js = \frac{55}{10} \times 500.000[/tex]

[tex]js = 55 \times 50.000[/tex]

[tex]js = 2.750.000 \: cm[/tex]

[tex]js = 27,5 \: km[/tex]

.

Jadi, jarak sebenarnya adalah 27,5 km (B)

.
