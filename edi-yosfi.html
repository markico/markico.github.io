25. Edi menabung sebesarRP4.000.000.00 dibank dikasi suku bunga tunggal sebesar8% per thn jika tabung edi sekarang menjadi RP4.400.000.00 maka edi telah menabung selama


M = 4.000.000
B = 8%
M akhir = 4.400.000

Ditanya lama menabung (n)

Jawaban :

➡Besar Suku bunga
= Tabungan akhir - tabungan awal
= 4.400.000 - 4.000.000
= 400.000

Maka, Lama menabung :

400.000 = n/12 x B x M

400.000 x 12 = n x 8/100 x 4.000.000

4.800.000 = n x 32.000.000/100

4.800.000 = n x 320.000

4.800.000/320.000 = n

15 = n

Jadi, lama menabung adalah 15 bulan
