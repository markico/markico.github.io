3A = 900.000

4B = 900.000

4C = A + B + D

B = 4C - A - D

5C = 900.000

C = 900.000/5

C = 180.000

sehingga,

A + B + C + D = 900.000

300.000 + 225.000 + 180.000 + D = 900.000

705.000 + D = 900.000

20. Empat orang siswa akan mengikuti suatu perlombaan karya inovatif.untuk itu diperlukan biaya Rp 900.000.00. karna masing-masing memiliki kondisi keuangan yang berbeda, besar kontribusi masing-masing siswa tidak sama.siswa A memberikan kontribusi setengah dari jumlah kontribusi dari tiga siswa lainnya. siswa B memberikan kontribusi sepertiga dari jumlah kontribusi tiga. Siswa lainnya. sisetC memberikan kontribusi seperempat dari jumlah kontribusi tiga siswa lainnya. besar kontribusi siswa D adalah Rp...​


Buat model matematis dari permasalahan tersebut agar lebih mudah.

Diketahui :

A + B + C + D = 900.000

A = 1/2(B + C + D)

B = 1/3(A + C + D)

C = 1/4(A + B + D)

Ditanyakan : D ?

Jawab :

Pertama, ubah ke bentuk yang nyaman dilihat.

A + B + C + D = 900.000...(1)

2A = B + C + D...(2)

3B = A + C + D...(3)

4C = A + B + D...(4)

Kita mulai dari persamaan 4 dahulu.

4C = A + B + D

D = 4C - A - B...(5)

Selanjutnya, substitusi (5) ke persamaan 1 s/d 3, diperoleh :

A + B + C + (4C - A - B) = 900.000...(1)

5C = 900.000

C = Rp 180.000

2A = B + C + (4C - A - B)...(2)

3A = 5C = 900.000

A = Rp 300.000

3B = A + C + (4C - A - B)...(3)

4B = 5C = 900.000

B = Rp 225.000

Karena A, B, dan C telah ditemukan, tinggal substitusikan saja ke persamaan 5, maka :

D = 4(180.000) - 300.000 - 225.000

D = 720.000 - 300.000 - 225.000

D = Rp 195.000

Jadi, besar kontribusi siswa D adalah Rp 195.000.

Semoga membantu.
