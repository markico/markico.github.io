Pada sebuah peta berskala 1:10.000.000, jarak sebenarnya antara kota A dan kota B adalah 500km. Berapakah jarak antara kota A dan kota B pada peta?

Dik:
Jarak sebenarnya = 500km = 50.000.000cm
Skala = 1:10.000.000

Jawab:
Jarak pada peta = Skala x Jarak sebenarnya
= ( 1 / 10.000.000 ) x 50.000.000cm
= 5cm

Jadi, jarak pada peta antara kota A dan kota B adalah 5cm.

Skala = 1 : 10.000.000

Jarak pada peta

= JS x Skala

= 80 Km x [1 : 10.000.000]

= 80(100.000 cm) x [1 : 10.000.000]

= 80 x [1 : 100]

= 8 / 10

= 0,8 cm ✔
