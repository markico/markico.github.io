11. tentukan jarak antara titik B (5,-12)dengan titik pangkaltentukan jarak titik (8,12) dan (1,-12)​


Jawaban:

13 satuan

dan

25 satuan

Penjelasan dengan langkah-langkah:

Jarak Titik B (5, -12) dan O (0,0) :

[tex] |OB| = \sqrt{ {( x_{B} - x_{O})}^{2} + {( y_{B} - y_{O})}^{2}} [/tex]

[tex] |OB| = \sqrt{ {(5 - 0)}^{2} + {( ( - 12) - 0)}^{2}} [/tex]

[tex] |OB| = \sqrt{ {(5 )}^{2} + {( - 12)}^{2}} [/tex]

[tex] |OB| = \sqrt{ {25 + 144}}[/tex]

[tex] |OB| = \sqrt{ {169}}[/tex]

[tex] |OB| = 13[/tex]

Jadi, jarak titik B ke titik pangkal adalah 13 satuan

○○○○○○○

Jarak Titik (8, 12) dan (1, -12) :

[tex] \text{jarak} = \sqrt{ {( x_{2} - x_{1})}^{2} + {( y_{2} - y_{1})}^{2}} [/tex]

[tex] \text{jarak} = \sqrt{ {( 1 - 8)}^{2} + {( ( - 12) - 12)}^{2}} [/tex]

[tex] \text{jarak} = \sqrt{ {( - 7)}^{2} + { ( - 24)}^{2}} [/tex]

[tex] \text{jarak} = \sqrt{ 49 + 576} [/tex]

[tex] \text{jarak} = \sqrt{625} [/tex]

[tex] \text{jarak} = 25[/tex]

Jadi, jarak titik (8, 12) dan (1, -12) adalah 25 satuan

13. Persamaan garis yang melalui titikpangkal dan titik P(24, 12) adalah​


Jawaban:

[tex]x - 2y = 0[/tex]

Penjelasan:

Titik pangkal : (0, 0)

Persamaan garis melalui titik (0, 0) dan (24, 12)

[tex] \frac{y - y1}{y2 - y1} = \frac{x - x1}{x2 - x1} [/tex]

[tex] \frac{y - 0}{12 - 0} = \frac{x - 0}{24 - 0} [/tex]

[tex] \frac{y}{12} = \frac{x}{24} [/tex]

[tex]24y = 12x[/tex]

[tex]12x - 24y = 0[/tex]

[tex]x - 2y = 0[/tex]

16. gradien garis yang melalui pangkal 0(0,0) dan titik s(-6,-12) adalah​


Jawaban:

2

Penjelasan dengan langkah-langkah:

Diketahui :

Garis melalui titik O(0,0) dan titik S(-6,-12)

Ditanya ;

Gradien garis (m) = ?

Jawab :

O(0,0) = (x1,y1)

x1 = 0

y1 = 0

S(-6,-12) = (x2,y2)

x2 = -6

y2 = -12

Gradien garis

[tex]m = \frac{∆y}{∆x}[/tex]

[tex]m = \frac{y2-y1}{x2-x1}[/tex]

[tex]m = \frac{-12-0}{-6-0}[/tex]

[tex]m = \frac{-12}{-6}[/tex]

[tex]\bold{m = 2}[/tex]

Jadi, gradien garis yang melalui pangkal O(0,0) dan titik S(-6,-12) adalah 2

Rotasi {P(0,0), γ = 270} maka x= -y'  dan  y = x'

-4y' - 3x' - 12= 0  atau  3x + 4y + 12 = 0
