paket Regular = x

paket Cargo = y

( I ) y - x = 50

( II ) y + x = 70

x - y = 50

x + y = 70

0 - 2y = -20

- 2y = -20

y = -20/-2

y = 10

Jadi, jumlah paket Cargo yang dikirim dari Kota A ke Kota B adalah 10 paket.

30. 4. Sebuah kapal pengangkut berat 16000 ton diberikan muatan cargo 1800 ton. CG cargo berada di 7.2 meter bagian depan dari posisi CG kapal. Tentukan sejauhmana CG kapal bergerak setelah diberikan muatan!​


Jawaban:

Untuk menyelesaikan masalah ini, kita perlu menggunakan prinsip kekekalan momen (momentum) dalam mekanika. Prinsip ini menyatakan bahwa momen total sebelum dan sesudah suatu sistem mengalami perubahan harus tetap sama.

Dalam hal ini, kita dapat menggunakan rumus:

Momen awal = momen akhir

Momentum kapal awal + momentum muatan awal = momentum kapal akhir + momentum muatan akhir

Karena kapal awalnya tidak memiliki momentum (dalam keadaan diam), maka rumus ini dapat disederhanakan menjadi:

Momentum muatan awal = Momentum kapal akhir + Momentum muatan akhir

Selanjutnya, kita perlu menentukan posisi CG (center of gravity) dari kapal setelah diberi muatan. Misalkan x adalah jarak CG kapal dari bagian depan kapal setelah diberi muatan.

Momentum muatan awal dapat dihitung sebagai berikut:

Momentum muatan awal = m x d

dalam mana m adalah massa muatan (dalam ton) dan d adalah jarak CG muatan dari bagian depan kapal (dalam meter).

Dalam hal ini, d tidak diberikan, tetapi kita dapat menggunakan fakta bahwa CG muatan berada di 7.2 meter bagian depan dari posisi CG kapal untuk menentukannya. Dengan demikian, jarak CG muatan dari bagian depan kapal adalah:

x + 7.2 meter

Momentum muatan awal = (1800 ton) x (x + 7.2 meter)

Momentum kapal akhir dan momentum muatan akhir juga dapat dihitung dengan rumus yang sama, tetapi kali ini kita perlu mempertimbangkan kecepatan kapal. Misalkan v adalah kecepatan kapal setelah diberi muatan (dalam meter/detik).

Momentum kapal akhir = (m + M) x v

dalam mana M adalah massa kapal (dalam ton). Kita diberikan bahwa M = 16000 ton.

Momentum muatan akhir = m x (v/2)

Karena muatan bergerak bersama dengan kapal setelah diberikan, maka kecepatannya setengah dari kecepatan kapal.

Kembali ke rumus momentum awal = momentum akhir, kita dapat menulis:

Momentum muatan awal = Momentum kapal akhir + Momentum muatan akhir

(1800 ton) x (x + 7.2 meter) = (16000 + 1800) ton x v + (1800 ton) x (v/2)

Dengan menyederhanakan, kita dapat menghilangkan satuan ton dan menyamakan kecepatan dengan satuan meter/detik:

1800 (x + 7.2) = 17800v

v = (1800/17800) (x + 7.2)

v = 0.101 (x + 7.2)
