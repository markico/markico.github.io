29. tentukan keempat bilangan kuantum pada elektron terakhir periode dan golongan untuk nomor atom 97​


Elektron terakhir atom tersebut memiliki bilangan kuantum n = 5, ℓ = 3, m = -1, s = -¹/₂. Unsur tersebut terletak pada periode 7. Golongan yang ditempati adalah golongan transisi dalam (aktinida).

Pembahasan

Bilangan kuantum elektron terakhir dapat ditentukan jika diketahui posisi elektron terakhir pada sebuah atom. Berikut beberapa jenis bilangan kuantum:

1) Bilangan kuantum utama ( n )

Bilangan kuantum ini menyatakan letak elektron dalam kulit atom atau bilangan yang mewakili nomor kulit,  

jenisnya : K ( n = 1 ), L ( n = 2 ), M ( n = 3 ), N ( n = 4 ), dst.

2) Bilangan kuantum azimuth ( ℓ ) → (n-1)

Bilangan kuantum ini menyatakan letak elektron dalam sub kulit.atau bilangan yang mewakili subkulit atom yang ditempati elektron. Aturan menentukan banyaknya subkulit adalah n – 1 (dengan n adalah nomor kulit),  

Jenis subkulit :  

s = sharp (ℓ = 0);    

p = principal (ℓ=1);  

d = diffuse (ℓ = 2);  

f = fundamental (ℓ = 3)

3) Bilangan kuantum magnetik (m)

Bilangan kuantum ini menyawakan letak elektron dalam orbital atau bilangan yang mewakili orbital yang ditempati elektron. Aturan banyaknya orbital adalah mulai dari – ℓ  sampai dengan +ℓ. Berikut jenis orbital sesuai subkulitnya:

jika ℓ = 0, maka orbitalnya = 0

jika ℓ = 1, maka orbitalnya mulai dari -1 s/d +1 = -1, 0, +1

jika ℓ = 2, maka orbitalnya mulai dari -2 s/d +2 = -2, -1, 0, +1, +2

jika ℓ = 3, maka orbitalnya mulai dari -3 s/d +3 = -3, -2, -1, 0, +1, +2, +3

4) Bilangan kuantum spin (s)

Bilangan kuantum yang menyatakan arah elektron putaran elektron dalam orbital. Jenisnya:  

+ ½  untuk tanda panah ke atas (↑) => putaran searah jarum jam

- ½  untuk tanda panah ke bawah (↓)=> putaran berlawanan arah jarum jam

Pelajari lebih lanjut tentang bilangan kuantum di: https://brainly.co.id/tugas/20216480.  

Pada kasus di atas

Pada soal diketahui atom dengan nomor 97. Kalian dapat menentukan bilangan kuantum, golongan dan periode jika diketahui konfigurasi elektronnya. Konfigurasi elektron dapat ditentukan menggunakan jumlah elektron pada kulit atom. Pada kasus tersebut jumlah elektron sama dengan nomor atomnya yaitu 97. Konfigurasi elektonnya dapat dituliskan sebagai berikut:

1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶ 5s² 4d¹⁰ 5p⁶ 6s² 4f¹⁴ 5d¹⁰ 6p⁶ 7s² 5f⁹

konfigurasi singkatnya: [Rn] 7s² 5f⁹

Periode:  

Karena nomor kulit terbesar adalah 7, maka unsur tersebut terletak pada periode 7.

golongan:

Unsur tersebut berada pada blok f, sehingga termasuk ke dalam golongan transisi dalam. Karena letaknya pada periode 7 maka nama golongannya adalah aktinida.

Bilangan kuantum elektron terakhir:

elektron terakhir terletak di subkulit 5f⁹

n = 5 (didapat dari nomor kulit)

ℓ = 3 (didapat dari subkulit f)

m = -2 (didapat orbital yang ditempati elektron terakhir)

[↑↓][↑↓][↑][↑][↑][↑][↑]

-3   -2  -1   0  1  2  3

s = -¹/₂ (didapat dari arah putaran elektron terakhir yaitu ↓)

Jadi dapat disimpulkan bahwa elektron terakhir atom tersebut memiliki bilangan kuantum n = 5, ℓ = 3, m = -1, s = -¹/₂. Adapun letak atom tersebut adalah pada periode 7 golongan transisi dalam (aktinida).
