5. Jumlah berat badan kurniawan dan lukman adalah 132 kg. Jika perbandingan berat badan kurniawan dan lukman sama dengan 5 : 6, maka berat badan kurniawan adalah


5:6 = 5 + 6 = 11

Maka,
5/11 x 132
= 60 kg


Berat badan kurniawan
= 5 / (5 + 6)  {132 kg}
= 5/11 (132 kg)
= 5 (12 kg)
= 60 kg

16% = 12 bulan
8% = 6 bulan
bunga = 8/100x400.000 = 32.000
uang prada = 400.000 + 32.000 = 432.000Diket :
Bunga = 16% per tahun
Uang = 400.000
Waktu = 6 bulan = 1/2 tahun

Ditanya :
Jumlah Uang (J)

Jawab ;
J = 400.000 + (16/100 x 1/2 x 400.000)
J = 400.000 + (16/100 x 200.000)
J = 400.000 + (16 x 2000)
J = 400.000 + 32000
J = 432.000
