ingat bahwa :
aᵇ × aᶜ = aᵇ⁺ᶜ

maka:
w³ × w⁴ = w³⁺⁴
= w⁷

jadi, bentuk paling sederhana dari w³ × w⁴ adalah w⁷sifat perpangkatan
[tex] {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

maka
[tex] {w}^{3} \times {w}^{4} = {w}^{3 + 4} = {w}^{7} [/tex]

18. Tuliskan bentuk w3 x w4 kedalam bentuk perpangkatan paling sederhana. Berapakah hasilnya? Apakah kamu juga dapat menyederhanakan bentuk w3 x n4? Jelaskan!


Bentuk paling sederhana dari  w³ x w⁴ adalah w⁷,

Bentuk paling sederhana dari w³ x n⁴ adalah n(wn)³.

Sifat-sifat perpangkatan

aᵇ x aⁿ = aᵃ⁺ⁿ

aᵇ : aⁿ = aᵇ⁻ⁿ

( aᵇ)ⁿ = a ᵇ ˣ ⁿ

(a/b)ⁿ = aⁿ/ bⁿ

a ⁻ⁿ = 1 / aⁿ

(ab)ⁿ = aⁿ x bⁿ

Pembahasan

w³ x w⁴ berdasarkan sifat aᵇ x aⁿ = aᵃ⁺ⁿ, maka

w³ x w⁴ = w³⁺⁴

            = w⁷

w³ x n⁴ berdasarkan sifat (ab)ⁿ = aⁿ x bⁿ, maka

w³ x n⁴ = w³ x n³ x n

            = n (w³ x n³)

            = n(wn)³
