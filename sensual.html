28. sebuah dadu setimbang dilempar sensual 90 Kali. frekuensi Harapan Muncul mata dadu 4 Sama dengan?


frekunsi harapan = peluang x percobaan
pluang dadu = [tex] \frac{4}{6} [/tex] =[tex] \frac{2}{3} [/tex]
percobaan = 90
FH = [tex] \frac{2}{3} [/tex] x 90
=60
Diketahui :
n = 90 kali
P(A) = n(A)/n(S)
       = 4/6
       = 2/3

Dijawab :
Fh = P(A) x n
     = 2/3 x 90
     = 2 x 30
     = 60

Itu jawabannya.
