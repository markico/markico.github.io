8. Untuk membatalkan penghapusan surat di webmail Yahoo,Anda dapat menggunakan folder


Untuk membatalkan penghapusan surat di webmail Yahoo, Anda dapat menggunakan folder sampah atau trash dimana dalam folder tersebut akan terdapat menu atau pilihan untuk mengembalikan surat yang dihapus ke folder email masuk maupun folder lainnya.

Pembahasan:

Pada dasarnya setiap email Yahoo akan dimasukan kedalam sebuah direktori atau folder yang bernama folder sampah/trash. Biasanya setiap surat sampah yang masuk dalam folder tersebut akan dihapus secara permanen setelah melewati 30 hari dari mulai penghapusan awal, dimana kamu tidak akan bisa lagi mengakses surat tersebut.

Nah, untuk mencegah kesalahan Yahoo memberikan kita opsi untuk membatalkan penghapusan atau lebih tepatnya pilihan untuk memindahkan surat yang sudah terlanjur dihapus ke folder atau direktori lain. Dengan catatan email tersebut belum terhapus secara permanen dari folder sampah tadi.

14. membentuk kalimat contoh kalimat passive voice - dia akan membatalkan makan malam (suara aktif) - makan malam akan dibatalkan olehnya (suara pasif) A. dia membaca buku ituB teman saya menulis surat a minggu lalu C.mereka melakukan ujianD. dia mempercayai mereka ​


Jawaban

1. Dia membaca buku itu.

Active Voice : He is reading the book.

Passive Voice: The book is read by him.

2. Teman saya menulis surat minggu lalu.

Active Voice : My friend wrote a letter last week.

Passive Voice: The letter is written by my friend last week.

3. Mereka melakukan ujian.

Active Voice : They do the test.

Passive Voice: The test is done by them.

4. Dia mempercayai mereka.

Active Voice : He trusted them.

Passive Voice: They are trusted by him.

Pembahasan

Passive voice adalah kalimat yang subjeknya dikenai suatu pekerjaan atau tindakan dari kata kerjanya, artinya bukan subjek tersebut yang melakukan suatu aksi.

Rumus Passive VoiceSimple present tense

Active Voice : S + V1 + O

Passive Voice: S + to be (am, are, is) + V3 + by + O

Present continuous tense

Active Voice : S + to be (am, are, is) + V1-ing + O

Passive Voice: S + to be (am, are, is) + being + V3 + by + O

Present perfect tense

Active Voice : S + to be (am, are, is) + being + V3 + by + O

Passive Voice: S + have/has + been + V3 + by + O

Present perfect continuous tense

Active Voice : S + have/has + been + V1-ing+ O

Passive Voice: S + have/has + been + being + V3 + by + O

Simple past tense

Active Voice : S + V2 + O

Passive Voice: S + to be (was/were) + V3 + by + O

Past continuous tense

Active Voice : S + to be (was, were) + V1-ing + O

Passive Voice: S + to be (was/were) + being + V3 + by + O

Past perfect tense

Active Voice : S + had + V3 + O

Passive Voice: S + had + been + V3 + by + O

Past perfect continuous tense

Active Voice : S + had + been + V1-ing + O

Passive Voice: S + had + been + being + V3 + by + O

Simple future tense

Active Voice : S + will not + V1 + O

Passive Voice: S + will be + V3 + by + O

Future continuous tense

Active Voice : S + will + be + V1-ing + O

Passive Voice: S + will + be + being + V3 + by + O

Future perfect tense

Active Voice : S + will + have + V3 + O

Passive Voice: S + will + have + been + V3 + by + O

Future perfect continuous tense

Active Voice : S + will + have + been + V1-ing + O

Passive Voice: S + will + have + been + V1-ing + O

Simple past future tense

Active Voice : S + would + V1 + O

Passive Voice: S + would + be + V3 + by + O

Past future continuous tense

Active Voice : S + would + be + V1-ing + O

Passive Voice: S + would + be + being + V3 + by + O

Past future perfect tense

Active Voice : S + would + have + V1 + O

Passive Voice: S + would + have + been + V3 + by + O

Past future perfect continuous tense

Active Voice : S + would + have + been + V1-ing + O

Passive Voice: S + would + have + been + being+ V3 + by + O

Modals (present)

Active Voice : S + may/can/must + V1 + O

Passive Voice: S + may/can/must + be + V3 + by + O

Modals (past)

Active Voice : S + might/could/had to + V1 + O

Passive Voice: S + might/could/had to + be + V3 + by + O

Infinitive

Active Voice : S + has to/have to + V1 + O

Passive Voice: S + has to/have to + be + V3 + by + O

Going to

Active Voice : S + to be (am, are, is) + going to + V1 + O

Passive Voice: S + to be (am, are, is) + going to + be + V3 + by + O

25. contoh perikatan syarat tangguh dan batal​


Jawaban:

Perikatan bersyarat dengan syarat tangguh,yaitu perikatan yang akan lahir apabila yang dimaksud itu terjadi dan perikatan lahir pada detik terjadinya peristiwa itu. 28. apa bedanya sah dan batalContoh : Sholat nya tidak sah / Sholat nya batal​


Jawaban:

1.Sah

Sah adalah sesuatu yang di kerjakan atau di lakukan sesuai dengan ketentuan syarat rukunnya, Apabila suatu hal di katakan sah jika memenuhi syarat rukunnya yang ada. Namun apabila syarat bisa di penuhi dan rukunya tidak maka hal itu di anggap tidak sah atau batal.

2.Batal

Batal adalah suatu perkara yang dianggap tidak sah apabila tidak memenuhi syarat rukunnya. Jadi syarat rukunnya sangat penting sekali, misalkan dalam mengerjakan shalat tidak ada syarat dan rukun maka di anggap tidak sah. Oleh karena itu dalam mengerjakan suatu perkara pahami dulu apa saja syarat dan rukunnya agar apa yang di kerjakan tidak sia-sia.
