10. naon artina paimbaran


Kategori Soal : Bahasa Sunda

Kelas : - (SD)

Pembahasan :


Paimbaran hartina nyaeta tempat sholat imam di masjid anu posisina panghareupna. Jamaah anu jadi ma'mum dina sholat barjamaah dina sholat teh nangtungna henteu meunang ngaleuwihan anu jadi imam. Kusabab kitu, biasana ruangan pikeun imam di masjid dibangun ngelok ka luar atawa ka hareup supaya pasti moal aya ma'mum anu ngalewatan batas paimbaran.


Tapi lamun aya sholat barjamaah anu lobana ngan dua urang, anu jadi imam teh henteu sholat dina paimbaran, tapi biasana anu jadi imam sholat sajajar jeung ma'mum. Umpama dina waktu keur sholat aya jalma lianna anu milu sholat, terus imam kudu ngalengkah ka hareup kana paimbaran. Kitu kira-kira aturan anu sholat jeung ayana paimbaran.


Kata Kunci : Paimbaran, sholat

18. Bagaimana perhitungan tahun kasibat


Jawaban:

mohon maaf jika salah

Penjelasan:

Terdapat algoritme mudah untuk menentukan apakah suatu tahun termasuk tahun kabisat atau bukan sebagai berikut:

Jika angka tahun itu habis dibagi 400, maka tahun itu sudah pasti tahun kabisat.

Jika angka tahun itu tidak habis dibagi 400 tetapi habis dibagi 100, maka tahun itu sudah pasti bukan merupakan tahun kabisat.

Jika angka tahun itu tidak habis dibagi 400, tidak habis dibagi 100 akan tetapi habis dibagi 4, maka tahun itu merupakan tahun kabisat.

Jika angka tahun tidak habis dibagi 400, tidak habis dibagi 100, dan tidak habis dibagi 4, maka tahun tersebut bukan merupakan tahun kabisat.

20. pangrimbage Tembung kasembadan yaiku?​


Wangsulan

➡️ Pangrimbage tembung kasembadan yaiku ater-ater ka + sembada + panambang an.

Ka + sembada + an.

Penjelasan

{Wuwuhan}

- Ater-ater (Awalan)

→ Ater-ater yaiku wuwuhan sing manggone ana ing sangarepe tembung. Ater-ater iku sabenere ana akeh maceme, ning ing kene mung dirembug sing kerep digunakake kaya dene ater-ater anuswara (n-, m-, ng-, ny-), ater-ater tripurusa (dak-, ko-, di-), ka-, sa-, lan pi-.

» Tuladha :

tandur + n- → nandurwaca + m- → macakopi + ng- → ngopicakot + ny- → nyakotjupuk + dak- → dakjupuktutup + ko- → kotutuptabrak + di- → ditabrakgawe + ka- → kagawelusin + sa- → salusinwales + pi- → piwales

- Panambang (Akhiran)

→ Panambang yaiku wuwuhan sing manggon ana ing mburine tembung. Panambang iku sabenere ana akeh maceme, ning ing kene mung dirembug sing kerep digunakake kaya dene -ku, -mu, -e, -an, -i, -a, -na, -ana, -en, lan -ake.

» Tuladha :

sepatu + -ku → sepatukukursi + -mu → kursimusega + -e → segane (*dudu segae)tulis + -an → tulisansuwek + -i → suwekituku + -a → tukuasapu + -na → sapokna (*dudu sapuna)tali + -ana → talenana (*dudu taliana)garap + -en → garapengawa + -ake → gawakake (*dudu gawaake)

- Konfiks yaiku wuwuhan gabungan antarane ater- ater lan panambang sing digunakake bareng ing sajroning tembung lingga.

Tuladha :

Kekeselen » ke + kesel + en.Kewaregen » ke + wareg + en.Kasembadan » ka + sembada + an.Mundhutake » m- + pundhut + ake.Nukokake » n- + tuku + ake.
