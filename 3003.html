Pembahasan

Menggunakan sifat aljabar :

= a² - b²

= ( a - b ) ( a + b )

= 3003² - 3000²

= ( 3003 - 3000 ) ( 3003 + 3000 )

= 3 ( 6003 )

= ( 3 × 6003 )

= 18.009.

Gunakan sifat:

a² - b² = (a - b)(a + b)

...

Maka,

[tex] \rm {3.003}^{2} - {3.000}^{2} [/tex]

[tex] \rm = (3.003 - 3.000)(3.003 + 3.000)[/tex]

[tex] \rm = (3)(3.003 + 3.000)[/tex]

[tex] \rm = (3)(3.003) + (3)(3.000)[/tex]

[tex] \rm = 9.009 + 9.000[/tex]

[tex] \rm = 18.009 [/tex]
