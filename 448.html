Bilangan merupakan suatu konsep dalam matematika yang digunakan untuk melakukan pencacahan dan pengukuran.

Bilangan Bulat terdiri dari :

• Bilangan nol { 0 }

• bilangan positif = { 1, 2, 3, 4, 5 ... }

• bilangan cacah = { 0, 1, 2, 3, 4, ... }

• bilangan bulat negatif = { ..., -5, -4, -3, -2, -1 }

Dalam materi operasi bilangan bulat perkalian, ada inti dasar yang perlu diingat dalam mengerjakan soal, yakni sebagai berikut ;

a. penjumlahan :

1. ( + ) + ( + ) = ( + )

2. ( - ) + ( - ) = ( - )

3. ( + ) + ( - ) = ( + ) / ( - ) apabila bilangan pertama lebih besar maka hasilnya ( + ), sedangkan apabila bilangan kedua lebih besar maka hasilnya ( - ).

4. ( - ) + ( + ) = ( + ) / ( - ) apabila bilangan pertama lebih besar maka hasilnya ( + ), sedangkan apabila bilangan kedua lebih besar maka hasilnya ( - )

b. pengurangan :

1. ( + ) - ( + ) = ( + ) / ( - ) apabila bilangan pertama lebih besar maka hasilnya ( + ), sedangkan apabila bilangan kedua lebih besar maka hasilnya ( - )

2. ( - ) - ( - ) = ( + ) / ( - ) apabila bilangan pertama lebih besar maka hasilnya ( + ), sedangkan apabila bilangan kedua lebih besar maka hasilnya ( - )

3. ( + ) - ( - ) = ( + )

4. ( - ) - ( + ) = ( - )

c. perkalian :

1. ( + ) × ( + ) = ( + )

2. ( - ) × ( - ) = ( + )

3. ( + ) × ( - ) = ( - )

4. ( - ) × ( + ) = ( - )

d. pembagian :

1. ( + ) ÷ ( + ) = ( + )

2. ( - ) ÷ ( - ) = ( + )

3. ( + ) ÷ ( - ) = ( - )

4. ( - ) ÷ ( + ) = ( - )
