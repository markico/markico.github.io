9. KPK (3, 6, 8)

Faktor prima dari 3 = 3

Faktor prima dari 6 = 2 x 3

Faktor prima dari 8 = 2 x 2 x 2 = 2³

KPK (3, 6, 8) =  2³ x 3

= 8 x 3

= 24

Jadi, mereka akan pergi ke kolam bersama-sama lagi setelah 24 hari.

10. (FPB) dari :

A. 24, 48, dan 72

Faktor prima dari 24 = 2 x 2 x 2 x 3 = 2³ x 3

Faktor prima dari 48 = 2 x 2 x 2 x 2 x 3 = 2⁴ x 3

Faktor prima dari 72 = 2 x 2 x 2 x 3 x 3 = 2³ x 3²

FPB (24, 48, 72) = 2³ x 3

= 8 x 3

= 24

B. 28, 84, dan 96

Faktor prima dari 28 = 2 x 2 x 7 = 2² x 7

Faktor prima dari 84 = 2 x 2 x 3 x 7 = 2² x 3 x 7

Faktor prima dari 96 = 2 x 2 x 2 x 2 x 2 x 3 = 2⁵ x 3

FPB (28, 84, 96) = 2²

= 4
