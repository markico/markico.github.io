3. Jika diketahui inflasi yang terjadi pada bulan agustus 2018 20% inflasi pada bulan september 2018 40% dan inflasi pada bulan oktober 2018 50% tentukan besarnya laju inflasi pada bulan september 2018 dan oktober 2018tolong jangan ngasal dan tolong pakai cara​


Jawaban:

Penjelasan:

[tex]Laju \ Inflasi[/tex]

Diketahui:

Inflasi bulan Agustus 2018 sebesar 20%

Inflasi bulan September 2018 sebesar 40%

Inflasi bulan Oktober 2018 sebesar 50%

A. Ditanyakan Laju Inflasi bulan September?

[tex]Laju \ Inflasi = \frac{40 - 20}{20} \ x \ 100 \\\\Laju \ inflasi = \frac{20}{20} \ x \ 100\\\\Laju \ Inflasi = 100[/tex]

Kesimpulan:

Besar laju inflasi pada bulan september adalah sebesar 100%

B. Ditanyakan Laju Inflasi bulan Oktober?

[tex]Laju \ Inflasi = \frac{50 - 40}{40} \ x \ 100 \\\\Laju \ inflasi = \frac{10}{40} \ x \ 100\\\\Laju \ Inflasi = 25[/tex]

Kesimpulan:

Besar laju inflasi pada bulan september adalah sebesar 25%

9. pada bulan Agustus 2018 Felix menabung di bank sebesar Rp 25.000,bulan September Rp.40.000 bulan Oktober Rp.65.000 dan seterusnya. jumlah tabungan sampai dengan February 2019 adalah?​


Jawaban:

Agustus Rp 25.000

September Rp 40.000

Oktober Rp 65.000

Membentuk barisan aritmatika

25.000 ; 40.000 ; 65.000 ; ...

Penjelasan dengan langkah-langkah:

a = 25.000

b = 40.000 - 25.000 = 15.000

Bulan Agustus ke Februari = 7

Makan n = 7

Jumlah tabungan :

Sn = n/2 (2a+(n-1)b)

S7 = 7/2 (2 × Rp 25.000 + (7-1) Rp 15.000)

= 3.5 ( Rp 50.000 + Rp 90.000)

= 3.5 × Rp 140.000

= Rp 490.000


10. Data indeks harga konsumen negara Y pada bulan Oktober 2018 sebesar 125,25 dan bulan September 2018 sebesar 110,95. Berdasarkan data tersebut, laju inflasi pada bulan Oktober sebesar..​


Jawaban:

Data indeks harga konsumen negara Y pada bulan Oktober 2018 sebesar 125,25 dan bulan September 2018 sebesar 110,95. Berdasarkan data tersebut, laju inflasi pada bulan Oktober sebesar 12,8%

Penjelasan:

Pada soal diatas, sudah diketahui besarnya indeks harga yaitu pada bulan september indeks harga sebesar 110,95 sedangkan pada bulan oktober sebesar 125,25 maka

In = IHKn - IHKn-1 / IHKn-1 x 100%

Keterangan :

In = inflasi

IHKn = Indeks Harga Konsumen tahun dasar (biasanya nilainya 100)

IHKn–1 = Indeks Harga Konsumen tahun sebelumnya

Inflasi  = 125,25 - 110,95 / 110,95 x 100%

Inflasi = 12,8%

27. Sebuah toko mulai dibuka pada bulan Oktober 2016. Penjulan buku di setiap bulannya membentuk barisan aritmatika. Pada bulan Desember 2016 toko tersebut mampu menjual sebanyak 24 buku, pada bulan Juni 2017 mampu menjual sebanyak 66 buku. Berapakah banyak buku yang terjual pada bulan Oktober 2018 ?


Penjelasan dengan langkah-langkah:

Diket :

a = U1 = 24 dan U2 = 66

b = U2 - U1 = 66 - 24 = 42

Tentukan banyak buku yang terjual pada bulan Oktober 2018 (U3) !

Jawab :

Un = a + (n - 1).b

U3 = 24 + (3 - 1).42

U3 = 24 + 126 - 42

U3 = 108

Jadi, banyak buku yang terjual pada bulan Oktober 2018 adalah sebanyak

108 buku

Semoga Bermanfaat
