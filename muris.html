13. 1. Nilai rata rata hasil ulangan fisika dari 39 murid adalah 6,5. Jika seorang anak mengikuti ulangan susulan, nilai rata rata tersebut menjadi 6,56. Berapakah nilai muris yang mengikuti ulangan susulan? 2. Diketahui 1per3 bagian uang andi sama dengan 3 kali uang budi, sedang jumlah uang andi dan budi adalah Rp. 13.500. Selisih uang mereka adalah.. jawaban beserta cara dan penjelasan thx


1.
x(39) = 6,5
x(40) = 6,56
x' = ...
x(40).40 = x(39).39 + x'
6,56 x 40 = 6,5 x 39 + x'
262,4 = 253,5 + x'
x' = 262,4 - 253,5
x' = 8,9

2.
1/3 A = B (Definisi soal)
Anggap A(Andi) dan B(Budi)
A + B = 13.500
A + 1/3 A = 13.500
4/3 A = 13.500
A = 15.000 x 3/4
A = 11.250
dan,
B = 1/3 A
B = 1/3 x 11.250
B = 3.750
Maka selisihnya:
A - B = 11.250 - 3.750
A - B = Rp 7.500
