14. Diketahui tabungan awal Billy di bank sebesar Rp2.000.000,00 dan bank memberi bunga sebesar 9% per tahun. Jika total bunga tabungan Billy Weasly Rp420.000,00, maka lama Billy menabung adalah


Bunga = 9% × 2.000.000,-

= 9/100 × 2.000.000,-

= 180.000,-

Bunga perbulan = 180.000,- ÷ 12

= 15.000,-

Lama menabung = 420.000,- ÷ 15.000,-

= 28 bulan

Jadi lama Billy menabung adalah 28 bulan

27. uang alda Rp20.000,00 lebih banyak dari uang billy di tambah dua kali uang candra .jumlah uang alda,billy,dan candra RP190.000,00 selisi uang billy dan cindy RP10.000.00 jumlah uang alda dan billy adalah​


Jawaban:

Rp. 152.000

Penjelasan dengan langkah-langkah:

Alda = A

Billy = B

Candra = C

Uang Alda Rp. 20.000 lebih banyak dari uang Billy ditambah dua kali uang Candra:

A + 20.000 = B + 2C

A - B - 2C = -20.000.........(1)

Jumlah uang mereka Rp. 190.000:

A + B + C = 190.000.........(2)

Selisih uang Billy dan Candra:

|B - C| = 10.000......(3)

Eliminasi persamaan 1 dan 2:

A - B - 2C = -20.000

A + B + C = 190.000

——————————— -

-2B - 3C = -210.000

2B + 3C = 210.000.........(4)

Eliminasi persamaan 3 dan 4:

B - C = 10.000 |×2| 2B - 2C = 20.000

2B + 3C = 210.000 |×1 | 2B + 3C = 210.000

—————————— -

-5C = -190.000

C = 38.000

B - C = 10.000

B - 38.000 = 10.000

B = 48.000

A + B + C = 190.000

A = 190.000 - 48.000 - 38.000

A = 104.000

Jumlah uang Alda dan Billy:

A + B = 104.000 + 48.000

A + B = 152.000
