21. Nomor 2 sama nomor 3 saja ​


Tentukan himpunan penyelesaian dari persamaan kuadrat dengan menggunakan rumus abc!

2) [tex]-x^2 + 3x + 10 = 0[/tex]

3) [tex]2x^2 - 3x -5 = 0[/tex]

Berikut adalah penyelesaian dari himpunan-himpunan tersebut:

2) [tex]-x^2 + 3x + 10 = 0[/tex] --> HP = {2, -5}.

3) [tex]2x^2 - 3x - 5 = 0[/tex] --> HP = {2,5 , 1).

Penjelasan dengan langkah-langkah

2) Diketahui: [tex]-x^2 + 3x + 10 = 0[/tex]

Ditanya: Penyelesaian himpunan?

Jawab:

x = -b ± √(b^2 – 4ac) / 2a

x = -3 ± √(3^2 – 4(-1)(10)) / 2(1)

x = -3 ± √(9 + 40) / 2

x = -3 ± √(49) / 2

x = -3 ± 7 / 2

x1 = -3 + 7 / 2 = 4/2 = 2

x2 = -3 – 7 / 2 = -10/2 = -5

Sehingga HP dari persamaan tersebut adalah HP = {2, -5}.

3) Diketahui: [tex]2x^2 -3x -5 = 0[/tex]

Ditanya: Penyelesaian himpunan?

Jawab:

x = -b ± √b^2 – 4ac / 2a

x = -(-3) ± √((-3)^2 – 4(2)(-5)) / 2(2)

x = 3 ± √(9 + 40) / 4

x = 3 ± 7 / 4

x1 = 3 + 7 / 4 = 10/4 = 2,5.

X2 = 3 – 7 / 4 = -4/4 = -1.

Sehingga HP dari persamaan tersebut adalah HP = {2,5 , 1).
