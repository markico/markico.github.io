4. (Aplikasi)Toko kue menyediakan berbagai jenismanisan, terdiri dari manisan mangga,manisan nangka, dan manisan kolang,kaling. Perbandingan masing-masing jumlahmanisan adalah 5:9:13.Jika jumlah manisanmangga dan manisan nangka adalah 210 kg,maka selisih antara manisan kolang-kalingdan manisan mangga adalah ... kg.​


Manisan mangga : nangka : kolang-kaling = 5 : 9 : 13
Manisan mangga + nangka = 210 kg
Selisih manisan kolang-kaling dan mangga?

Mangga/nangka = 5/9
m/n = 5/9
m = 5/9 n

m + n = 210
5/9 n + n = 210
5/9 n + 9/9 n = 210
14/9 n = 210
n = 210 x 9/14
n = 15 x 9
n = 135 kg

m = 5/9 n = 5/9 (135) = 5 x 15 = 75 kg

mangga/kolang-kaling = 5/13
m/k = 5/13
75/k = 5/13
75 x 13/5 = k
15 x 13 = k
195 kg = k

Maka k - m = 195 - 75 = 120 kg
