15. Bu marisa membeli 10 potong kemeja .harga setiap kemeja Rp90.000.00.bu marisa memperoleh diskon sebesar 5% untuk setiap kemeja kemudian bu marisa menjual semua kemeja dengan keuntungan 10% harga jual 1 kemeja adalah


Bu marisa membeli 10 potong kemeja . Harga setiap kemeja Rp90.000.00. Bu marisa memperoleh diskon sebesar 5% untuk setiap kemeja kemudian Bu marisa menjual semua kemeja dengan keuntungan 10% harga jual 1 kemeja adalah Rp. 94.050

PEMBAHASAN

Sebelum mengerjakan soal nya kita perlu memahami beberapa rumus dasar tentang Aritmetika Sosial ini , yakni :

Untung = Harga Jual - Harga Beli

%Untung = ( Untung / Harga Beli ) x 100%

Rugi = Harga Beli - Harga Jual

%Rugi = ( Rugi / Harga Beli ) x 100%

Tara = Bruto - Netto

%Tara = ( Tara / Bruto ) x 100%

Okay dengan rumus dasar ini mari kita selesaikan soal yang dimaksud.

Bu marisa membeli kemeja dengan harga setiap kemeja Rp90.000.00 dan mendapat diskon 5%, maka :

Harga Beli Setiap Kemeja = ( 100%-5% ) x Rp. 90.000

Harga Beli Setiap Kemeja = ( 95% ) x Rp. 90.000

Harga Beli Setiap Kemeja = Rp. 85.500

Diketahui keuntungan 10% maka bisa di hitung :

Harga Jual Setiap Kemeja = ( 100% + 10% ) x Rp. 85.500

Harga Jual Setiap Kemeja = ( 110% ) x Rp. 85.500

Harga Jual Setiap Kemeja = ( 110% ) x Rp. 85.500

Harga Jual Setiap Kemeja = Rp. 94.050
