26. diketahui kubus PQRS.TUVW mempunyai sisi 4Cm. Titik A pada PQ sehingga PA = 1 Cm, titik B pada UV sehingga BV = 1 Cm, C titik tengah SW. Jarak C ke AB adalah


Jawab:

Diketahui kubus PQRS.TUVW mempunyai sisi 4 cm. Titik A pada PQ sehingga PA = 1 cm. Titik B pada UV sehingga BV = 1 cm. C titik tengah SW. Jarak C ke AB adalah....



Pendahuluan :

Untuk mencari jarak titik ke garis, maka perlu diproyeksikan titik tersebut ke dalam garis.


Pembahasan dengan langkah-langkah :

Jarak titik C ke AB adalah jarak titik C ke garis AB secara tegak lurus. Untuk mencari jarak titik C ke garis AB maka perlu diketahui terlebih dahulu panjang garis AB, garis BC, dan garis AC.  

Dari soal diketahui bahwa panjang sisi kubus = 4 cm

AP = 1 cm, PQ = 3 cm

UB = 3 cm, BV = 1 cm

SC = 2 cm, CW = 2 cm

Selanjutnya akan dicari panjang AB, BC, dan AC dengan denggunakan dalil phytagoras.


1. Panjang garis AB

Perhatikan gambar kubus dibawah

Garis AB merupakan garis yang menghubungkan titik A pada bidang PQUT dengan titik B pada bidang QRVU. Proyeksi titik A ke garis UV adalah AU.  

Panjang AU = [tex]\sqrt{AQ^{2} + QU^{2} } = \sqrt{3^{2} + 4^{2} } = \sqrt{25} = 5[/tex]  

Panjang AB = [tex]\sqrt{AU^{2} + UB^{2} } = \sqrt{5^{2} + 3^{2} } = \sqrt{34} [/tex]


2. Panjang garis BC

Perhatikan gambar kubus dibawah

Garis BC merupakan garis yang menghubungkan titik B pada bidang QRUV dengan titik C pada bidang SRVW. Proyeksi titik B ke garis SW adalah BW.  

Panjang BW = [tex]\sqrt{BV^{2} + VW^{2} } = \sqrt{1^{2} + 4^{2} } = \sqrt{17}[/tex]  

Panjang BC = [tex]\sqrt{BW^{2} + CW^{2} } = \sqrt{(\sqrt{17})^{2} + 2^{2} } = \sqrt{21} [/tex]


3. Panjang garis AC

Perhatikan gambar kubus dibawah

Garis AC merupakan garis yang menghubungkan titik A pada bidang PQUT dengan titik C pada bidang PSWT Proyeksi titik A ke garis SW adalah AS.  

Panjang AS = [tex]\sqrt{PA^{2} + PS^{2} } = \sqrt{1^{2} + 4^{2} } = \sqrt{17}[/tex]  

Panjang AC = [tex]\sqrt{AS^{2} + SC^{2} } = \sqrt{(\sqrt{17})^{2} + 2^{2} } = \sqrt{21} [/tex]


4. Perhatikan gambar segitiga ABC

Dari ketiga langkah sebelumnya didapatkan bahwa panjang AB = [tex]\sqrt{34} [/tex], panjang BC = [tex]\sqrt{21} [/tex], panjang AC = [tex]\sqrt{21} [/tex]

Karena panjang AC = panjang BC, maka segitiga ABC merupakan segitiga sama kaki.  

Jarak titik C ke garis AB adalah CO.

Panjang garis CO = [tex]\sqrt{CB^{2} -OB^{2} } =\sqrt{(\sqrt{21})^{2} -(\frac{\sqrt{34} }{2} )^{2} } = \sqrt{21 - \frac{34}{4} } = \sqrt{\frac{50}{4} } = \frac{5\sqrt{2}}{2} [/tex]


Jadi, jarak titik C ke garis AB adalah [tex] \frac{5\sqrt{2}}{2} [/tex]
