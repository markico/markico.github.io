13. umur pak dani 28 tahun lebih tua dari umur maria umur bu dani 6 tahun lebih muda dari pa dani jika jumlah umur pa dani, bu dani dan maria 119 tahun, maka junlah umur maria dan bu dani adalah


Materi SPLTV

Misalkan umur Pak Dani = x, Umur Bu Dani = y, dan umur Maria = z.

cari persamaannya
x = z + 28 .............. persamaan 1
y = x - 6 .................. persamaan 2
x + y + z = 119 ....... persamaan 3

subtitusikan persamaan 1 ke persamaan 2
=> y = (z + 28) - 6
=> y = z + 22 .......... persamaan 4

subtitusi persamaan 1 &4 ke persamaan 3
=> x + y + z = 119
=> (z + 28) + (z + 22) + z = 119
=> 3z + 50 = 119
=> 3z = 69
=> z = 23
Jadi Umur Maria = 23 Tahun

substitusi z = 23 ke persamaan 1
=> x = z + 28
=> x = 23 + 28 = 51
Jadi Umur Pak Dani = 51 Tahun

substitusi z = 23 ke persamaan 4
=> y = z + 22
=> y = 23 + 22 = 45
Jadi Umur Bu Dani = 45 Tahun

Jumlah umur Maria dan Bu Dani :
=> z + y
=> 23 + 45 = 68 Tahun.

27. Dani bersepeda dengan kecepatan 4 meter/detik. jika dani bersepeda dalam 20 menit. berapa jarak yang ditempuh oleh dani


Jawaban:

4800 meter

Penjelasan dengan langkah-langkah:

V = S/t

S = V. t

Diketehui V = 4 meter/detik

t = 20 menit

Jadikan menit ke detik. 1 menit = 60 detik,

20 menit = 1200 detik

S = V x t

S = 4m/detik x 1200detik

S = 4800 meter
