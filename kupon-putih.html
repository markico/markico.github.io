18. Seorang pemilik toko swalayan akan membuat kupon undian bagi para pelanggannya. Kupon dibuat dari angka: 1, 2, 3, 4, 5, 6, dan 7. Jika kupon disusun berdasarkan nilai yang terkecil, maka kupon dengan nomor 6741325 berada pada urutan ke... ?


Urutan ke-1 :

=> Kupon nomor 1

=> 1 = 7⁰ x 1 = 1 x 1 = 1

Urutan ke-2 :

=> Kupon nomor 2

=> 2 = 7⁰ x 2 = 1 x 2 = 2

Urutan ke-3 :

=> Kupon nomor 3

=> 3 = 7⁰ x 3 = 1 x 3 = 3

...

Urutan ke-7 :

=> Kupon nomor 7

=> 7 = 7⁰ x 7 = 1 x 7 = 7

Urutan ke-8 :

=> Kupon nomor 11

=> 11 = (7⁰ x 1) + (7¹ x 1) = (1 x 1) + (7 x 1) = 1 + 7 = 8

...

dst.

Maka, kupon nomor 6741325 berada pada urutan ke- :

(7⁰ x 5) + (7¹ x 2) + (7² x 3) + (7³ x 1) + (7⁴ x 4) + (7⁵ x 7) + (7⁶ x 6)

= (1 x 5) + (7 x 2) + (49 x 3) + (343 x 1) + (2.401 x 4) + (16.807 x 7) + (117.649 x 6)

= (5) + (14) + (147) + (343) + (9.604) + (117.649) + (705.894)

= 833.656

Jadi, kupon nomor 6741325 berada pada urutan ke - 833.656


19. Ada 100 kupon berhadiah dan budi membeli2 kupon. Peluang budi mendapat hadiah adalah


Jawaban:

[tex] \frac{2}{100} = \frac{1}{50} = 00.2[/tex]

Penjelasan dengan langkah-langkah:

Topik: Peluang

dik:

n(p) = 2 (kemungkinan menang)

n(s) = 100 (banyak ruang sampel)

dit:

Peluang budi terpilih

jawab:

Budi akan mendapat hadiah apabila 2 kupon yang dia punya lah yang dipilih dari 100 kupon yang ada

[tex]p = \frac{n(p)}{n(s)} = \frac{2}{100} = \frac{1}{50} = 0.02[/tex]

Mapel : Matematika

Materi : peluamg

Kelas : Sekolah Menengah Pertama

Peluang Budi untuk mendapat hadiah adalah 1/50 (dalam pecahan biasa) atau 0,02 (dalam pecahan desimal) atau 2% (dalam persenan)

Cara :

=Jumlah kupon yang dimiliki Budi ÷ Jumlah kupon yang ada

= 2 kupon ÷ 100 kupon

= 0,02

Rumus kombinasi:

nCr = (n!)/r! (n - r)!

20C15 = 20!/15! (20 - 15)!

28. Dari 20 kupon undian akan diambil dua kupon untuk menentukan juara pertama dan kedua. Tentukan banyak cara untuk menentukan 2 kupon tersebut !


Jika kupon undian diambil satu per satu dan tentunya tanpa pengembalian, maka

(i) banyak cara mengambil kupon yang pertama adalah C(20,1) = 20
(ii) banyak cara mengambil kupon yang kedua adalah C(19,1) = 19

sehingga banyak cara mengambil kupon adalah 20 x 19 = 380.

Jika kupon diambil secara bersamaan, maka banyak cara mengambil kupon adalah C(20,2) = 20! / (18! x 2!) = 20 x 19 / 2 x 1 = 190.

30. sebuah perusahaan menyelenggarakan baksos dgn memberi sumbangan berupa kupon yang dapat ditukar ke beras. kupon a 3kg beras dan kupon b 2kg beras. jika jumlah kupon semua 200 dan jumlah beras 425kg, brp jml kupon a?


3 kg = 25 kupon
2 kg = 175 kupon

3 × 25 = 75
2 × 175 = 350

75 + 350 = 425

jadi jumlah kupon a sebanyak 25 kupon.
