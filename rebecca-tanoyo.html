13. Rebecca menyimpan uangnya di bank sebesar Rp. 2.400.000. Setelah 1 tahun uang tabungannya menjadi Rp. 2.640.000. Tentukan:a. Besar bunga dalam 1 tahun! b. Suku bunga dalam 1 tahun! c. Besar bunga dalam 8 bulan!d. Jumlah tabungan Rebecca jika setelah 8 bulan di ambil semua?​


Diketahui:

tabungan awal = 2.400.000

n = 1 tahun

tabungan akhir = 2.640.000

Ditanya:

%bunga

Jawab:a. Tabungan akhir - Tabungan awal = Tabungan awal x %bunga x n

2.640.000 - 2.400.000 = 2.400.000 x %bunga x 1 tahun

240.000 = 2.400.000 x %bunga

%bunga = 240.000 : 2.400.000

%bunga = 10% per tahunb. suku bunga per bulan

n = 10% : 12

= 0,83% per bulan

c. %bunga dalam 8 bulan

= 8/12 x 10%

= 6,67% per 8 bulan

d. Jumlah tabungan

= tabungan awal ( 1 + %bunga x n)

= 2.400.000 (1 + 10% x 8/12)

= 2.400.000  x 106,67%

= 2.560.000

- Waktu = 30 menit = 1/2 jam

- Kecepatan :
