Berapakah harga 1kg Apel 1kg jeruk dan 1kg anggur ?? Jika aminah 3kg apel 5kg anggur dan 2kg jeruk. berapakah uang yang harus dikeluarkan untuk pembelian buah tersebut ?


misalkan     :   apel = x   anggur=y   jeruk z

aisah :     5x+3y+z = 115000

aminah:  6x+2y+z = 112000

abidah:   3x+3y+z = 106000

kita eliminasi aisah dan abidah dulu

5x+3y+z = 115000

3x+3y+z = 106000

--------------------------- --

2x           = 9000      maka x = 9000/2 = 4500   (apel)

sekarang kita eliminasi aisah dan aminah,,,lalu masukan harga apelnya

5x+3y+z = 115000

6x+2y+z = 112000

--------------------------- --

-x + y     = 3000

      y     = 3000 + x = 3000 + 4500 = 7500 (anggur)

harga apel dan anggur kita masukan ke belanjaan aisah untuk mendapat harga jeruk

5( 4500) + 3(7500) + z = 115000

22500 + 22500 + z = 115000

45000 + z = 115000
