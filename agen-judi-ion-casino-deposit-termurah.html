10. Apa kesimpulan dari ion sulfat hanya berperan sebagai agen pengoksidasi atau oksidator


Ion sulfat hanya berperan sebagai agen pengoksidasi atau oksidator karena bilangan oksidasi S pada Ion sulfat = SO₄²⁻ memiliki bilangan oksidasi paling tinggi, sehingga tidak bisa mengalami oksidasi (reduktor) tetapi mengalami reduksi (oksidator).

Pembahasan

[tex]\boxed{\boxed{\bold{Konsep~Reaksi~Oksidasi~dan~Reduksi }}}[/tex]

[tex]\mathbf{C.~Berdasarkan~Perubahan~Bilangan~Oksidasi }[/tex]

Reaksi redoks adalah reaksi dimana terjadi perubahan biloks dari atom unsur sebelum dan sesudah reaksi, perubahan biloks tersebut berupa reaksi oksidasi yaitu kenaikan biloks dan reaksi reduksi yaitu penurunan biloks.

Oksidasi adalah pertambahan (kenaikan) bilangan oksidasi, sedangkan zat yang mengalami oksidasi disebut reduktor

Reduksi adalah penurunan bilangan bilangan oksidasi, sedangkan zat yang mengalami reduksi disebut oksidator

[tex]\mathbf{Aturan~Penentuan~Bilangan~Oksidasi }[/tex]

1. Bilangan okidasi atom dalam unsur bebas sama dengan 0

2. Bilangan oksidasi ion monoatom sama dengan muatan ionnya

3. Jumlah biloks dalam Senyawa sama dengan 0. Sedangkan Jumlah biloks dalam Ion Poliatom sama dengan Muatan Ionnya.

4. Biloks unsur-unsur golongan VII A dalam senyawa biner logam adalah -1

5. Biloks H jika berikatan dg non logam sama dengan +1. Sedangkan biloks H jika barikatan dg logam dan boron adalah -1.

6. Biloks O dalam senyawa sama dengan -2, kecuali dalam senyawa biner fluorida, peroksida, dan superoksida.

7. Biloks logam golongan IA dalam senyawa sama dengan +1

8. Biloks logam golongan IIA dalam senyawa sama dengan +2


[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]

Ion sulfat = SO₄²⁻

1. biloks S + 4. biloks O = muatan

biloks S + 4. (-2) = -2

biloks S = -2 + 8

biloks S = +6

biloks S = +6 merupakan biloks tertinggi dari S sehingga sudah tidak dapat mengalami oksidasi (reduktor) maka hanya dapat mengalami reduksi (oksidator).


Contoh bilangan oksidasi S selain +6

a)H₂S

2. biloks H + biloks S = 0

2. (+1) + biloks S = 0

biloks S = -2


b)SO₂

biloks S + 2. biloks O = 0

biloks S + 2. (-2) = 0

biloks S = +4


c)H₂SO₄

2. biloks H + biloks S + 4. biloks O = 0

2. (+1) + biloks S + 4. (-2) = 0

+2 + biloks S - 8 = 0

biloks S = + 8 - 2

biloks S = +6


d)SO₃

biloks S + 3. biloks O = 0

biloks S + 3. (-2) = 0

biloks S = +6


e)H₂SO₃

2. biloks H + biloks S + 3. biloks O = 0

2. (+1) + biloks S + 3. (-2) = 0

+2 + biloks S - 6 = 0

biloks S = + 6 - 2

biloks S = +4


[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]

bilangan oksidasi S pada Ion sulfat = SO₄²⁻ memiliki bilangan oksidasi paling tinggi, sehingga tidak bisa mengalami oksidasi (reduktor) tetapi mengalami reduksi (oksidator).

19. apa yang dimaksud dengan primary deposit dan loan deposit


Primary deposit dan loan deposit adalah cara-cara untuk menciptakan uang giral.

Primary deposit adalah cara menciptakan uang giral dengan cara nasabah menyetorkan uang kartalnya ke bank untuk dicatat dalam rekening koran miliknya sebagai uang giral.

Loan deposit adalah cara menciptakan uang giral yang terjadi ketika seorang nasabah meminjam uang ke bank, namun uang tsb tidak ditarik menjadi uang kartal, namun tetap dibiarkan didalam rekening korang miliknya.
