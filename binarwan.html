7. Uang adinda Rp. 40.000 lebih banyak dari uang binar ditambah dua kali uang cindy. Jumlah uang Adinda, binar dan cindy rp. 200.000, selisih uang binary dan cindy Rp.100.000. Jumlah uang adinda dan binar??


Uang Adinda Rp40.000,00 lebih banyak dari uang Binar ditambah dua kali uang Cindy. Jumlah uang Adinda, Binar dan Cindy Rp200.000,00, selisih uang Binar dan Cindy Rp100.000,00, maka jumlah uang Adinda dan Binar adalah Rp172.000,00. Ada kesalahan dalam soal, seharusnya selisih uang Binar dan Cindy adalah Rp10.000,00 bukan Rp100.000,00. Cara penyelesaiannya bisa menggunakan metode eliminasi atau metode substitusi. Metode eliminasi adalah suatu metode untuk mencari penyelesaian dengan cara menyamakan salah satu variabel yang samanya, sehinga diperoleh nilai dari variabel yang lain. Metode substitusi adalah suatu metode untuk mencari penyelesaian dengan cara memasukkan suatu persamaan satu ke persamaan lain. Soal ini merupakan penerapan materi sistem persamaan linear dua variabel

Pembahasan

Misal

A = jumlah uang Adinda B = jumlah uang Binar C = jumlah uang Cindy

Uang Adinda Rp40.000,00 lebih banyak dari uang Binar ditambah dua kali uang Cindy

A = 40.000 + B + 2C ............ persamaan (1)

Jumlah uang Adinda, Binar dan Cindy Rp200.000,00

A + B + C = 200.000 ............. persamaan (2)

Selisih uang Binar dan Cindy Rp10.000,00

B – C = 10.000

B = C + 10.000 ............ persamaan (3)

Substitusikan persamaan (3) ke persamaan (1)

A = 40.000 + B + 2C

A = 40.000 + (C + 10.000) + 2C

A = 3C + 50.000 .............. persamaan (4)

Substitusikan persamaan (3) dan (4) ke persamaan (2)

A + B + C = 200.000

(3C + 50.000) + (C + 10.000) + C = 200.000

5C + 60.000 = 200.000

5C = 140.000

C = 28.000

Substitusikan nilai C = 28.000 ke persamaan (2)

A + B + C = 200.000

A + B + 28.000 = 200.000

A + B = 172.000

Jadi jumlah uang Adinda dan uang Binar adalah Rp172.000,00

Jika Selisih uang Binar dan Cindy tetap Rp100.000,00, maka jawabannya akan negatif, dan tidak mungkin jumlah uang yang mereka miliki adalah negatif,  

B – C = 100.000

B = C + 100.000 ............ persamaan (3)

Substitusikan persamaan (3) ke persamaan (1)

A = 40.000 + B + 2C

A = 40.000 + (C + 100.000) + 2C

A = 3C + 140.000 .............. persamaan (4)

Substitusikan persamaan (3) dan (4) ke persamaan (2)

A + B + C = 200.000

(3C + 140.000) + (C + 100.000) + C = 200.000

5C + 240.000 = 200.000

5C = –40.000

C = –8.000 (tidak mungkin uang Cindy negatif)

Jadi memang soalnya harus diralat

27. Uang adinda Rp. 40.000 lebih banyak dari uang binary ditambah dua kali uang cindy. Jumlah uang Adinda, binar dan cindy rp. 200.000, selisih uang binary dan cindy Rp.10.000. Jumlah uang adinda dan binary??


uang adinda = a
uang binary = b
uang cindy = c

a = (40.000 + b) + (2 × c)
a + b + c = 200.000
b - c = 10.000 ........ b = 10.000 + c
a + b = ?

maka
a = (40.000 + b) + (2 × c)
a = 40.000 +(10.000 + c) + 2c
a = 50.000 + c + 2c
a = 50.000 + 3c

substitusi
a + b + c = 100.000
(50.000 + 3c) +(10.000 + c) + c = 200.000
60.000 + 5c = 200.000
5c = 200.000 - 60.000
5c = 140.000
c = 140.000/5
c = 28.000

b = 10.000 + c
b = 10.000 + 28.000
b = 38.000

a = 50.000 + 3c
a = 50.000 + (3×28.000)
a = 50.000 + 84.000
a = 134.000

sehingga a + b = 134.000 + 38.000
= 172.000

jumlah uang adinda dan binary 172.000
