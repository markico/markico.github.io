Sifat-sifat eksponen yang digunakan pada soal ini adalah Â sebagai berikut:

[tex] \boxed{ \ x^{a}. x^{b} \leftrightarrow x^{a+b} \ } [/tex][tex] \boxed{ \ x^{a} : x^{b} \leftrightarrow \frac{x^{a}}{x^{b}} = x^{a-b} \ } [/tex][tex] \boxed{ \ \frac{1}{x^{a}} \leftrightarrow x^{-a} \ } [/tex][tex] \boxed{ \ x^0 = 1 \ } [/tex]

[Soal A]

[tex]\boxed{ \ 18t^3 \times 2t^{-3} = \ ? \ }[/tex]

[tex]\boxed{ \ 18t^3 \times 2t^{-3} = 18 \times 2 \times t^{(3 + (-3))} \ }[/tex]

[tex]\boxed{ \ 18t^3 \times 2t^{-3} = 36 \times t^0 \ }[/tex]

[tex]\boxed{ \ 18t^3 \times 2t^{-3} = 36 \times 1 \ }[/tex]

Jadi bentuk sederhana dari [tex]\boxed{ \ 18t^3 \times 2t^{-3} \ }[/tex] adalah 36.

[Soal B]

[tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} = \ ? \ }[/tex]

[tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} = 2y^{(0-6)}t^{(3+2)}} \ }[/tex]

[tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} = 2y^{-6}t^{5}} \ }[/tex]

Hasil ini dapat ditulis dalam bentuk pangkat positif sebagai,

[tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} = \frac{2t^{5}}{y^6}} \ }[/tex]

Jadi bentuk sederhana dari [tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} = \ ? \ }[/tex] adalah [tex]\boxed{ \ \frac{2t^{5}}{y^6}} \ }[/tex]

Quick Steps:

Pindahkan variabel dengan pangkat lebih rendah menuju ke variabel sejenis dengan pangkat lebih tinggi.

[tex]\boxed{ \ \frac{2y^0t^3}{y^6t^{-2}} \rightarrow \boxed{ \ \frac{2t^{(3+2)}}{y^{(6-0)}}} \rightarrow \boxed{ \ \frac{2t^{5}}{y^6}} \ }[/tex]

[Soal C]

[tex]\boxed{ \ 2m^0 \times m^{-7} = \ ? \ }[/tex]

[tex]\boxed{ \ 2m^0 \times m^{-7} = 2 \times m^{0 + (-7)} \ }[/tex]

[tex]\boxed{ \ 2m^0 \times m^{-7} = 2 \times m^{0-7} \ }[/tex]

[tex]\boxed{ \ 2m^0 \times m^{-7} = 2 \times m^{-7} \ }[/tex]

Hasil ini dapat ditulis dalam bentuk pangkat positif sebagai,

[tex]\boxed{ \ 2m^0 \times m^{-7} = \frac{2}{m^7}} \ }[/tex]

Jadi bentuk sederhana dari [tex]\boxed{ \ 2m^0 \times m^{-7} \ }[/tex] adalah [tex]\boxed{ \ \frac{2}{m^7}} \ }[/tex]

Quick Steps:

[tex]\boxed{ \ 2m^0 \times m^{-7} \rightarrow \boxed{ \ 2 \times 1 \times m^{-7} \ } \rightarrow \boxed{ \ \frac{2}{m^7}} \ }[/tex]

[Soal D]

Kemungkinan soal tipe-1: [tex]\boxed{ \ \frac{m^{-3} + 4}{m^{-3}} = \ ? \ }[/tex]

[tex]\boxed{ \ \frac{m^{-3} + 4}{m^{-3}} = \frac{m^{-3}}{m^{-3}} + \frac{4}{m^{-3}} \ }[/tex]

[tex]\boxed{ \ \frac{m^{-3} + 4}{m^{-3}} \rightarrow \boxed{ \ 1 + 4m^{3} \ } \ }[/tex]

Kemungkinan soal tipe-2: [tex]\boxed{ \ m^3 + \frac{4}{m^{-3}} = \ ? \ }[/tex]

[tex]\boxed{ \ m^3 + \frac{4}{m^{-3}} = \ m^3 + 4m^3 \ }[/tex]

[tex]\boxed{ \ m^3 + \frac{4}{m^{-3}} \rightarrow \boxed{ \ \ 5m^3 \ } \ }[/tex]

Ingat, [tex]\boxed{ \ px + qx = (p + q)x \ }[/tex]

Harap disimak dengan cermat untuk Soal D. Terdapat dua kemungkinan penulisan soal yang dapat digunakan sebagai sarana latihan.
