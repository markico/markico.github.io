6. de simpulkanJumlah besarBesar sudut pada sudut pada segibonyokNomaBanyak sisisegi banyak​


Jawab:

1. Segi tiga / Trigon

Banyak sisi = 3

Besar sudut = (n-2) x 180° / n = (3-2) x 180° / 3 = 60°

Jumlah besar sudut = (n-2) x 180° = (3-2) x 180° = 180°

2. Segi empat / Tetragon

Banyak sisi = 4

Besar sudut = (n-2) x 180° / n = (4-2) x 180° / 4 = 90°

Jumlah besar sudut = (n-2) x 180° = (4-2) x 180° = 360°

3. Segi lima / Pentagon

Banyak sisi = 5

Besar sudut = (n-2) x 180° / n = (5-2) x 180° / 5 = 108°

Jumlah besar sudut = (n-2) x 180° = (5-2) x 180° = 540°

4. Segi tujuh / Heptagon

Banyak sisi = 7

Besar sudut = (n-2) x 180° / n = (7-2) x 180° / 7 = 128,5°

Jumlah besar sudut = (n-2) x 180° = (7-2) x 180° = 900°

5. Segi enam / Hexagon

Banyak sisi = 6

Besar sudut = (n-2) x 180° / n = (6-2) x 180° / 6 = 120°

Jumlah besar sudut = (n-2) x 180° = (6-2) x 180° = 720°

Jadi, semakin banyak sisi yang membentuk suatu bangun maka semakin besar besaran tiap sudut dan jumlah besar sudutnya.
