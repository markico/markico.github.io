Misalkan Luaran yang dihasilkan apabila dimasukkan bilangan n adalah Un, maka diperoleh persamaan berikut:

U₁ = 1 = 1³ + 0 = 1³ + (1 - 1)

U₂ = 9 = 2³ + 1 = 2³ + (2 - 1)

U₃ = 29 = 3³ + 2 = 3³ + (3 - 1)

U₄ = 67 = 4³ + 3 = 4³ + (4 - 1)

U₅ = 129 = 5³ + 4 = 5³ + (5 - 1)
