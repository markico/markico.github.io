3. Sebuah saham PT. Sinarmas disepakati banyak analis tidak akan membagikan dividen. Jika harganya tahun depan adalah Rp 4000 sedangkan harga pasarnya sekarang adalah Rp 3200, berapakah return yang diharapkan investor?


Untuk menghitung return yang diharapkan investor, kita dapat menggunakan rumus return yang diharapkan (expected return) yaitu:

expected return = (harga tahun depan - harga saat ini) / harga saat ini

Dengan menggunakan rumus tersebut, maka return yang diharapkan investor adalah:

expected return = (4000 - 3200) / 3200

expected return = 800 / 3200

expected return = 0,25 atau 25%

Jadi, return yang diharapkan investor adalah 25%.

Siklus akuntansi perusahaan dagang tidak berbeda jauh dengan siklus akuntansi perusahaan jasa. Yang membedakan hanyalah pada perhitungan laba/rugi dan pelakuan akuntansi untuk akun akun khusus yang terdapat pada perusahaan dagang. Perusahaan dagang ialah perusahaan yang kegiatan usahanya membeli barang dagangan dengan tujuan untuk dijual kembali tanpa mengubah bentuk dari barang tersebut, seperti contohnya agen, toko, dan supermarket.
