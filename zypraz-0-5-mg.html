17. berapakah pH larutan Mg (OH)2 jika diketahui kelarutan Mg (OH) adalah 5×10^-5?


Dik : sMg(OH)2= 5 x 10^-5
Dit : pH Mg(OH)2
Jwab :
Mg(OH)2 --> Mg^2+ + 2OH-
s. 2s
Karena kelarutan sudah diketahui nilai nya maka,
[OH-] = 2s
= 2 x 5 x 10^-5
= 10 x 10^-5 = 10^-4

pOH = -log [OH-]
= -log 10^-4
= 4

pH = pKW-pOH
= 14-4
=10

18. Bilangan oksidasi Cl dalam Mg(ClO3)2 adalah... A. +5 B. +3 C. -5 D. -3 E. 0


Jawaban: A. +5

Penjelasan:

Mg(ClO3)2 bila diuraikan maka akan menghasilkan ion Mg 2+ dan ion ClO3^-.

Salah satu aturan penentuan biloks menyebutkan bahwa jumlah bilangan oksidasi unsur-unsur dalam suatu poliatom = muatannya

Bilangan oksidasi O umumnya adalah -2.

Bilangan oksidasi Cl + (3× bilangan oksidasi O ) = -1

Bilangan oksidasi Cl + (3 x (-2)) = -1

Bilangan oksidasi Cl + (-6) = -1

Bilangan oksidasi Cl = -1 +6 = +5
