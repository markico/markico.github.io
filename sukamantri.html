1. Bebegig Sukamantri nyaeta salahsahiji rupa kasenian has anu aya di daerah Ciamis. Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah. Sedengkeun ari bebegig sawah mah mangrupa jajalmaan anu rupana pikasieuneun anu ditancebkeun di tengah sawah atawa di sisi balong anu tujuannana pikeun nyingsieunan atawa ngusir manuk.

2. Bebegig Sukamantri mangrupa jalma asli anu make kedok gede anu rupana pikasieuneun jeung buukna cawigwig.

3. Ngamaenkeun Bebegig Sukamantri mah kudu gede tanaga, sabab topengna wae beuratna kurang leuwih 20 kg, eta teh beurat topengna wungkul, can kaitung beurat pakean anu sejenna anu beurat oge.

4. Mimiti medalna kasenian Bebegig Sukamantri teh nyaeta ti Kabupaten Ciamis, anu perenahna nyaeta ti Desa Cempaka, Kacamatan Sukamantri, Kabupaten Ciamis.

5. Topeng Bebegig Sukamantri teh mangrupa gambaran kagagahan Prabu Sampulur.

6. Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.

7. Bebegig Sukamantri teh mangrupakeun bagian tina ritual dina upacara pikeun ngusir roh anu jahat. Kusabab ayeuna warga Ciamis geus ngagem agama Islam, kasenian Bebegig dirobah tina upacara mistik ngusir roh jahat jadi kasenian pikeun ngahibur masarakat Ciamis saperti dina acara agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.

4. Tolong Bantu Jawab Kumaha ari kaayaan bebegig sukamantri kiwari ? ​


Kaayaan bebegig sukamantri kiwari nyaeta kangge hiburan dina acara Agustusan atanapi sunatan.

Penjelasan:

Bebegig Sumantri nyaeta salah sahiji rupa kasenian khas anu aya di daerah Ciamis, Jawa Barat.  Kasenian Bebegig Sukamantri biasa dipintonkeun saperti dina acara Agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.  Bebegig Sukamantri mangrupakeun jalma asli anu make kedok gede anu rupana pikasieuneun jeung buukna cawigwig. Anu kedok eta tea gambaran kagagahan Prabu Simpulur.

Bebegig Sukamantri oge mangrupakeun bagian tina ritual dina upacara pikeun ngusir roh anu jahat. Kusabab ayeuna warga Ciamis Jawa Barat parantos ngagem agama Islam, kasenian bebegig dirobah tina upacara mistik ngusir roh jahat jadi kasenian pikeun ngahibur masyarakat Ciamis saperti dina acara Agustusan, hiburan dina acara sunatan atawa dina acara ulang taun hiji daerah.
