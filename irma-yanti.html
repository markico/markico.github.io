Diketahui: Uang Irma Rp. 500.000

Dibelanjakan sebanyak 75%

Ditanya: Sisa Uang?

Jawab

Uang yang dibelanjakan:

[tex]75\% \times 500.000 = \\ \frac{75}{100} \times 500.000 = \\ \frac{37.500.000}{100} = \\ 375.000[/tex]

Uang sisa

[tex]500.000 - 375.000 = 125.000[/tex]

Jadi, sisa uang Irma Rp. 125.000

15. Jmlh uang ina dan yanti sama dengan 76000 sedangkan uang yanti 12000 lebih banyak dari uang ina.perbandingan uang ina dan yanti adalah....


Pembahasan :

Ina = x, Yanti = y

# Diketahui :

• Jumlah uang ina dan yanti sama dengan 76.000 (x + y = 76.000) ... Pers. 1
• Uang yanti 12000 (y = 12.000)

# Ditanya :
• Perbadingan Uang ina dan Yanti
(x : y = .... : .... )

# Penyelesaian :

• Substitusikan y = 12.000 ke pers. 1

x + y = 76.000
x + 12.000 = 76.000
x = 76.000 - 12.000
x = 64.000

• Perbandingan Uang Ina (x) dan Yanti (y)

x : y
= 64.000 : 12.000
= 16 : 3
