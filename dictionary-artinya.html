4. We will use a dictionary. Apa artinya


artinya kami akan menggunakan kamus

Soal:

We will use a dictionary.

Apa artinya?

Jawab:

Kita akan menggunakan sebuah kamus.

Perkata:

We = kita / kami

Will = akan

Use = menggunakan

A = sebuah

Dictionary = kamus

Rumus yg digunakan adalah Future Tense

(+) S + will + Verb 1 + O

= we + will + use + a dictionary

= saya + akan + menggunakan + sebuah kamus

==== Semoga Membantu =====

Mapel: Bahasa Inggris

Kelas: SMP

Materi: Menerjemahkan

Kode Mapel: 5
