Penjelasan:

Diketahui

v=0,005 m³

Wu=150N

p=1000 kg/m³

g=10m/s²

Ditanya

Fa=..?

Wbf=...?

Jawab

Fa=p.gv

=1000×10×0,005

=50N

Wbf=Wu-Fa

=150-50

=100N

23. PT. Batumega memiliki dua jenis investasi yaitu proyek A senilai Rp 1.000.0000.000,00 diperkirakan menghasilkan 8% per tahun, umur proyek 3 tahun. Proyek B investasi bernilai Rp 500.000.000,00, diperkirakan menghasilkan 1% untuk tahun pertama dan kedua, dan 22% untuk tahun ketiga. Saudara sebagai ahli keuangan diminta menghitung : 1. Berapa rata-rata tingkat bunga selama tiga tahun kedua investasi tersebut ? 2. Investasi mana yang lebih menguntungkan? 3. Berapa return setiap proyek ?


Jawaban:

1. rata-rata tingkat bunga selama tiga tahun kedua proyek adalah 8.12% untuk proyek A dan 8.51% untuk proyek B.

2. B lebih menguntungkan dibandingkan dengan proyek A.

3. proyek A menghasilkan return sebesar Rp 240.000.000,00 dan proyek B menghasilkan return sebesar Rp 220.000.000,00

Penjelasan:

Untuk pertanyaan 1:

Untuk menghitung rata-rata tingkat bunga selama tiga tahun kedua investasi tersebut, kita dapat menggunakan metode geometric mean return.

Proyek A:

hasil per tahun = Rp 1.000.000.000,00 x 8% = Rp 80.000.000,00

geometric mean return = (1 + 8%)^(1/3) - 1 = 8.12%

Proyek B:

hasil per tahun 1 dan 2 = Rp 500.000.000,00 x 1% = Rp 5.000.000,00

hasil per tahun 3 = Rp 500.000.000,00 x 22% = Rp 110.000.000,00

geometric mean return = (1 + 1%)(1 + 1%)(1 + 22%)^(1/3) - 1 = 8.51%

Jadi, rata-rata tingkat bunga selama tiga tahun kedua proyek adalah 8.12% untuk proyek A dan 8.51% untuk proyek B.

Untuk Pertanyaan 2:

Dari hasil rata-rata tingkat bunga yang didapat, proyek B memiliki tingkat bunga yang lebih tinggi dibandingkan dengan proyek A. Oleh karena itu, dapat disimpulkan bahwa proyek B lebih menguntungkan dibandingkan dengan proyek A.

Untuk Pertanyaan 3:

Proyek A:

Return = hasil per tahun x 3 = Rp 80.000.000,00 x 3 = Rp 240.000.000,00

Proyek B:

Return = hasil per tahun 1 dan 2 + hasil per tahun 3 = (Rp 5.000.000,00 x 2) + Rp 110.000.000,00 = Rp 220.000.000,00
