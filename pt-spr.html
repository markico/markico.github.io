4. Tentukan besar SPR +TQS+ PRT+ QSP+RTQ


Caranya
1. buat jari-jari lingkatannya (OP,OQ,OR,OS dan OT)
2. diketahu bahwa sudut-sudut POQ, QOR, ROS, SOT dan POT membentuk sudut 1 lingkaran sehingga ∠POQ+∠QOR+∠ROS+∠SOT+∠POT=360°
3. perhatikan bahwa ∠SPR=½∠SOR, ∠TQS=½∠TOS, ∠PRT=½∠POT, ∠QSP=½∠QOP dan ∠RTQ=½∠QOR
4. hitung
SPR +TQS+ PRT+ QSP+RTQ
= ½∠SOR+½∠TOS+½∠POT+½∠QOP+½∠QOR
= ½(∠SOR+∠TOS+∠POT+∠QOP+∠QOR)
= ½(360°)
= 180°

10. Tentukan besar sudut SPR, sudut PRQ, dan sudut QPR!​


<SPR + <QPR = 180°

3x + x = 180°

4x = 180°

x = 180°÷4 = 45°

<SPR = 3x = 3 x 45° = 135°

<QPR = x = 45°

<PRQ = 180° - <RQP - <QPR = 180° - 80° - 45° = 55°

<SPR = 135°

<PRQ = 55°

<QPR = 45°

Penjelasan dengan langkah-langkah:

• Mencari nilai x,

(3x°+x°) + 80° = 180°

4x° = 180° - 80°

4x° = 100°

x° = 100° : 4

x° = 25°

Jawab :

- Mencari besar sudut SPR,

<SPR = 3x°

= 3(25)°

<SPR = 75°

jadi, besar sudut SPR adalah 75°.

- Mencari besar sudut QPR,

<QPR = x°

= 25°

jadi, besar sudut QPR adalah 25°.

- Mencari besar sudut PRQ,

<PRQ = 180° - ( <QPR + <PQR )

= 180° - (25° + 80°)

= 180° - 105°

<PRQ = 75°

jadi, besar sudut PRQ adalah 75°.
