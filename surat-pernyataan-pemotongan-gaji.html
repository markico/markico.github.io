8. gajih beni 115% dari gaji toni sedangkan gaji meri 90% dari gaji toni jumlah gaji ketiganya adalah 6100,000,00 maka gaji meri adalah


Gaji Toni dimisalkan x = 100/100 x

Gaji Beni = 115/100 x
Gaji Meri = 90/100 x

6.100.000 = 100/100 x + 115/100 x + 90/100 x
6.100.000 = 305/ 100 x
x = 6.100.000 . 100/305
x = 2000.000

maka gaji meri adalah 90/100 . 2000.000 = 1800.000

Gaji Beni = 115/100 x
Gaji Meri = 90/100 x

6.100.000 = 100/100 x + 115/100 x + 90/100 x
6.100.000 = 305/ 100 x
x = 6.100.000 . 100/305
x = 2000.000

maka gaji meri adalah 90/100 . 2000.000 = 1800.000Misal :

115% b + b + 90% b = 6.100.000

305/100 b = 6.100.000
61/20 b = 6.100.000
