A1.V1 = A2.V2

10.2 = 5.V2

V2 = 20/5 = 4 m/s

Persamaan Bernouli

P1 + ρ.g.h1 + 1/2.ρ.v1² = P2 + ρ.g.h2 + 1/2.ρ.v2²

40.10³ + 10³.10.0 + 1/2.10³.2² = P2 + 10³.10.0,6 + 1/2.10³.4²

40.10³ + 2.10³ = P2 + 6.10³ + 8.10³

P2 = 42.10³ - 14.10³

P2 = 28.10³ Pa

P2 = 28 k.Pa

Tekanan pada pipa yang kecil 28 kPa.
