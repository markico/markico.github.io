17. apa itu c++ dan apa itu Ruby?​


Pembahasan :

Apa Itu C++?

C++ adalah bahasa pemrograman yang berorientasi objek, dimana untuk menyelesaikan sebuah masalah C++ melakukannya dengan menjelaskan class-class yang merupakan anak class sebagai abtraksi dari objek-objek fisik.

Apa Itu Ruby ?

Para programmer menyebut Ruby sebagai bahasa yang praktis juga cantik, dan juga artistik. Adapun Ruby itu sendiri merupakan sebuah bahasa yang diciptakan dengan seimbang. Lahir dari buah pikiran Yukihiro “Matz” Matsumoto, ia telah menggabungkan beberapa bagian-bagian terbaik dari beberapa bahasa favorit beliau. Contohnya Perl dengan segala kehebatannya, Phyton yang objektif, Smalltalk, Eiffel, serta Ada dan Lisp, semuanya digabungkan dan menjadi bahasa baru yang seimbang, antara pemrograman imperatif serta pemrograman fungsional.

1. GitHub

GitHub adalah salah satu source code management Git terpopuler. Platform ini digunakan para developer untuk meng-host dan mereview code, mengelola proyek, dan membangun software aplikasi. GitHub memungkinkan para developer di seluruh dunia untuk bisa berkolaborasi.

Shopify memfasilitasi orang untuk membangun website dan portal e-commerce.
