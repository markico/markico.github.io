16. Ane wa komuin desu apaan artinya ?


姉は公務員です 「 あねはこうむいん 」Ane wa koomuin desu: Kakak Perempuan ( Saya ) adalah pegawai negeri (sipil)

Pembahasan

姉は公務員です 「 あねはこうむいん 」Ane wa koomuin desu: Kakak Perempuan ( Saya ) adalah pegawai negeri (sipil)

Ane 姉: Kakak perempuan

wa : Partikel

Koomuin: Pegawai negeri ( sipil )

desu: tidak memiliki arti berfungsi sebagai penghalus kalimat

Contoh kalimat Positif

Pola kalimat

Kb ( Orang ) + は ( wa ) + kb ( Jenis pekerjaan ) + です ( Desu )

私は医者です「わたしはいしゃです」Watashi wa Isha desu

Saya adalah Dokter

Watashi: Saya

Boku: Aku Ore : Gue

Wa: Partikel

Isha: Dokter

家族 ( Kazoku ) : Keluarga

父 ( Chichi ) : Ayah母 ( Haha ) : Ibu祖父 ( Sofu ) : Kakek祖母 ( Sobo ) : Nenek叔父 ( Oji ) : Paman叔母 ( Oba ) : Bibi

私の兄弟 - Watashi No Kyoodai: Saudara saya

お姉さん ( Onee san ) : Kakak perempuanお兄さん ( Onii-san ) : Kakak laki-laki妹 ( Imouto ) : Adik perempuan弟 ( Otouto ) : Adik laki-laki
