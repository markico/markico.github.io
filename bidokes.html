-biloks atau bilangan oksidasi didefinisikan sebagai jumlah muatan negatif dan positif dalam atom, yang secara tidak langsung menandakan jumlah elektron yang telah diterima atau diserahkan. Atom yang menerima elektron akan bertanda negatif, atom yang melepaskan elektron bertanda positif

makasiiBilangan oksidasi ( biloks ) didefinisikan sebagai jumlah muatan negatif dan positif dalam atom, yang secara tidak langsung menandakan jumlah elektron yang telah diterima atau diserahkan. Atom yang menerima elektron akan bertanda negatif, atom yang melepaskan elektron bertanda positif.

Redoks adalah istilah yang menjelaskan berubahnya bilangan oksidasi (keadaan oksidasi) atom-atom dalam sebuah reaksi kimia. Hal ini dapat berupa proses redoks yang sederhana seperti oksidasi karbon yang menghasilkan karbon dioksida, atau reduksi karbon oleh hidrogen menghasilkan metana (CH4), ataupun ia dapat berupa proses yang kompleks seperti oksidasi gula pada tubuh manusia melalui rentetan transfer elektron yang rumit.

11. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4

12. tentukan biloks dari atom dan senyawa - biloks fe pada senyawa fe2(SO4)3- biloks cr pada senyawa k2crO4- biloks s pada senyawa H2SO4\- biloks Cl pada senyawa HCIO4


Biloks dari senyawa berikut :

a. Biloks Fe dari Fe₂(SO₄)₃

Biloks Fe₂(SO₄)₃ = (2 × Biloks Fe) + (3 × Biloks S) + (12 × Biloks O)

0 = (2 × Biloks Fe) + (3 × 6)) + (12 × (-2))

0 = (2 × Biloks Fe) + 18 + (-24)

0 = (2 × Biloks Fe) - 6

2 × Biloks Fe = + 6

Biloks Fe = +6/2

Biloks Fe = +3

b. Biloks Cr dari K₂CrO₄

Biloks K₂CrO₄ = (2 × Biloks K) + Biloks Cr + (4 × Biloks O)

0 = (2 × 1) + Biloks Cr + (4 × (-2))

0 = 2 + Biloks Cr - 8

0 = Biloks Cr - 6

Biloks Cr = +6

c. Biloks S dari H₂SO₄

Biloks H₂SO₄ = (2 × Biloks H) + Biloks S + (4 × Biloks O)

0 = (2 × 1) + Biloks S + (4 × (-2))

0 = 2 + Biloks S - 8

0 = Biloks S - 6

Biloks S = +6

d. Biloks Cl dari HClO₄

Biloks HClO₄ = Biloks H + Biloks Cl + (4 × Biloks O)

0 = 1 + Biloks Cl + (4 × (-2))

0 = 1 + Biloks Cl - 8

0 = Biloks Cl - 7

Biloks Cl = +7

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks N + (+4) = +1

biloks N = (+1) - 4

biloks N = -3

29. biloks dari 2nah2po3 ?biloks dari 2ph3 ?​


Jawaban:

Biloks dari 2NaH2PO3 (natrium hipofosfit) adalah:

- Biloks natrium (Na) adalah +1 karena ia adalah logam alkali.

- Biloks hidrogen (H) adalah +1 karena ia umumnya memiliki bilangan oksidasi positif ketika terikat dengan nonlogam.

- Biloks fosfor (P) adalah +1 karena ada dua hidrogen yang terikat ke fosfor dan masing-masing hidrogen memiliki bilangan oksidasi +1. Namun, ada dua atom oksigen di sebelah fosfor, yang masing-masing memiliki bilangan oksidasi -2. Karena total muatan dari senyawa harus netral, maka bilangan oksidasi dari fosfor harus +1 untuk menetralkan total bilangan oksidasi hidrogen dan oksigen. Oleh karena itu, bilangan oksidasi fosfor dalam 2NaH2PO3 adalah +1.

Jadi, biloks dari 2NaH2PO3 adalah: Na(+1) + 2H(+1) + P(+1) + 3O(-2) = 0.

Sedangkan, biloks dari 2PH3 (difosfina) adalah:

Biloks fosfor (P) adalah -3 karena ia terikat dengan tiga atom hidrogen (H), dan masing-masing hidrogen memiliki bilangan oksidasi +1. Oleh karena itu, untuk menetralkan total bilangan oksidasi hidrogen dan fosfor harus -3.

Jadi, biloks dari 2PH3 adalah: 2P(-3) + 6H(+1) = 0.
