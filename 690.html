7n-3 = 690

7n = 690-3

7n = 693

n = 693/7

n = 99

Jawaban:

n = 99

Penjelasan dengan langkah-langkah:

Un = 7n - 3

Un = 690

=> 690 = 7n - 3

=> 7n = 690 + 3

=> 7n = 693

=> n = 693/7

=> n = 99
