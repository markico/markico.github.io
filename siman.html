Jawaban:

Diketahui:

I₁ = I

R₁ = 3

R₂ = 9

Ditanya: I₂

Maka:

\frac{I}{I_{2} } =\frac{15^{2}}{3^{2}}\\\\ I_{2} = \frac{9I}{225} = \frac{I}{25}

2. Hubungan antara Intensitas dan Amplitude: \frac{I_{1} }{I_{2} } = \frac{A_{1}^2 }{A_{2}^2}

Diketahui:

I₁ = I

I₂ = 1/25 I (dari sebelumnya)

A₁ = A

Ditanya: A₂

Maka:

\frac{I}{\frac{I}{9} } } =\frac{A^{2}}{A_{2} ^{2}}\\\\ A_{2}^2 = \frac{A^2}{1/25} \\ \\ A_{2} = \frac{1A}{5}
