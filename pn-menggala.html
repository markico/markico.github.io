17. pola bilangan persegi panjang jika Pn = 552 suku keberapa Pn ?


Matematika
Pola Bilangan VIII SMP

Pembahasan :
Pola Persegi Panjang

Pn = 552
n(n + 1) = 552
n² + n = 552
n² + n - 552 = 0
(n - 23)(n + 24) = 0
n = 23 —» Pada suku ke-23

22. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722
