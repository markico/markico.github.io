13. jika suatu suku pada pola bilangan persegi panjang pn=1722 maka suku ke berapakah pn?​


Jawaban:

Pn = 1722 adalah suku ke 41

Penjelasan dengan langkah-langkah:

Pola bilangan persegi panjang

= n x (n+1)

Pn = n x (n + 1)

1722 = n² + n

n² + n - 1722 = 0

(n + 42) (n - 41) = 0

n = -42 atau n = 41

Jadi, nilai n ambil yang positif yaitu n = 41

Pn = n x (n + 1)

P41 = 41 x (41 + 1)

= 41 x 42

= 1722


14. Menghitung pn metode pendapatan


Ekonomi

Pendapatan Nasional

⠀⠀⠀⠀⠀⠀⠀⠀⠀

⠀⠀⠀⠀⠀⠀⠀⠀⠀

Jawaban:

r (Rent/ sewa) + w (Wage/ upah) + i (Interest/ bunga) + p (Profit/ laba)

⠀⠀⠀⠀⠀⠀⠀⠀⠀

Pembahasan:

Pendapatan nasional merupakan jumlah pendapatan yang diterima oleh Rumah Tangga Keluarga (RTK) pada suatu negara dalam jangka waktu satu periode atau satu tahun.

⠀⠀⠀⠀⠀⠀⠀⠀⠀

Dengan adanya pendapatan nasional, suatu negara dapat mengetahui detail sektor-sektor ekonomi, perkembangan ekonomi serta membandingkan perekonomian suatu negara dengan negara lain.

⠀⠀⠀⠀⠀⠀⠀⠀⠀

Untuk mencari pendapatan nasional, terdapat 3 metode yaitu:

1. Metode pendekatan produksi

⠀ Rumus:

⠀ (P1 x Q1) + (P2 x Q2) + …. (Pn x Qn)

⠀⠀⠀⠀⠀⠀⠀⠀⠀

⠀ Keterangan :

⠀ P1 = Harga barang ke-1

⠀ Pn = Harga barang ke-n

⠀ Q1 = Jenis barang ke-1

⠀ Qn = Jenis barang ke-n

⠀⠀⠀⠀⠀⠀⠀⠀⠀

2. Metode pendekatan pendapatan

⠀ Rumus:

⠀ r + w + i + p

⠀⠀⠀⠀⠀⠀⠀⠀⠀

⠀ Keterangan :

⠀ r = Pendapatan dari sewa

⠀ w = Pendapatan dari upah

⠀ i = Pendapatan dari bunga

⠀ p = Pendapatan dari keuntungan perusahaan/ usaha

⠀⠀⠀⠀⠀⠀⠀ ⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀

3. Metode pendekatan pengeluaran

⠀ Rumus:

⠀ C + I + G + ( X – M )

⠀⠀⠀⠀⠀⠀⠀⠀⠀

⠀ Keterangan :

⠀ C = Consumption/ Konsumsi

⠀ I = Investment/ Investasi

⠀ G = Government Expenditure/ Pengeluaran pemerintah

⠀ X = Ekspor

⠀ M = Impor
