● Kode kategorisasi: 6.2.9

D. 98 kelereng

Jumlah Kelereng

= jumlah perbandingan/selisih perbandingan x 28

= (9+5)/(9-5) x 28

= 14/4 x 28

= 14 x 7

= 98 kelereng
