2. Diketahui uang agung:uang Anang=5:4 dan uang Anang: uang catur=3:4 jika jumlah uang agung,anang,dan catur Rp. 43.000,maka uang Anang adalah


Perbandingan Agung : Anang = 5 : 4

Perbandingan Anang : Catur = 3 : 4

Samakan dulu semua perbandingannya maka perbandingannya menjadi Agung : Anang : Catur = (5×3) : 12 : (4×4) = 15 : 12 : 16.

Jadi uang Anang adalah 12/(15+12+16) × 43.000 = 12/43 × 43.000 = 12 × 1.000 = 12.000. Jadi uang Anang adalah Rp.12.000,00

24. Perbandingan uang Anang dan Basar adalah 2: 1. Perbandingan uang Anang dan Cecep adalah 4:1. Jika jumlah uang Anang dan Basar Rp6.000.000, tentukan jumlah total uang mereka!


seperti Pada gambar yah Pembahasan
A : uang anang
B : uang Basar
C : uang cecep

A : B = 2 : 1
Bearti
A = 2B

A : C = 4 : 1
Bearti
A = 4C
Atau C = 1/4 A

A + B = Rp. 6.000.000
2B + B = Rp. 6.000.000
3B = Rp. 6.000.000
B = Rp. 6.000.000 / 3 = Rp. 2.000.000

A = 2B = 2 x Rp. 2.000.000
A = Rp. 4.000.000

C = 1/4 A = 1/4 x Rp. 4.0000.000
C = Rp. 1.000.000

Jadi
Uang Anang Rp. 4.000.000
Uang Basar Rp. 2.000.000
Uang Cecep Rp. 1.000.000

Total uang mereka = Rp. 4.000.000 + Rp. 2.000.000 + Rp. 1.000.000 = Rp. 7.000.000

25. pada saat ini umur adi 3 kali umur anang, jika umur adi 8 tahun bu lebih tua dari anang, maka umur anang adalahtolong jawab pake cara yakk​


Jawaban:

8 tahun

Penjelasan dengan langkah-langkah:

Misalkan x adalah umur Anang saat ini. Berdasarkan informasi soal, kita bisa menuliskan persamaan berikut:

Umur Adi = 3 x Umur Anang

Umur Adi - Umur Anang = 8

Kita bisa menggunakan persamaan kedua untuk menyelesaikan persamaan pertama dengan mengganti Umur Adi dengan 3 x Umur Anang - 8 (karena Umur Adi - Umur Anang = 8):

3 x Umur Anang - 8 - Umur Anang = 8

2 x Umur Anang = 16

Umur Anang = 8

Jadi, saat ini umur Anang adalah 8 tahun.
