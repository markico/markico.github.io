Mapel : Matematika
Tentang : Skala
Pertanyaan yg diajukan : 1
Pembahasan :

Diketahui :
- Skala (S) : 1 : 3.300.000
- Jarak Pada Peta (JP) : 3 cm

Ditanya :
- Jarak Sebenarnya (JS)

Cara :

[tex]js = \frac{jp}{s} \\ \\ js = \frac{3}{ \frac{1}{3.300.000} } [/tex]

[tex]js = 3 \times \frac{3.300.000}{1} \\ \\ js = 9.900.000 \: cm[/tex]

[tex]js = \frac{9.900.000}{100.000} \\ \\ js = 99 \: km [/tex]

Jawaban : 99 km
