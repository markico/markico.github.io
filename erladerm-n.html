4. Kuis [+50]: n % = n / ___ n ‰ = n / ___ n² = n × ___ n³ = n × ___ × ___


Penjelasan dengan langkah-langkah:

Perlu diingat :

Persen (%) adalah per seratus

Permil (‰) adalah perseribu

Kuarat (²) adalah perkalian antara dua angka yg sama

Kubik (³) adalah perkalian tiga kali dgn angka yg sama

Maka,

n% = n/100 (persen = per 100)

n‰ = n/1000 (permil = per 1000)

n² = n x n (kuadrat)

n³ = n x n x n (kubik)

1. n % Artinya adalah n persen.

Apa itu persen?

Persen itu Perseratus.

Jadi semua pecahan yang perseratus disebutnya persen.

n % = n per seratus = n / 100

2. n ‰ Artinya adalah n permil.

Apa itu permil?

Permil itu Perseribu.

Jadi semua pecahan yang perseribu disebutnya permil.

n ‰ = n per seribu = n / 1000

3. n^2

Artinya adalah variabel n dikalikan lagi dengan variabel n sebanyak satu kali.

n^2 = n x n

4. n^3

Artinya adalah variabel n dikalikan lagi dengan variabel n sebanyak dua kali.

n^3 = n x n x n
