Luas lingkaran

Keliling lingkaran = 2 π

2 πr = 2 π

r = 1


Jadi

Luas lingkaran

= πr²

= π(1)²

= π

25. nilai sin phi/3 + cos phi/4 + tan phi/3.. berapa??


[tex]Answer[/tex]

Sin π/3 + Cos π/4 + Tan π/3

Nilai π dalam Materi Trigonometri yaitu 180. Jadi, substitusi nilai π tersebut!

Sin 180/3 + Cos 180/4 + Tan 180/3

Sin 60 + Cos 45 + Tan 60

Dari sini, kita harus tahu sudut - sudut istimewa. Karena sangat dibutuhkan dalam pelajaran trigonometri.

~ Sinus (Sin) ~

Sin 0 = 0

Sin 30 = ½

Sin 45 = ½ √2

Sin 60 = ½ √3

Sin 90 = 1

~ Cosinus ~

(Kebalikan Dari Sin)

Cos 0 = 1

Cos 30 = ½ √3

Cos 45 = ½√2

Cos 60 = ½

Cos 90 = 0

~ Tangen ~

Tan 0 = 0

Tan 30 = ⅓ √3

Tan 45 = 1

Tan 60 = √3

Tan 90 = (Tak Terdefenisi)

Jadi,

=> Sin 60 + Cos 45 + Tan 60

=> ½ √3 + ½ √2 + √3

=> 3/2 √3 + ½ √2
