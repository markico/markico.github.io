D. x² - 3x - 1 = 0

Penjelasan dengan langkah-langkah:

Diketahui a dan b merupakan akar dari persamaan kuadrat 3x² - 5x + 1 = 0. Persamaan kuadrat baru yang mempunyai akar akar (3a - 1) dan (3b - 1) adalah...

3x² - 5x + 1 = 0

a + b = -(- 5)/3

a + b = 5/3

a.b = 1/3

Persamaan kuadrat baru: (dengan akar x1 dan x2 baru)

x² - (x1 + x2)x + (x1. x2) = 0

Persamaan kuadrat baru dengan akar (3a - 1) dan (3b - 1)

x² - (3a - 1 + 3b - 1)x + (3a - 1)(3b - 1) = 0

x² - (3a + 3b - 2)x + (9ab - 3a - 3b + 1) = 0

x² - ( 3(a + b) - 2)x + ( 9(ab) - 3 (a + b) + 1) = 0

x² - (3 (5/3) - 2)x + ( 9(1/3) - 3(5/3) + 1) = 0

x² - (5 - 2) x + (3 - 5 + 1) = 0

x² - 3x - 1 = 0

360°/24 jam = 15°. Jadi setiap rotasi bumi sebanyak 15 derajat berarti menghabiskan waktu sebanyak 1 jam.

Maka perbedaan waktu kota x dan Greenwich adalah 60/15 = 4 jam

BB = Dikurang

BT = ditambah

= 11 - 4

= 07.00

19. Coba saudara diskusikan apa yang dsaudara ketahui tentang integer programming​


Integer programming adalah program linier yang menyatakan variabel memiliki nilai berupa angka baik negatif atau positif.

Pembahasan

Programming merupakan sebuah proses yang dilakukan untuk memelihara kode-kode program di dalam komputer.

Pemrograman memiliki beberapa macam, sebagai berikut:

Pemrograman web merupakan pemrograman yang digunakan untuk mendesain situs web dengan komponen utamanya yaitu HTML dan CSS. Pemrograman sistem operasi merupakan pemrograman yang digunakan untuk membuat sistem operasi komputer. Pemrograman aplikasi merupakan pemrograman yang digunakan untuk membuat suatu aplikasi baru atau pengembangan dari yang sudah ada dengan bantuan bahasa pemgrograman dan editor teks. Pemrograman game merupakan pemrograman yang digunakan untuk membuat aplikasi game dengan bantuan bahasa pemgrograman.

Macam-macam bahasa pemrograman, yaitu:

Python  

PHP

Java

Delphi

C++

C#

JavaScript
