17. Pak wardoyo menabung di koperasi sebesar rp. 750.000 dan memoeroleh bunga 12% per tahun. Setelah beberapa bulan tabungan pak wsrdoyo menjadi rp. 810.000 berapa lama pak wardoyo telah menabung?


Besar nya bunga dalam suatu pinjaman di nyatakan dalam % ( persen ) dan disebut dengan suku bunga. Suku bunga per tahun adalah perbandingan antara bunga yg diperoleh dengan modal awal per tahun.

Suku bunga = [tex] \frac {bunga yg diperoleh} {modal awal}[/tex] x 100%

Sedangkan menghitung besar pinjaman untuk beberapa periode adalah sebagai berikut:  

Mn = Mo ( 1 + nb )

dimana :

Mn = modal akhir

Mo = modal awal  

n = jumlah periode  

b = suku bunga dalam % ( persen ) per periode

Pembahasan  

Mo = Rp 750.000

Mn = Rp 810.000

b = 12%

Suku bunga tabungan yg diberikan bank per tahun adalah :

Mn = Mo ( 1 + nb )

Rp 810.000 = Rp 750.000 ( 1 + n x 0,12 )

Rp 810.000 = Rp 750.000 ( 1 + 0,12n )

Rp 810.000 = Rp 750.000 ( 1 + [tex] \frac {12} {100}[/tex]n )

> gunakan sifat distributif perkalian

Rp 810.000 = Rp 750.000 + ( Rp 750.000 x [tex] \frac {12} {100}[/tex]n )

Rp 810.000 = Rp 750.000 + ( Rp 750.000 : 100 x [tex] \frac {12} {100 : 100}[/tex]n )

Rp 810.000 = Rp 750.000 + ( Rp 7.500 x 12n )

Rp 810.000 = Rp 750.000 + Rp 90.000n

> kurangkan masing - masing ruas dengan Rp 750.000  

Rp 810.000 - Rp 750.000 = Rp 750.000 + Rp 90.000n - Rp 750.000  

Rp 60.000 = Rp 90.000n

> bagilah masing - masing ruas dengan Rp 90.000

Rp 60.000 : Rp 90.000 = Rp 90.000n : Rp 90.000

n = [tex]frac {60.000} {90.000}[/tex] tahun

n = [tex]frac {60.000 : 30.000} {90.000 : 30.000}[/tex] tahun

n = [tex]\frac {2} {3}[/tex] tahun

n = [tex]\frac {2} {3}[/tex] x 12 bulan

n = 8 bulan

Sehingga lama Pak Wardoyo menabung adalah 8 bulan.
