9. Jumlah uang astri dan bela adalah rp.500.000. perbandingan uang astri dan bela 2:3. jika uang astri rp.200.000 maka besar uang bela adalah


Jawab:

uang astri adalah A dan uang bela adalah B

Diketahui

A : B = 2x : 3x

A + B = 500.000

A = 200.000

Ditanyakan

B = ?

Penyelesaian

A + B = 500.000

2x + 3x = 500.000

5x = 500.000

x = 500.000/5

x = 100.000

B = 3x

  = 3(100.000)

  = 300.000

Jadi uang Bela sebesar Rp300.000,00

17. Jumlah uang Alma dan Astri besarnya Rp. 100,000,. Perbandingan uang Alma dan Astri adalah 3 : 2, tentukan besaran uang mereka masing -masing


Jawaban:

Uang Alma = Rp 60.000

Uang Astri = Rp 40.000

Penjelasan dengan langkah-langkah:

Diketahui :

Jumlah uang Alma dan Astri = Rp 100.000

Perbandingan uang Alma dan Astri = 3 : 2

Ditanya :

Besar uang mereka masing masing?

Dijawab :

Uang Alma

[tex]\sf = \frac{3}{3 + 2} \times Rp 100.000[/tex]

[tex]\sf = \frac{3}{5} \times Rp 100.000[/tex]

[tex]\sf = \frac{300.000}{5} [/tex]

= Rp 60.000

Uang Astri

[tex]\sf = \frac{2}{3 + 2} \times Rp 100.000[/tex]

[tex]\sf = \frac{2}{5} \times Rp 100.000[/tex]

[tex] = \sf \frac{200.000}{5} [/tex]

= Rp 40.000

28. Perbandingan umur Firda dan umur Astri adalah 3 : 5. Jika umur Astri 20 tahun lebih tua dari umur Firda, maka jumlah umur mereka adalah..


20/2=10
3x10=30 umur firda
5x10=50 umur astri
30+50=80th
jadi jumlah umur mereka adalah 80thF : A = 3 : 5
   5F = 3A

A = F + 20

Masukkan nilai A :
5F = 3A
5F = 3(F + 20)
5F = 3F + 60
2F = 60
  F = 30 tahun

Masukkan nilai F :
A = F + 20
A = 30 + 20
A = 50 tahun

Sehingga, jumlah umur mereka adalah :
F + A = 30 tahun + 50 tahun
F + A = 80 tahun
