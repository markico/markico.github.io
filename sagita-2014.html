4. Akar dari persamaan (x – 2014)2 + (x – 4029)2 – (x + 1)2 = 0 adalah .... A. (2014, 10074) B. (2013, 10073) C. (-2014, 10074) D. (-2014, 2014) E. (-2014, -10004)


(x - 2014)² + [(x - 4029)² - (x + 1)²] = 0
(x - 2014)² + [(x - 4029) + (x + 1)] [(x - 4029) - (x + 1)] = 0
(x - 2014)² + (x - 4029 + x + 1) (x - 4029 - x -1) = 0
(x - 2014)(x - 2014) + (2x - 4028) (-4030) = 0
(x - 2014)(x - 2014) + 2(-4030) (x - 2014) = 0
(x - 2014)(x - 2014) - 8060 (x - 2014) = 0
(x - 2014) (x - 2014 - 8060) = 0
(x - 2014)(x - 10074) = 0
x₁ = 2014
x₂ = 10074

jawaban (A)

26. Sagita membelanjakan 55% dari uangnya untuk membeli buku.1/5 bagian untuk membeli alat tulis, dan Rp5.000,00 diberikan adiknya .Sagita masih mempunyai uang Rp 15.000,00. Uang Sagita mula mula adalah


Uang Sagita mula - mula adalah Rp 80.000,00

Hasil tersebut diperoleh dengan pengurangan dan pembagian pecahan.

Pembahasan

Diketahui:

Sagita membelanjakan 55% uangnya untuk membeli buku, 1/5 bagian untuk membeli alat tulis, dan memberikan Rp 5.000,00 kepada adiknya.

Sagita masih mempunyai uang Rp 15.000,00

Ditanya :

Uang Sagita mula - mula ?

Dijawab:

uang Sagita mula - mula = a = 100/100

a - 55/100 a - 1/5 a - 5.000 = 15.000

(100/100 - 55/100 - 20/100)a = 15.000 + 5.000

(100 - 55 - 20 / 100) a = 20.000

25/100 a = 20.000

25 a = 20.000 x 100

25 a = 2.000.000

a = 2.000.000 / 25

a = 80.000

Uang Sagita mula - mula = Rp 80.000,00
