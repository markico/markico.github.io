PEMBAHASAN

Langkah-Langkahnya:

1. Mencari SHU jasa modal Pak Vincenzo

2. Mencari SHU jasa usaha Pak Vincenzo

3. Mencari total SHU Pak Vincenzo

A. SHU JASA MODAL PAK VINCENZO

= (Simpanan Pokok Pak Vincenzo + Simpanan Wajib Pak Vincenzo) / (Simpanan Pokok Koperasi + Simpanan Wajib Koperasi) x %SHU Jasa Modal x SHU Koperasi

= (2.500.000 + 15.000.000) / (250.000.000 + 150.000.000) x 30% x 120.000.000

= 17.500.000 / 400.000.000 x 36.000.000

= Rp1.575.000

= 17.500.000 / 450.000.000 x 30% x 120.000.000

= Rp1.400.000

C. TOTAL SHU PAK VINCENZO

= SHU Jasa Modal + SHU Jasa Usaha

= 1.575.000 + 1.400.000

= Rp2.975.000
