A = Terpilih ketua laki-laki

n(A) => Ketua = 6,Sekretaris = 9(sisa),Bendahara = 8(sisa)

n(A) = 6 x 9 x 8 = 432

B = Terpilih sekretaris wanita

n(B) => Sekretaris = 4,Ketua = 9(sisa),Bendahara = 8(sisa)

n(B) = 4 x 9 x 8 = 288

A n B = Terpilih ketua laki-laki dan sekretaris wanita

n(A n B) = Ketua = 6,Sekretaris = 4,Bendahara = 8(sisa)

n(A n B) = 6 x 4 x 8 = 192

P(AUB) = P(A) + P(B) - P(A n B)

= 437/720 + 288/720 - 192/720

= 528/720 ........ atas bawah bagi 48

= 11/15
