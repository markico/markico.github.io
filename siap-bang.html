Keterangan :

Np = Jumlah lilitan (kumparan) primer

Ns = Jumlah lilitan (kumparan) sekunder

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Dik= Vp= 110 Volt

        Vs= ? = 88,000

        Np=800 L

        Ns=?= 1 L

        Ip=10A

        Is=?

Dit= hitunglah Vs, Ns, Is

Jawaban;

Mencari Vs

[tex]\frac{Np}{Ns}[/tex]=[tex]\frac{Vp}{Vs}[/tex]

=vs=[tex]\frac{800x110}{?}[/tex]=88000 vs

mencari Ns

[tex]\frac{Np}{Ns}[/tex]=[tex]\frac{Vp}{Vs}[/tex]

=Ns=[tex]\frac{800 x 110}{88,000 x ?}[/tex]=1 L

mencari rumus Is

[tex]\frac{Vp}{Vs}[/tex]=[tex]\frac{Ip}{Is}[/tex]

= Is= [tex]\frac{110 x 10 A }{88,000 x ?}[/tex]=0.0125 A

jadi di Vs, Ns dan Is adalah

Vs=88,000 V

Ns= 1L

Is=0.0125 A

Penjelasan:

Ciri-ciri transformator step up :

1. Ns > Np (jumlah lilitan sekunder lebih banyak dibandingkan jumlah lilitan primer

2. Vs > Vp (tegangan sekunder lebih tinggi daripada tegangan primer)

3. Is < Ip (Kuat arus listrik sekunder lebih kecil daripada kuat arus primer)

Ciri-ciri transformator step up :

1.Ns < Np (jumlah lilitan sekunder lebih sedikit dibandingkan jumlah lilitan primer

2.Vs < Vp (tegangan sekunder lebih rendah daripada tegangan primer)

3. Is > Ip (Kuat arus listrik sekunder lebih besar daripada kuat arus primer)

f(1) = 6                       dan f(-1) = -4

f(x) = h(x).(x+1)(x-1) + ax + b
f(1) = a(1) + b
6 = a + b

f(-1) = a(-1) + b
-4 = - a + b

eliminasi  a = 5, b = 1 jadi sisa 5x + 1Misal :
f(x) : x^2 - 1 hasilnya H(x) dan sisa ax + b
f(x) = (x^2 - 1) H(x) + ax + b
f(x) = (x - 1)(x + 1) H(x) + ax + b
f(1) = (1 - 1)(1 + 1) H(1) + a(1) + b => 6 = a + b
f(-1) = (-1 - 1)(-1 + 1) H(-1) + a(-1) + b => -4 = -a + b
