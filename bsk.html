23. bantuin dong buat bsk


Dik:
Sudut BCD = 90 derajat
Sudut BED = (15n + 20) derajat
Sudut BAD = (9m - 10) derajat

Dit: m + n?

Jawab:
Sudut BAD = Sudut BCD
(9m - 10) derajat = 90 derajat
9m - 10 = 90
9m = 90 + 10
9m = 100
m = 100/9
m = 11 1/9 derajat

Sudut BED = Sudut BCD
(15n + 20) derajat = 90 derajat
15n = 90 + 20
15n = 110
n = 110/15
n = 22/3

m + n
= 11 1/9 + 22/3
= 100/9 + 22/3
= 100/9 + 66/9
= 166/9
= 18 4/9
= dibulatkan menjadi 18 (C)
