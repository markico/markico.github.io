15. Diberikan implikasi "jika heri wibowo bisa log on ke server maka heri wibowo mempunyai pasword yang sah" . tentukan negasi,konvers, invers dan kontraposisi dari proposisi majemuk tersebut !


Jawab:

Negasi:

Alternatif 1: "tidak benar bahwa jika heri wibowo bisa log on ke server maka heri wibowo mempunyai password yang sah"Alternatif 2: "heri wibowo bisa log on ke server dan heri wibowo tidak mempunyai password yang sah"

Konvers:

"jika heri wibowo mempunyai password yang sah, maka heri wibowo bisa log on ke server"

Invers:

"jika heri wibowo tidak bisa log on ke server, maka heri wibowo tidak mempunyai password yang sah"

Kontraposisi:

"jika heri wibowo tidak mempunyai password yang sah, maka heri wibowo tidak bisa log on ke server"

Penjelasan dengan langkah-langkah:

Implikasi: "jika heri wibowo bisa log on ke server maka heri wibowo mempunyai password yang sah" dapat dinotasikan secara simbolik dengan:

p ⇒ q

dengan:

p : heri wibowo bisa log on ke server

q : heri wibowo mempunyai password yang sah

NEGASI IMPLIKASI

Tanpa diolah lebih lanjut, negasi dari p ⇒ q adalah ~(p ⇒ q).

Jadi, negasi dari pernyataaan di atas adalah:

"tidak benar bahwa jika heri wibowo bisa log on ke server maka heri wibowo mempunyai password yang sah"

Namun, jika ingin diolah lebih lanjut, maka diperoleh seperti berikut ini.

Menurut hukum implikasi, (p ⇒ q) ≡ (~p ∨ q).

Maka, negasi dari p ⇒ q adalah ~(~p ∨ q).

Sesuai hukum DeMorgan:

~(~p ∨ q) ≡ ~~p ∧ ~q ≡ p ∧ ~q

Jadi, negasi dari pernyataan di atas adalah:

"heri wibowo bisa log on ke server dan heri wibowo tidak mempunyai password yang sah"

KONVERS IMPLIKASI

Konvers dari p ⇒ q adalah q ⇒ p.

Maka, konvers dari pernyataan di atas adalah:

"jika heri wibowo mempunyai password yang sah, maka heri wibowo bisa log on ke server"

INVERS IMPLIKASI

Invers dari p ⇒ q adalah ~p ⇒ ~q.

Maka, invers dari pernyataan di atas adalah:

"jika heri wibowo tidak bisa log on ke server, maka heri wibowo tidak mempunyai password yang sah"

KONTRAPOSISI IMPLIKASI

Kontraposisi dari p ⇒ q adalah ~q ⇒ ~p.

Maka, kontraposisi dari pernyataan di atas adalah:

"jika heri wibowo tidak mempunyai password yang sah, maka heri wibowo tidak bisa log on ke server"
