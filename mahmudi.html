4. Rerata nilai ulangan Kiko, Leli, Mahmud, Nala, dan Oni adalah 85. Rerata ulangan Kiko, Leli, dan Mahmud adalah 80, sedangkan rerata ulangan Mahmud, Nala, dan Oni adalah 88. Berapakah nilai ulangan dari Mahmud?


Dimisalkan :

K = nilai ulangan Kiko

L = nilai ulangan Leli

M = nilai ulangan Mahmud

N = nilai ulangan Nala

O = nilai ulangan Oni

Rerata nilai ulangan Kiko, Leli, Mahmud, Nala, dan Oni adalah 85.

==> (K + L + M + N + O) ÷ 5 = 85

K + L + M + N + O = 85 x 5 = 425 --> pers. (i)

Rerata ulangan Kiko, Leli, dan Mahmud adalah 80.

==> (K + L + M) ÷ 3 = 80

K + L + M = 80 x 3 = 240

(K + L) = 240 - M --> pers. (ii)

Rerata ulangan Mahmud, Nala, dan Oni adalah 88.

==> (M + N + O) ÷ 3 = 88

M + N + O = 88 x 3 = 264

(N + O) = 264 - M --> pers. (iii)

Substitusikan pers. (ii) & pers. (iii) ke pers. (i). Maka didapatkan :

(K + L) + M + (N + O) = 425

(240 - M) + M + (264 - M) = 425

504 - M = 425

M = 504 - 425 = 79

Maka nilai ulangan Mahmud adalah 79

26. Mahmud ingin membuat kerangka balok dari kawat dengan panjang 6cm lebar 2cm tinggi 4cm Mahmud mempunyai kawat sepanjang 50 cm berapa sisa kawat Mahmud ​


2) Diketahui ukuran kerangka balok :

p = 6 cm

l = 2 cm

t = 4 cm

Panjang persediaan kawat = 50 cm  

Ditanyakan :

Sisa panjang kawat = ..... cm  

Jawab :

Panjang kawat untuk membuat kerangka balok

= 4 x (p + l + t)

= 4 x (6 cm + 2 cm + 4 cm)

= 4 x 12 cm

= 48 cm

sisa kawat = persediaan kawat- panjang kawat membuat balok

50 cm - 48 cm = 2 cm
