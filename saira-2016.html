4. Hasil dari [tex]\sqrt{2017+2016\sqrt{2017+2016\sqrt{2017+2016\sqrt{...} } } }[/tex] adalah...


Penjelasan dengan langkah-langkah:

Misal

√(2017 + 2016(√(2017 + 2016(√(2017 + 2016(...)))) = x

dikuadratkan

2017 + 2016x = x²

x² - 2016x - 2017 = 0

(x - 2017) . (x + 1) = 0

x + 1 = 0

x = -1 (tidak memenuhi karena negatif)

x - 2017 = 0

x = 2017 (memenuhi karena positif)

Maka

√(2017 + 2016(√(2017 + 2016(√... ))) = 2017

21. f(216) = 3,dan f (2016) = 4 nilai f(2 2016)+ f(5 2016) adalah


Kelas        : VIII SMP
Pelajaran  : Matematika
Kategori    : Relasi dan Fungsi
Kata kunci : fungsi, bilangan bulat, banyak angka

Penjelasan :

contoh : 

f(216) = 3 
bilangan 216 yang terdiri dari 3 digit (angka)

f(2.016) = 4 
bilangan 2016 yang terdiri dari 4 digit (angka)

Perintah soal :

banyaknya angka (digit) dari x yang meruapakan nilai f (2²⁰¹⁶) + f (5²⁰¹⁶) adalah ...

Pembahasan :

Soal diatas bukan mencari hasil fungsi melainkan menghitung banyak digit (angka) pada fungsi tersebut

Untuk menjawab soal tersebut kita menggunakan pendekatan logaritma

pada f (2²⁰¹⁶) 
bilangan 2 dan basis 10 sehingga log 2 = 0,301

pada  f (5²⁰¹⁶)
bilangan 5 dan basis 10 sehingga log 5 = 0,699

f (2²⁰¹⁶) + f (5²⁰¹⁶)

⇒ log 2²⁰¹⁶ = 2016 . log 2
                   = 2016 × 0,301
                   = 606,8
f (2²⁰¹⁶) dengan banyak digit (angka) = 607

⇒ log 5²⁰¹⁶ = 2016 . log 5
                  = 2016 × 0,699
                  = 1409,18
f (5²⁰¹⁶) dengan banyak digit (angka) = 1410

banyak angka (digit) = 607 + 1410 = 2017

Jadi banyak angka (digit) dari nilai fungsi f (2²⁰¹⁶) + f (5²⁰¹⁶) adalah 2017 digit.


Atau juga kita bisa menggunakan cara pendekatan hasil perkalian

f (2²⁰¹⁶) + f (5²⁰¹⁶) 
⇒  2²⁰¹⁶ × 5²⁰¹⁶ = (2 × 5) ²⁰¹⁶
                         = 10²⁰¹⁶
      dengan banyak angka (digit) 2016
     
Banyak digit (angka) dari 2²⁰¹⁶ + 5²⁰¹⁶ = 2016 + 1 = 2017 (jumlah digit dari hasil dua kali bilangan adalah ≥ 1 dari jumlah digit dua bilangan yang dikalikan)

Jadi banyak angka (digit) dari nilai fungsi f (2²⁰¹⁶) + f (5²⁰¹⁶) adalah 2017 angka.
