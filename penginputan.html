10. . berikut ini yang merupakan tahapan proses kerja computer yang benar adalah .................. * 1 poin input – proses – output output – proses –input input – output – proses output – input – proses proses – input output


Jawaban:

input – proses – output

Penjelasan:

1. Pemasukan (Input)

Tahap pemasukan (input) merupakan tahap awal dari proses pengolahan yang terjadi pada sistem komputer.

2. Pemrosesan (Process)

Pada tahap ini, data yang telah dimasukan melalui peralatan input tadi akan diproses.

3. Pengeluaran (Output)

Pada tahap ini, data yang tadinya telah dimasukan melalui peralatan input, kemudian diproses oleh CPU akan bisa dicetak apabila sudah siap.


11. input dari rangkaian paralel dibuat dari input yang


Input dari rangkaian paralel dibuat dari input yang bercabang cabang. Rumus rangkaian paralel yang perlu dipahami: 1/Rs = 1/R1 + 1/R2 + 1/R3 + . . . 3.

PEMBAHASAN

Rangkaian paralel merupakan rangkaian yang terbentuk akibat dua buah lampu atau lebih yang dihubungkan secara berjajar, sehingga menjadi rangkaian bercabang. Arus yang diterima oleh setiap cabang, akan lebih besar dibandingkan arus pada rangkaian seri, hal tersebut akan membuat nyala lampu terlihat lebih terang.

Ciri-ciri rangkaian paralel adalah :

Rangkaiannya bercabang.Kekuatan listriknya sama pada rangkaiannya.Jika satu lampu dimatikan lampu lainnya tetap hidup.Membutuhkan banyak kabel karena saat membuat rangkaian paralel.Punya banyak sakelar untuk mematikan dan menghidupkan lampu.

Contoh rangkaian listrik paralel yaitu listrik dalam kantor, rumah, dan pabrik. Sifat-sifat Rangkaian Paralel yakni tegangan pada masing-masing beban listrik persis dengan tegangan sumber. Masing-masing cabang dalam rangkaian parallel merupakan rangkaian mandiri/individu. Arus masing-masing cabang yaitu tergantung besar tahanan cabang.

Kelebihan rangkaian paralel yaitu tiap komponen bisa bekerja secara bebas, tanpa dipengaruhi oleh komponen lain. Jika salah satu komponen mati atau rusak, maka komponen lainnya juga tidak akan ikut mati berfungsi seperti semestinya.

12. apa perbedaan antara signal input dengan maintenance input


Maintenance input adalah energi yang dimasukkan supaya sistem tersebut dapat beroperasi. Signal input adalah energi yang diproses untuk didapatkan keluaran. Sebagai contoh didalam sistem komputer, program adalah maintanance input yang digunakan untuk mengoperasikan komputernya dan data adalah signal input untuk diolah menjadi informasi.

14. Apa perbedaan antara batch input dan interactive input? dan contohnya


Pada batch processing, data yang dimasukkan akan dihimpun dahulu menjadi satu kelompok yang kemudian akan dimasukkan ke dalam database untuk mengupdate master file.

Contoh dari penggunaan batch processing adalah e-mail dan transaksi batch processing

Sedangkan pada online processing, data yang dimasukkan a akan langsung dimasukkan ke dalam sebuah database untuk mengupadate master file pada saat itu juga.

Contoh penggunaan online processing adalah transaksi online. Dalam sistem pengolahan online, transaksi secara individual dientri melalui peralatan terminal, divalidasi dan digunakan untuk meng-update dengan segera file komputer. Hasil pengolahan ini kemudian tersedia segera untuk permintaan keterangan atau laporan.


15. alat input suara? alat input video ? alat input game? apa saja ya minimal 5


Alat input atau perangkat input memiliki banyak jenis, diantaranya alat input suara, alat input video dan alat input game. Alat input merupakan mikroprosesor yang dipakai dengan tujuan untuk menghubungkan perangkat luar dan komputer. Benda-benda yang merupakan perangkat input diantaranya :

Alat input untuk suara adalah keyboard, mouse, joystick, touchpad, dan microphone.Alat input untuk video adalah webcam, flashdisk, keyboad, CD/DVD ROM, dan mouse.Alat input untuk game adalah modem, flashdisk, joystick, headset, digitizer, dan barcode scanner.Pembahasan :

Perangkat input atau alat input merupakan suatu sistem yang terdapat pada komputer. Komputer sendiri memiliki pengertian suatu sistem yang terdiri atas banyak komponen yang tersusun sedemikian rupa menjadi suatu kesatuan. Komponen-komponen itu nantinya akan saling terhubung dan bekerja sama untuk melakukan suatu proses sehingga mampu menghasilkan suatu informasi pada komputer yang akan ditampilkan pada layar.

Perangkat input memiliki 2 jenis, diantaranya adalah perangkat input langsung dan perangkat input tidak langsung. Perangkat input langsung adalah perangkat untuk memasukkan data secara langsung maksudnya, tanpa perlu diterjemahkan lagi oleh operator. Sehingga untuk perangkat input komputer yang satu ini akan memiliki resiko kesalahan yang sangat kecil dan waktu proses inputnya relatif lebih cepat.

Sedangkan perangkat input tidak langsung memerlukan bantuan operator  dengan cara manual diketik untuk pengerjaannya, sehingga resiko kesalahannya cukup tinggi.

Alat input untuk suara adalah keyboard, mouse, joystick, touchpad, dan microphone.Alat input untuk video adalah webcam, flashdisk, keyboad, CD/DVD ROM, dan mouse.Alat input untuk game adalah modem, flashdisk, joystick, headset, digitizer, dan barcode scanner.

17. jelaskan arti dari komputer bertugas menerima input, memproses input, menyimpan input serta memproses output​


PENDAHULUAN

Saat komputer menerima data yang dimasukan pengguna, komputer akan memproses data tersebut dan menghasilkan output yang diminta oleh pengguna.

Berikut adalah proses komputer saat memproses data yang diteirma sampai menghasilkan hasil.

PEMBAHASAN

• Proses dalam komputer secara singkat

Input → Menyimpan → Proses → Menyimpan → Output

Semua proses ini terhadi di dalam perangkat keras bernama Computer Processing Unit (CPU) yang merupakan otak dari komputer.

1. Menerima Input Bagian komputer yang berfungsi untuk menerima input adalah perangkat keras. Input dapat diketik menggunakan perangkat keras keyboard dan dimasukan dengan mengklik menggunakan mouse. Input yang sudah dimasukan oleh pengguna akan di terima oleh komputer dan akan dibawa ke proses selanjutnya yaitu menympan input menggunakan bus.2. Menyimpan Input (1) Input yang sudah diterima akan dibawa ke register memori dimana data input akan di simpan sementara di sana untuk nanti diproses. Kenapa data tidak langsung di proses dan harus di simpan terlebih dahulu? Karena, data yang dimasukan itu ada banyak, jadi untuk memproses data, data harus dibawa satu - satu sehingga akan memakan waktu dan butuh tempat penyimpanan. 3. Memproses Input Input akan satu - satu diproses dan masuk ke dalam Computer Unit (CU). Di dalam CU, data input akan dipisah dengan 2 cara yaitu secara aritmatika dan logika. Data yang berupa aritmatika akan dikirim dan diproses di dalam Arithmetic Logic Unit (ALU) dan data yang berupa logika akan tinggal diproses oleh Computer Unit (CU) 4. Menyimpan Input (2)Sebenernya data yang di simpan di bagian ini bukanlah 'input' karena sebagian besar sudah di proses sebelumnya di dalam CU dan ALU. Tapi, ya pokonya input yang sudah di proses akan disimpan kembali di dalam register memori pada ALU untuk persiapan output selanjutnya. 5. Memproses OutputData input yang disimpan di ALU, akan dikirimkan kembali ke CU untuk diproses/dirapihkan agar dapat ditampilkan sebagai output. Sebelumnya data - data hanyalah diproses dan hasilnya disimpan, dalam bagian 'memproses output' hasil yang sudah di proses akan dirapihkan agar siap menjadi output dan ditunjukan kepada pengguna. 6. OutputOutput yang selesai dirapihkan akan dikeluarkan oleh pernagkat keras kembali. Hasil data (output) dapat dilihat di dalam monitor atau dapat dipegang dan dilihat dengan cara diprint menggunakan perangkar keras printer.
