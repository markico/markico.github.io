Diketahui :
Vo = 3 m/s
t = 1,5 m/s
g = 9,8 m/s²
Ditanyakan : a. S dan b. Vt ?

dijawab :
a. Jarak :
St = Voxt + 1/2xgxt²
    = 3x1,5 + 1/2 x 9,8 x 1,5²
    = 4,5 + 4,9 x 2,25
    = 4,5 + 11,025
    = 15,525 m

b. Kecepatan saat ditangkap :
Vt = Vo + g x t
    = 3 + 9,8 x 1,5
    = 17,7 m/s
