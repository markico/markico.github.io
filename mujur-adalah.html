1.260.000 = 35.000 n

n = 1.260.000 : 35.000

n = 36 bulan

karyawan menabung uang sebesar

7.000.000

=7.000.000×6%bunga

=42.000

=8.260.000-7.000.000

=3

26. "mujur".kak tolong jadiin aksara jawa beserta tanda petik sama titiknya​


Jawaban:

"ꦩꦸꦗꦸꦫ꧀"꧉ = "Mujur".

Sistem Penulisan:

• " (Tanda petik [Alfabet]) + ꦩ (Ma [Wyanjana]) + ꦸ Sandhangan U (Suku) = "ꦩꦸ (Mu)

• "ꦩꦸ (Mu) + ꦗ (Ja [Wyanjana]) + ꦸ Sandhangan U (Suku) = "ꦩꦸꦗꦸ (Muju)

• "ꦩꦸꦗꦸ (Muju) + ꦫ (Ra [Wyanjana]) + ꧀ (Mati [Pangkon]) + " (Tanda Petik [Alfabet]) + ꧉ (Titik [Pada Lungsi]) = "ꦩꦸꦗꦸꦫ꧀"꧉ ("Mujur.")

Catatan:

Pada aksara jawa, tidak ada tanda petik, maka bisa dalam penggunaan tanda petik kita mengikuti tanda petik dalam alfabet internasional saja yaitu ( " ). Tetapi adapula tanda pada aksara jawa untuk sisipan pada kalimat yaitu Adeg-Adeg Pisèlèh ( ꧌...꧍ ). Adeg-Adeg Pisèlèh dapat dipakai sebagai sisipan atau bagian penting suatu kalimat atau bisa digunakan sebagai tanda kurung, dan sebagai alternatifnya untuk tanda petik. Jadi kita pakai dalam kata "Mujur". adalah "ꦩꦸꦗꦸꦫ꧀"꧉ ataupun jika menggunakan Adeg-Adeg Pisèlèh yaitu ꧌ꦩꦸꦗꦸꦫ꧀꧍꧉ walaupun Adeg-Adeg Pisèlèh untuk tanda kurung, tetapi kita bisa dijadikannya sebagai tanda petik (penggunaan alternatif).
