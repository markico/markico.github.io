19. vv.²√576 + ²√289 + 4² = ...​


Penjelasan dengan langkah-langkah:

√576 + √289 + 4²

=√(24²) + √(17²) + (4×4)

=24 + 17 + 16

=41 + 16

=57

Penjelasan dengan langkah-langkah:

[tex] = \sqrt[2]{576} + \sqrt[2]{289} + {4}^{2} [/tex]

[tex] = \sqrt[2]{ {24}^{2} } + \sqrt[2]{289} + {4}^{2} [/tex]

[tex] = {24}^{ \frac{2}{2} } + \sqrt[2]{289} + {4}^{2} [/tex]

[tex] = {24}^{1} + \sqrt[2]{289} + {4}^{2} [/tex]

[tex] = 24 + \sqrt[2]{289} + {4}^{2} [/tex]

[tex] = 24 + \sqrt[2]{ {17}^{2} } + {4}^{2} [/tex]

[tex] = 24 + {17}^{ \frac{2}{2} } + {4}^{2} [/tex]

[tex] = 24 + {17}^{1} + {4}^{2} [/tex]

[tex] = 24 + 17 + {4}^{2} [/tex]

[tex] = 24 + 17 + (4 \times 4)[/tex]

[tex] = 24 + 17 + (4 + 4 + 4 + 4)[/tex]

[tex] = 24 + 17 + (8 + 4 + 4)[/tex]

[tex] = 24 + 17 + (12 + 4)[/tex]

[tex] = 24 + 17 + 16[/tex]

[tex] = 41 + 16[/tex]

[tex] = 57[/tex]
