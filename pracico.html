4. Jelaskan perbedaan blocked practice dan random practice


Perbedaan antara blocked practice dan random practice ialah kalau blocked practice merupakan suatu model latihan yang mana berfungsi untuk melatih gerak para siswa- siswa dan model ini  di lakukan secara berpusat sedangkan random practice merupakan suatu model latihan yang mana berfungsi untuk melatih gerak para siswa- siswa dan model ini  di lakukan secara acak.
